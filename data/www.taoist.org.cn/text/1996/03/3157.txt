　　道教是我国土生土长的传统宗教，已有近二千年的历史。它是具有浓厚中国古代文化色彩的宗教，在宇宙观、世界观、人生观上具有它独特的见解。
　　它在长期发展过程中，对我国古代社会的政治、经济、哲学、医学、养生等以及民族关系、民族心理和社会习俗等各个方面都曾产生过深刻的影响。
　　就思想信仰而言，它以“道”为“教”，奉元始天尊、灵宝天尊、道德天尊 ( 即太上老君 ) 为最高尊神，并奉太上老君为教祖，尊奉《道德经》为至理经训，认为“道”是宇宙人生以及万事万物的根本。道教继承先圣们清静自然、返朴归真的思想。更为提倡的是道法自然的入世与出世思想。
　　金代王重阳祖师开创的全真教，倡导全神炼气，抱元守一，外修真性，内炼金丹，主张修自身元炁，葆自身元神，提出“神气相合，谓之神仙”，强调修炼之始，“先明正理，悟天地生成、造化之玄机，然后登堂入室，方有所获” ; 更为提倡道儒释三教合一。教旨以“澄心定意，抱玄守一，存神固气为真功” ; 以“济贫拔苦，先人后己，与物无私为真行”。功行俱全，故名全真。它认为通过个人不懈的修炼能达到得道成仙与大自然融为一体的境界。
　　至于“自然”二字，近百年来，人类物质文明之发展，日新月异，前所未有，人类正在竭尽全力地试图去认识自然，掌握自然，甚至有人企图去创造自然。
　　在人与自然方面，人们往往只知道从自然中索取，却忽略了对自然的保护，于是自然生态受到破坏，人类生存环境受到破坏。最后人自己也受到自然的惩罚。至于有人企图再去创造一个自然，其结果是牺牲一个现实的自然为代价。这也就是我们道教认为的得失相依的“天道”。另一方面则是表现在人与人之间的关系上，人们往往只知道向社会索取，而不愿意多向社会作贡献及付出。于是当今的人类社会出现了许多物欲横流、金钱万能、鱼肉他人的丑闻和坏事。
　　我们的“道教”和其它宗教一样处于如此的世界之中，有的道教徒感到困惑，不知所措 ; 有的则随波逐流，迷惑道性，如此一来，有的道教徒就同一般俗人混同一起，没有多大的区别。“道士”本应是“有道之士”，道教徒一旦失去了“道”的本性，那么道教作为一个宗教在社会上就失去了教化的功能和存在的意义。其结果将是逐渐趋向衰落。
　　今天我们这一代道教徒正面临着这一趋势，将如何去对待这一现实呢 ? 那就要靠我们道教徒自己。首先要提高自身的道德修养，以道来武装自己，努力使自己回到道的本来状态。大道本来是素朴清静的，宇宙间的一切生命力都隐含于此。我们必须要保持“道”的信仰特征，体现道上的精神风貌，是在当代社会中振兴道教的关键所在。然而保持信仰特征和体现道教精神风貌就要靠我们对“道德”的理解与信仰的力量。
　　我们的道教信仰，是以“道”为核心，以得道为追求的目标，有道之士应如《道德经》中所说 :“处无为之事，行不言之教”。这里的“不言之教”就是要我们以自己的“行为”去表现出来，我们的行为就是要体现“道”的信仰，就是处处要做符合自然和社会要求的事。
　　那些山居的道士，在进行宗教活动的同时，还特别注意植树造林，种植花草，保护环境，开发旅游点，使自己有个良好的修道的环境 ; 处于城市的宫观，应特别注意绿化和美化生存环境，注意安全消防等事项。只要我们做了符合天道的事，天道也会赋予我们好的自然条件。
　　《道德经》说 : “善人者，不善人之师，不善人者善人之资。”正因为如此，道教历来不强调只注重形式上的布道的作法，但并不等于说道教不要宣传和讲述教理教义。而是要口传身教以实际行动去感化世人。《道德经》还说 ; “居善地，心善渊，与善仁，言善信，政善治，事善能，动善时。”由此可见对于我们道教徒的行为，太上教诲关键的是一个“善”字。一般人理解“善”是善良的意思，古人言 : “乐善好施”。说的是要帮助他人，救济贫困，乃至不宰杀生灵等。可是太上的教诲远远超出这个范围。“动善时”就是要我们所做的一切事情都要符合自然的规律。不落后于时代，更不能拉时代的后腿 ; “与善仁”就是我们的所作所为都要符合“仁”的要求，所做的任何事情都要有利他人。正如古人云 : “先天下之忧而忧，后天下之乐而乐。”要具有这样的胸襟，谈何容易。但我们应努力去做到。“心善渊”就是要我们的心肠善得如同百丈深水，对于国家、民族及我们道教、同道和信徒的事，都要有“善”心。对于别人的困难、灾难要同情和帮助 ; 对于他人的努力要支持。“言善信”就是要我们的行为不失分寸，承诺不失信用。如我们能按照太上的“善”字去规范自己的一举一动、一言一行，道教和我们道教徒在民众中的形象就能真正得到提高。
　　历来我们的道教主张“我命属我，不属天地。”其意是鼓励我们要自立，《道德经》第五十四章说 : “修之于身，其德乃真 ; 修之于家，其德乃余 ; 修之于乡，其德乃长 ; 修之于国，其德乃丰 ; 修之于天下，其德乃普。”太上说到修道的五个范围，从修身逐渐扩大到家庭、乡里、国家和天下各个领域。显然人在这五个领域里行为的内容是不同的，因此道德的表现也有所不同，但是太上并没有规定这五个领域的“道”和“德”的具体内容。这个具体内容也要我们后世的学道者按照太上的教诲，自已去订出规矩来作为我们自己的规范准则。这也就是自立。历代道教在不同的时期，就有许许多多的不同的清规戒律，那些戒律也是先圣们根据当时的情况而制定出来的。有些规戒已经不适应今天的社会生活了。这就要求我们学道的人根据太上的教诲和当今的社会状况在执行规戒时也给自己订立一些新的规章，作为我们自己平时的行为准则。
　　以前我总认为，我们这些出家的道士理应隐居脱俗入山修炼才是。当今随着社会物质文明的发展，城乡差别越来越小。在这种情况下，学道修身就要如同北魏侯楷所说的“道在方寸，何必山林。”也就是说只要自己方寸有道，而不依赖于山林环境，若方寸有道，哪怕置身于车水马龙之中，仍可以一尘不染。方寸之道，是在三宝。《道德经》中说 : “我有三宝，持而宝之，一曰慈，二曰俭，三曰不敢为天下先 ; ”“慈”是对人而言，在思想上保持爱心，一人学道还要带动周围信徒共同学道。“俭”则是对己而言，在思想上常保持清虚，以学道为标准，生活中保持勤奋，以修道为宗旨，而不敢稍有怠惰。“不敢为天下先”，对天下而言，在思想上保持积极的进取之心，坚持不懈地学道。《道德经》又云 : “圣人后其身而身先，外其身而身存。”由此不敢为天下先，并不是甘为天下之后的意思，而是为而不露，行而不耀，积极进取，自然“后其身而身先”，若是天下人都学道，我们学道要有“身先”的水平 ; 若是天下人都不学道，我们仍要坚持学道，修道以成“身先”。
　　总之，作为我们这一代人不仅仅要懂得道理，还要在修持方面去实行。承道、弘道理应是我们的天职，将怎样担起这重任呢 ? 则要我们以太上的教诲和当前的中华人民共和国的有关法律、法规来严格要求自己，我认为首先必须要有较高的道德文化修养以及无私奉献的精神 ; 再就是要有较高的政治素质。以身作则，民主团结，齐心协力，才能与社会主义社会相适应，才能担起承道及弘道的重任。
