　　王真是唐代后期的一位政治思想家，他的《道德经论兵要义述》，是一部道家军事政治学的名著，先后收入明《正统道藏》和新编《道藏》等重要道家和道教丛书之中，其中所阐述的“用其所不用”、“权与道合”的用兵之道，不仅反映出他在当时思想的进步性，而且也为我们今天如何加强国防建设、解决国际争端提供了可借鉴的精神资源。
一
　　王真曾有感于中后唐社会政治动荡、兵战给国家、人民所造成的巨大灾难，明确提出“天下之利害，莫大于用兵” ( 《论兵要义述》卷四 ) 。因此，对于用兵，是他非常关心、反复探讨的一个重要问题。他认为，对于国家民族的安危而言，没有强大的兵力，是不足以保障长治久安的。因此，他指出 : “夫文者， 武之 君 ;武者，文之备也。斯盖二柄兼行，两者同出，常居左右，孰可废坠 ? 故曰 : 忘战则危，好战则亡。是知兵者可用而不可好也，可战而不可忘也” ( 同上，卷一 ) 。
　　他曾针对兵战之祸 . 提出“天下之利，莫大于戢兵” ( 同上，卷三 ) 的主张。但是，止兵息战只是一种社会政治理想，并不是现实政治策略。对于现实的国家、人民之安危来说，还是不能废武、忘战，而必须文武兼行。只有这样，才能逐渐接近和达到止兵息战的社会理想，使天下家国永远太平无事。王真的这种观念，体现了理想与现实的对立与统一，说明《论兵要义述》并不是像有的学者所认为的那样“迂腐不堪，实不足观者” （李泽厚）中国古代思想史论》，第 77页。人民出版社， 1986 年版 ) 。
　　王真认为，用兵之道，首先要深刻地体会道君老子“不争”的意蕴。他深有体会地说：
　　窃喜习读五千之文，每至探索奥旨，详研大归，未尝不先于无为，次于不争以为教父。凡人之情，不能无争。唯圣人乃能无争，又争之徒众矣。今臣略举梗概，皆起于无思虑，无礼法，不畏俱，不容忍，故乱逆必争，刚强必争，暴慢必争，忿至必争，奢泰必争，矜伐必争，胜尚必争，违愎必争，进取必争，勇猛必争，爱恶必争，专恣必争，宠嬖必争。夫如是，王者有一于此，则师兴于海内。诸侯有一于此，则兵交于其国。卿大夫有一于此，则贼乱于其家。士庶有一于此，则害成于其身。是以王者知能官人、能安人之道，必当先除其病，稗之无争，则战可息矣。战可息矣，则兵自敢矣。是故其要在于不争。且夫争城，杀人盈城 ; 争地，杀人满野。……故下经末章云，天之道，利而不害 ; 圣人之道，为而不争。此所谓知始终之义也。 ( 同上，卷一 )
　　这段话非常精彩。因为好斗争是人的天性，世界上的一切事情，都会由于认识上的分歧而产生争斗。有争斗，就难免杀伐，家国安危直接受到危胁。所以，王真将“理兵之要”归结为“不争”，正说明要真正制止和减少战争对社会发展和人民的生命财产带来的巨大威胁，就应当深入到对人性的研究与调适之中，不能简单地去消灭兵战之事。这种见解，表现出了其军事政治思想的理论深刻性。
　　反对“不争”，并不等于完全“无为”。王真指出，“圣人之道，为而不争”，道君老子之所以“不言有为与无为，而直言为者，欲其人君为无为也，又欲其无不为也。” ( 同上，卷四 ) 因为“夫一家不争，即斗讼息矣。一国不争，即战阵急矣。天下不争，则征伐息矣。夫斗讼息于家，战阵息于国，征伐息于天下，此圣人之理也。故曰 : 圣人之道，为而不争，其此之谓欤。”( 同上引 )
　　很显然，要真正实行“为而不争”的“圣人之道”，关键在如何“为”上。“圣人之道”不是完全无兵，而在于有兵如何“为”。“无为而无不为”只是企盼的理想。王真把“为”理解为固守疆域、恐吓来犯之敌，而不是用以发动战争，对内以刑罚惩治百姓，对外侵略和掠夺他民族与国民。他说 :
　　圣人用兵之道，不以其愠怒也，不以其争夺也，不以其贪爱也，不以其报怨也。盖整而理之，蓄而藏之，以谨无良，以威不德。非用之于战阵，非用之于杀伐，非用之于田猎，非用之于强梁。此圣人用兵之深旨也。 ( 同上书，《叙表》)
　　这里强调了一个很重要的思想，即用兵不能因性情而定，不能不三思而后行。实际上强调了理性分析与决策能力的重要性。这种观念，当然是对历代、特别是唐代一些兵戎战事常常出于一人或少数人之私心杂念而兴起的历史教训的总结。这种强调用兵必须依据理性分析、而不能因个人性情之喜怒爱恶的思想，是有其积极的合理性的。另一方面，他也还强调了“用兵”不是目的，而只是“谨不良，威不德”的手段而已，因而不能轻易地用之于战阵、杀伐、田猎和强梁。这种思想也是有其合理的积极内容的。
　　正是由于王真把“用兵”只看作是手段，而不是所追求的目的，因而他主张对于外来之敌，应当采取政治与武力相结合的方式，特别是要尽可能地以政治方式加以解决。他说 : “夫有道之君，纵有凶暴之寇妄动而来，我师告之以文词，舞之以干羽，彼必闻义而退，自然无致。”( 同上，卷四 ) 。这段话粗看起来，确实显得比较迂腐，但是，试对照一下我们今天国际社会所面临的现实状况，细细想来，也确实很有道理。国际争端，一味以武力来解决，收效往往并不大。但是，如果有强大的武力后盾，通过政治谈判的方式，往往更有成效。应该说，王真的上述思想，剔除其旧的外壳，确实是我们民族思想传统中一个优良的资源，很有现实意义。
二
　　武力不能没有，但要尽可能地以政治方式解决国家之间的争端问题，王真称此“用兵之道”为“用其所不用”。他说 :
　　夫五兵之属，亦当其无有兵之用。且弧矢之利以威天下，不必伤然后为用。故知兵者，备之以为有，敢之以为无，此即用其所不用者也。 ( 同上 . 卷一 )
　　很显然，“用其所不用”，其前提条件是必须有“兵”。兵不可不备，这是国家赖以安全的保障。问题是 : “备兵”、“有兵”、“用兵”毕竟不是政治目的，而只是实现和保障国家安泰的一种必要手段。因此，“用其所不用”，就是将手段与目的有机地结合起来，既保障了天下国家的安全，又没有造成无辜的杀伤。这是一种人本主义的用兵思想。
　　不过，目的固然是根本，手段也不可因此而遭轻视。人不犯我，我不犯人 ; 人若犯我，我必犯人。所以，有兵、备战，固然不是为了侵略别国，而只是为了对付可能来犯的顽敌，但是，绝不可轻敌。道君老子曾告诫说 : “祸莫大于轻敌。”轻敌，必招致国家灭亡、人民涂炭。关键是如果有外敌来侵犯我疆域、欺侮掠夺我人民，我不得已而用兵，政治方一式解决不了，非要诉诸武力不可，也应当“应敌而起”，不可先敌而妄动，以免造成侵犯他国之嫌。他说 : “夫兵者，必先举为主，后应为客也。且圣人之兵，常为不得已而用之，故应敌而后起。” ( 同上，卷一 ) 不主动挑起战争，但是，战争既已发生并不能避免，也毫无退缩，勇敢地捍卫自己的领土、主权与尊严。
　　老子在《道德经》中专门阐述了“上善若水”的道理。王真认为，老君对水性的揭示，颇能用以指导用兵，“特谕理兵之要深至矣”。 ( 同上引 ) 他指出 : ‘“夫上善之兵之于水。然水之溢也，有昏垫之灾 ; 兵之乱也，有涂炭之害。故水治则润泽万物，通济舟楫。兵理则镇安兆庶，保卫邦家。若理兵能象水之不争，又能居所恶之地不侵害者，则近于道矣。” ( 同上引 ) 中国自古以来是个农业国家，对水的认识和治理有着几千年的历史经验。王真发挥老子“上善若水”的思想，也在一定程度反映出其思想具有传统农业社会的素朴性与辩证性。
　　手段毕竟是要为目的服务。为了达到和平安全的目的，要尽可能避免动用武力。王真进一步认为，不仅政治手段高于武力手段，而且道德手段高于政治手段。要真正解决争斗所引起的对人的无辜杀伤，还得从增强道德修养、提高道治能力着手。因而，他说，理兵之最要者，不在强兵好战，而在“慈能勇”、“俭能广”。他指出 :
　　慈能勇……俭能广……是以三皇用之以克九黎，五帝用之而去四凶，汤武用之而以兵胜天下，成康文景用之而刑罚皆措。及其桀纣舍之而国灭，幽厉舍之而身亡，秦赢舍之而二代夷戮，项籍舍之而五体割分，汉武舍之而天下减半，曹公舍之而吴蜀鼎峙。 ( 同上，卷四 )
　　他甚至指出，“兵者，剑戟戈矛之类也。”即使是遇到外寇侵犯，“君子必不得已而用之”，也应“当以恬淡为上” ( 同上，卷二 ) ，而不应当侍机侵犯杀伐。不得已而用兵，不是要报复别国人民，而是争取早日解决争端，减少无辜的杀伐。他说 : “王者治人事大，必当以仁爱为宗。……夫仁爱之道行焉，则天下早服。天下早服，故谓之重积德。重积德者，以战则胜，以守则固，故无不克。” ( 同上引 ) 而且，“王者守国有道，自然根蒂深固，以享长生久视之福也。”( 同上引 ) 更何况，只要不主动挑起争端，有了争端力求避免扩大事态，尽力以和平的政治方式加以解决，则“天下莫能与之争，则兵战自然息矣。” ( 同上引 ) 因此，“王者执大象，不失其道，则天下之人无不归往” ( 同上引 ) 。
　　不仅为人君者应当“重积德”，对于将相辅弼之臣来说，也应当提高道德索养，力求“以道佐人主”。他指出，在秦代，李斯、赵高、白起、蒙恬等辅臣，都不是以“道”来辅佐秦王，而是重用武力、暴强，对天下大肆吞噬、攫搏，还焚毁诗书，坑杀儒生，杀伐赵人四十余万。“其于所下之处，悉残杀之，使无遗种。始皇犹独鹗视天下，未足其心。虽天禄已终，而毒螫之余，仍相残害，不经时而土崩鱼烂矣。是以胡亥弑于望夷，子婴戮于咸阳，扶苏死于长城，李斯父子糜溃于云阳，白起齿剑于杜邮，赵高取灭于宫闱，此皆事之还也。” ( 同上引 ) 因此，辅弼之臣如果暴虐无忌，不能以道辅佐人主，必致国家灭亡之灾。
三
　　王真认为，归根结蒂，圣人用兵之道，仍在“为而不争”。“不争”是目的，“为”是手段。“不争之义，与天同德，美利万物，不言善应，周流六虚，不谋善胜” ( 同上引 ) 。他将“不争”看作是“道”的必然要求，而将“用兵”动武看作是一种政治“权”谋，极力主张“权与道合。”
　　他说 : “治国者以政，政者正也。君率以正，孰敢不正 ? 用兵者以奇，奇者权也。权与道合，庸何伤乎 ? ”如果“使天下之人皆得肆其权谋，操其利器，非昏而何 ? 昏犹乱也。” ( 同上，卷四 ) 很显然，所谓“权与道合”，就是指用兵不能脱离大道这个根本。大道要求以“正”治国、以“奇”用兵。“奇”与“正”，要相互配合，且以“正”为主导和归依。这种观念，仍然是强调政治方式比武力 ( 军事 ) 方式更重要。因此，我们不能简单地将王真的《论兵要义述》看作是一部论兵的专著。实际上，它是一部阐述治国与用兵相互关系的政治著作。它不是专门论述如何用兵，更重要的是阐述如何对待用兵。
　　在他看来，用兵只是一种权谋手段，是“奇”，而治理好国家，国泰民安，才是真正的目的，即“正”。他之所以一再强调要“为而不争”，就是强调手段必须以目的为根本。否则，兴师动众，只可谓祸国殃民。岂知“兴师十万，日费千金。十万之师在野，则百万之人流离于道路矣。加之杀气感害，早疫相乘，灾之深，莫甚于此。故善为将者，当须果而已，不敢以取强。果而勿强，果而勿矜，果而勿骄，果而不得已。” ( 同上，卷四 ) 合理的用兵，是面临来犯之敌，政治手段不能发挥作用了，而不得已动用武力。但是，此时切不可逞强、骄躁。争端解决了，就应当马上止兵。所以说 : “果而勿强，不得已者，是圣人用兵之要道也。” ( 同上引 ) 因为，逞强、骄躁，不仅会给人民造成巨大伤害，而且也将耗费自身的大量财力与物力，其结果，实际上是两败俱伤。
　　王真的上述思想，与当时的政治需要有密切关系。王真奉献《论兵要义述》给唐宪宗，而这个唐宪宗是唐代后期的“中兴英主”。“其为政宗旨在矫正大历、贞元姑息、苟安之积习，即用武力削平藩镇，重振中央政府之威望。当时主张用兵之士大夫大抵属于后来所谓李党，反对用兵之士大夫则多为李吉甫之政敌，即后来所谓牛党，而主持用兵之内廷阉寺一派，又与外朝之李党互相呼应。自不待言，是以元和一朝此主用兵派之阉寺始终柄权，用兵之政策因得以维持不改。及内廷阉寺党派竞争既烈，宪宗为别一反对派之阉寺所弑，穆宗因此辈弑逆徒党之拥立而即帝位，于是‘销兵'之议行，而朝局大变矣” ( 陈寅恪《唐代政治史述论稿》，载《 陈寅恪先生论集》，第 170 — 171 页。台湾中央研究院历史语言研究所特刊之三 ) 。王真献书唐宪宗，亦正值宪宗指派吐突承璀出兵讨伐王承宗，以革河北藩镇世袭之弊。王真不仅没有因阐述载兵息战之理而引起唐宪宗的反感，而且还得到唐宪宗的赞誉，亲敕《手诏》，肯定王真“本乎道德之旨，参以理化之源，用究玄微”( 《道藏》，第 13 册，第 632 页。天津古籍出版社、文物出版社， 1983 年版 ) 。这显然与五年后金州刺史张仲方“驳李吉甫溢议，言用兵征发之弊由吉甫而生，宪宗怒” ( 《旧唐书》卷一七二《萧　传》) ，将张仲方“贬为遂州司马”( 《旧唐书》卷一七一《张仲方传》) 大相径庭。这说明王真的戢兵思想并没有张仲方那样带有明显的反对李吉甫的倾向，不会与一向看重李吉甫的唐宪宗产生明显的对立，同时，由于他肯定了在不得已用兵的合理性与必要性，实际上为唐宪宗“合理”用兵从李家祖典《道德经》中找到了理论根据。因此，可以认为，王真的思想，在当时牛李党争之中表现出一定的调和倾向。但究其实质，在很大程度上与穆宗朝“销兵”派，即新兴阶级之急进派牛党的观念更接近。这也反映出王真的用兵之道是自觉地顺应着正在崛起中的新兴地主阶级。
