　　往昔，北京庙会时间最长的要属白云观的“燕九节”，从夏历正月初一开庙，至初八日祭星日，游人始盛，十八、十九两日游人最多，庙西垣外有走马场，逐日比赛。“燕九节”之称始见元代熊梦祥著《析津志》 : “正月一日至十九日，都城人谓之“燕九节”，倾城士女曳竹杖，俱往南城长春宫、白云观，宫观蒇扬法事烧香，纵情宴玩以为盛节”，“燕九节”又曰讌邱，宴邱节，是庆祝邱真人诞辰的节日。“真人生于金皇统八年戊辰正月十九日。自元以来历数百　，京畿黎庶每于是日致浆祠下，不啻归市。於时松下多玄门，结圜室十余所，趺坐说法。至于冶郎游女，纷纭杂沓，则又谑浪无忌，恬然不以为怪也” ( 明蒋一葵著《长安客话》) 。
　　正月十九日，“相传是日，真人必来，或化冠绅，或化游士冶女，或化乞丐。故羽士十百，结圜松下，冀幸一遇之” ( 明刘侗、于奕正著《帝京景物略》 ) 。
　　自元以来，特别是明清时代，不少文人作诗以纪“燕九节”所见。如明人潘之恒六言诗去：
　　燕市重逢燕九，春游载选春朝。寒城旭日初丽，暖谷微阳欲娇。公子高类锦障，侍中齐插金貂。书传海外青鸟，箭落风前皂鵰，翟　①烟尘骤合，马蹄冰雪全消。张罗释兔求雉，投博②呼卢得枭。剑说荆卿匕首，舞怜蛮女纤腰。闹峨③人胜争贴，怖鸽天花乱飘。台上试听萧史，峰头方驾王乔。宝幢星斗斜挂，仙乐云　碎敲。高辅少年任侠，倡楼大道相邀。寄言洛社豪举，莫笑春光不饶。 ( 载《长安客话》)明代吴宽《燕九诗》 ;
京师胜日称燕九，少年尽向城西走。
白云观前作大会，射箭击球人马蹂。
古祠北与学宫依，箫鼓不来牲醴稀。
如何义士文履善，不及道人邱处机。
( 载吴宽《家藏集》)
　　清代戏曲作家孔尚任等撰《燕九竹枝词》词序云 : “……唯燕九之游，差有昔人遗意。是日( 指正月十九 ) 为陈子健夫见召，走马春郊，开筵茅屋，命简抽毫，各为十绝句。虽难叶于巴渝之歌，或有合于吴趋之节，但按之琵琶羌管，恐未有当耳。”
　　《燕九竹枝词》题下说明 : “集陈健夫山斋限用《庾开府集》“结客少年场，春风满路香”句，为韻十首。唱和者有孔尚任、陈于王 ( 健夫 ) 、袁启旭、蒋景祁、陆又嘉、王位坤等九人，共得九十首。仅录孔尚任十首。
一
春宵过了春灯灭，利有燕京燕九节。
才走星桥又步云，真仙不遇心如结。
二
秧歌忽被金吾革，袖手游春真可惜。
留得凤阳旧乞婆，漫锣紧鼓拦游客。
三
头顶炉香问道妙，村翁扶杖坛边笑。
年年来看求仙人，今日不如千年少。
四
官吃东涞商吃泉，烧刀只赚小民钱。
连秋黍贵无乡贩，醉汉不如庚午年。
五
七贵五侯势莫当，挨肩都是羽林郎。
他家吹唱般般有，立马闲看　 ( 音帚、执、持义 ) 戏场。
六
金桥玉洞隔凡尘，藏得乞儿疥癞身。
绝粒三旬无处诉， 被人指作邱长春。
七
结伴儿童　褶红，手提　索骂天公。
人人夸你春来早， 欠我风筝五丈风。
八
凤城城外晴春暖， 咫尺仙源路甚坦。
不把香车让画船， 一般载得笙箫满。
九
紫云茶社斟甘露， 八宝元宵效内做。
今日携钱忍饿归，便门不及前门路。
十
千里仙乡变醉乡， 参差城阙掩斜阳。
雕鞍　辔争门入，带得红尘朴鼻香。
　　其它八人的同咏，选录九首。
陈于王三首
白云观前沙如雪，羊角风吹沾眉睫。
可怜游女鬓成丝，绿衫汗污丁香结。
 
黄冠趺坐围松柏，欲募青蚨 ( 钱的别名）谈炼药。
好事颇多城阙人，鹑衣皆认为丹客 ( 指长春真人 ) 。
 
锣鼓喧阗满钵堂，莺弹花旦学边妆。
三弦不数江南曲，唯有　　独擅场。
　　诗下注云 : “内有钵堂，向为羽士闹钵之地，今为戏场矣。”
 
袁启旭二首
秧歌初试内家装，小鼓花腔唱凤阳。
如蚁游人拦不住，纷纷挤过蹴球场。
 
神仙端的是谁人，笑杀黄冠羽服身。
一片软红迷去住，白云何处觅长春。
 
蒋景祁一首
缭绕云烟涨碧空，醮坛西转画桥通。
何因到处闻仙乐，天半吹来面面风。
 
王位坤三首
帝城才过元宵节，摩肩击毂何更迭。
观前观后笛声高，白云飞去红尘结。
 
宫观巍峨霄汉边，九重当日拜神仙。
醮坛莺鹤知来否，狼藉春风又一年。
 
今年燕九春风暖，日暮游人犹未散。
传言若个遇仙真，前车不行后车满。
　　又清人潘荣陛《帝京岁时纪胜》云 : 燕九节“车马喧阗，游人络绎。或轻裘缓带簇雕鞍，较射锦城濠畔 ; 或凤管莺箫敲玉版，高歌紫陌村头。已而夕阳在山，人影散乱，归许多烂醉之神仙矣。”
　　从上面的诗文中，可以见到往昔“燕九节”的盛况。
