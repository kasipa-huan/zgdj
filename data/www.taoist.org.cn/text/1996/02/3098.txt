一、八年工作成就
　　河南省道教协会，自 1987 年 10 月成立以来，至今已走过八年零五个月的路程。在这八年多的时间里，省道协在党和政府的领一导下，带领全省道教徒艰苦创业，取得了较大成就。
　　( 一 ) 协助党和政府，落实宗教政策，合理安排了活动场所。
　　十一届三中全会以来，党和政府全面贯彻落实了宗教信仰自由政策。省道协带领全省道教徒在政府宗教部门领导支持下，齐心协力，艰苦奋斗八年，协助政府共开放道教活动场所六十二处。其中郑州市二处 ; 洛阳市三处 ; 平顶山市二十九处 ; 南阳二十三处 ; 周口二处 ; 鹤壁一处 ; 焦作二处。成立道教协会十三个，分别是 : 郑州市道教协会、洛阳市道教协会、平顶山市道教协会、南阳市道教协会、鲁山县道教协会、襄城县道教协会、叶县道教协会、南召县道教协会、方城县道教协会、邓州市道教协会、内乡县道教协会、鹿邑县道教协会、架川县道教协会。
　　( 二 ) 开展教务工作，提高道人素质。
　　随着党的宗教政策的全面落实，道人已有六百多人 ( 含未开放庙宇 ) ，但普遍存在素质较低问题。为了提高道人素质，省道协向中国道教协会举办的道教知识专修班、中国道教学院输送学员，向中国道协开办的全真戒坛、正一箓坛授戒授箓。通过省道协的两次经韵班的开办等多种渠道，共计培训老、中、青道人约一百五十人次。八年筹措培训投资，计三万多元 ( 含培训班、全真戒坛资助等 ) 。八年前我省能够坚持早晚功课的开放宫观甚少，八年后的今天，全省大部分开放庙宇不但能够坚持正常的早晚功课，而且有的庙宇还能在庙内或出外做各种节日道场。通过以上工作的开展，道人的素质进一步提高。
　　 ( 三 ) 发展经济，维修宫观，为社会做贡献。
　　解放前，我省庙宇破烂不堪，道人生活无保障，解放后，政府对我省庙宇的修缮投入了一定数额的资金，但因管理不善而成效甚微。省道协成立以来，我省道人精神振奋，使我省开放庙宇能够很快实现自养。其经济收入来源是 : 门票收入，香资收入，布施捐资收入，第三产业收入等。这些经济收入主要用于宫宇维修，道人生活，社会福利等。八年来，全省开放庙宇共整修、重建殿堂、道房、厅堂计三千余间，全塑神像三千余尊，栽植花草树木十万多棵 ; 向灾区捐款捐物计人民币约三十万元，向希望工程捐款五万元 ; 向残疾军人、学校、老弱病残、桥补路等捐资计款额二十多万元。
　　( 四 ) 加强爱国主义教育，学好时事政策。
　　爱国爱教是中国道教界的优良传统，道教界认为，没有一个巩固安定团结的国家，社会就会动荡，人民无法生存，更谈不上道教的发展，所以道教历来是倡导和平，祈祷风调雨顺，国泰民安。省道协发扬爱国爱教爱和平这一优良传统，利用大小会议和调查间隙向道众宣传。 1995年 8 月15日是中国人民抗日暨世界反法西斯战争胜利五十周年纪念日，省道协响应中国宗教和平委员会的号召，在我省中岳庙、兴阳观举行了为期三天的道场法会 (8 月 14 日至 16日 ) ，通过这次活动，体现了河南省道教爱国爱教爱和平的愿望。
　　( 五 ) 加强宫观管理，加强自身建设。
　　宫观管理对于道教界来说，是一项巨大的工程。为管理好开放宫观，八年来，省道协列入议事日程做了很多工作。首先于 1988 年制定了《河南省道教活动场所管理办法》，开办了一次宫观管理学习班。召开了两次宫观管理表先会，编印了融法律、法规、宗教活动场所管理办法为一体的《宗教活动场所管理法规·道教活动场管理办法汇编》小册子发放到全省各开放庙宇、道协组织。为管好庙宇省道协两次组团到四川、陕西、湖北、北京、辽宁等省市开放庙宇学习管理经验。并多次调解宫观管理中矛盾，使庙与庙之间、道教管理组织与管理组织之间、道人与道人之间的团结进一步加强。在道教文化方面，八年来，购《正统道藏》一部，购《藏外道书》一部，购道教书刊五百余册。全省有的开放庙宇还购置了电视机、电脑打字机、录放像机等文化用品，丰富了道众的文化生活。在道协工作方面，省道协召开了八次理事会议，坚持了每年一次办公会议制度，协助地、市、县成立了地方道教协会组织十三个，接待中外团体三十余次。
　　在取得工作成就的同时，河南道教还存在一些不足之外，如河南道教人才少，老中青结合不十分理想，不能适应道教发展的需要 ; 宫观管理的制度也不够健全。这些问题，亟待在今后的工作中加以解决。
二、十五年工作设想
　　党的十四届五中全会关于制定九五计划和2010 年远景目标的建议，向全国人民展示了实现国家富强，人民富裕，民族振兴和社会长治久安的跨世纪宏伟蓝图，我们道教应为实现这一宏伟蓝图做出应有的贡献。八年工作成就奠定了工作的基础，今后的十五年，就是省道协及全省道教界为之奋斗的十五年，创业的十五年。根据河南道教目前的状况和今后工作发展的需要，特制定 1996 年至 2010年的十五年工作规划，供第二届理事会议讨论、修改、补充，经第三届全省代表会议通过后执行。
　　( 一 ) 继续协助党和政府贯彻落实宗教政策，进一步合理安排道教活动场所。
　　在八年工作期间，开放了一批道教活动场所，基本上解决了有庙无道人的状况。但是，还有的宫观没有开放，今后需要努力工作的目标是 : 开放条件成熟的大中型宫观和城镇的庙宇。
　　( 二 ) 以两个宗教《法规》为法律依据，加强对开放条件成熟的、在历史上有名望的开放庙宇管理，完善开放宫观的登记、发证、年检工作。
　　两个宗教《法规》是道教界的法律依据。今后十五年开放宫观的管理组织在对本宫观的日常事务中，在对庙宇的基本建设，对外联系，维护道教界合法权益等工作方面要以两个宗教法规为准绳，在殿堂、消防、财务、吸收培养中青年道人等方面以中国道教协会、省道协的《道教活动场所管理办法》为依据，加强对宫观的管理，在管理好宫观的同时，宫观管理组织根据本宫观的基本情况制定出本宫观的《管理细则》若干条。完善管理体制。今后十五年，省道协开办形式多样的学习班，进一步使管理组织的成员年轻化、知识化，做到责任到人，制度健全，新老配合，团结互助。如殿堂布置庄严，庙院规划合理，支持开放庙宇发展林业、素餐业，经营小商品、道教特产品等第三产业，支持开放宫观在修建好宫观和保障道人生活的同时，要支援贫穷宫观，支持社会福利事业，支援灾区、希望工程等。
　　( 三 ) 培养和造就一批新生力量，作好道教文史工作。
　　青年道徒是河南道教的未来，也是河南道教的希望，今后十五年，省道协要化大力气做好这一基础工作，除了向外推荐学员外，还要大量吸收有文化知识、有宗教信仰的年轻道徒，开办多种形式的培训班、专修班、讲习班，如若条件成熟，成立河南道教学院。省道协要成立研究室，广泛征集道教史、志、著作、经典等文献资料，开展学术研究，出版《中州道教》杂志。组织人才，撰写《河南省道教志》，地方《道教志》、《宫观志》，搜集整理出版河南道教方面的善本书、孤本书。吸收外来人才，重视人才的使用力度，成立河南省道教协会经师团，实行向外宣传。
　　( 四 ) 加强自身建设，做好各项工作。
　　在今后十五年工作中，省道协力争迁郑州市办公，成立研究室、财务室、办公室、教务处、对外联络处，实行会长、副会长、秘书长工作负责制。今后十五年道教要换届两次，要召开八次理事会议，十六次会长办公会议，十六次常务理事会议。协助地方道教人士成立周口地区道教协会、三门峡市道教协会、开封市道教协会、鹤壁市、焦作市道教协会等道协组织，协助地方道教协会做好换届选举工作。加强对港澳台、韩国、新加坡等国家地区的道教界的联系工作，加强互访，加强了解，增进友谊，召开研讨会，促进中外道教文化的交流。进一步引进外资，把我省的开放宫观管理好，修建成为优美的环境，真正成为神仙福地，进一步提高我省道教对外宣传的力度，使国内外都能认识河南、了解河南。
( 河南省道协 )
