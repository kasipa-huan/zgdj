　　吕纯阳是道教内丹派倡导者。“钟吕金丹道”为道教全真派所继承。其拯救苦难、普度众生的仙迹，为海内外广大信众所崇奉。在道教神仙系统中，影响之大，无与伦比。然而，对他史无专传，有关他的生年记述，众说纷纭，故就有关记载加以考证。
　　吕洞宾姓吕名岩字洞宾号纯阳子。《宋史·陈抟传》仅云 : “关西逸人吕洞宾，有剑术，百徐岁而童颜，步履轻疾顷刻数百里，世以为神仙，皆数来抟斋中，人咸异之。”
　　宋·曾　撰《集仙传》云 : “吕岩字洞宾，唐礼部侍郎渭之后，唐末举进士不第”。
　　金·兴定六年袁用之撰《有唐纯阳真人祠堂记》云 : “真人讳岩字洞宾，道号纯阳子，世为河东永乐人。礼部侍郎渭即其祖也。当敬宗宝历元年 (825) 举进士甲科中选。未及调官，时季春出游沣水上，遇汉隐士钟离公……”
　　元·秦志安撰《金莲正宗记》云 : 吕祖生于“唐德宗兴元十四年丙子”。
　　按 : 唐德宗兴元年号只用一年即改元贞元。所以兴元年号无十四年。由兴元元年至贞元十二年丙子，乃为十三年，所谓十四年者乃虚数耳。
　　元·刘志玄等撰《金莲正宗仙源像传》云 :吕祖生于“唐德宗贞元丙子 (796)”。
　　元·全阳子赵道一撰《历世真仙体道通鉴》云 : “先生吕岩字洞宾号纯阳子。世传为东平人。一云西京河南府蒲坂县永乐镇人，即今河东河中府也。曾祖延之仕唐终浙东节度使。祖渭第进士，德宗贞元中官至礼部侍郎，晚为唐州刺史，有四子，曰温，字化光，官至衢州刺史 ; 二恭，岭南府判官 ; 曰俭，为御史 ; 曰让，历太子右庶子，或曰终于海州刺史。先生乃让之子也，贞元十二年丙子 (796) 四月十四日生于林檎树下。”
　　明·万历《续道藏·吕祖志·真人本传》云 : “贞元十四年四月十四日巳时生”。
　　明·王圻《续文献通考》云 : “贞元十四年四月十四日生”。
　　清·康熙年间徐道集《历代神仙通鉴》云 :“生于贞观十四年丙子”。
　　按 : 贞观十四年 ( 640) 是庚子而不是丙子。贞观恐是贞元之误。
　　刘体恕《吕祖全书·吕祖本传》云 : “吕祖名岩字洞宾，世为河中府永乐县人，曾祖延之，仕唐终河东节度使，祖渭终礼部侍郎，父让海州刺史。贞元十四年四月十四日巳时生。一云贞观丙午年 (646) 生。一云生于唐德宗贞元丙子”。
　　近年所出《藏外道书》所收《钟吕叙传》云 :“纯阳真人吕公 ( 云 ) ‘予大唐丙子四月十四日生'”。
　　根据以上所引文献，有年号与“甲子纪年”并记的 ; 有只用年号记的，有只用“甲子纪年”的，有年号与“甲子纪年”不符的 ; 只有《历世真仙体道通鉴》所记年号与“甲子纪年”相符。根据史书纪年惯例，往往是年号与“甲子”并记，且“甲子纪年”很少有误。据上引文献相互印证，吕纯阳生于唐德宗丙子年即贞元十二年 (796) 是可靠的。今年 (1996) 应是吕祖诞辰一千二百周年。
