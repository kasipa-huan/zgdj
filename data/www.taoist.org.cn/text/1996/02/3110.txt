　　性命双修之道，乃天人合一之绝学。其道易简，其理易明。惟非真履实践，究竟不知冷暖。故对丹道内景，不可不探求印证之。惜诸丹经，历来略谈内景，令人感到道海茫茫，无从取证，甚至有望而却步、退作他求者，此“道不轻传”之旧弊也。仆本不敏，兹录丹道内景十八则，汇以成文，以与道友共参焉。
一、虚室生白
　　丹经曰 : “上德无为，不以察求。”意谓童子修真，不用有为之法刻意追求，只学清静无为之法，便可深入道境，了证真源。
　　童年之体，因其性灵初自虚空中来，尚未丧失本来之天真自我。若能澄心定意，于无念中自可臻入道境。其时周身光明，虚白亮耀，内外照彻，浩渺无际。一切思虑念想均悉销溶，万事万物归于无形无象之中。独有一性灵光，不知所之，瞬息之间，脱化周身。一身俱化，而为光气，只见光即为身，身亦为光。此时周身与大化合为一体，自性灵觉成为主人矣。得此境界，端有无比之清净莹澈，洒脱舒畅，妙不可言。
　　童年出现的这一现象，当系童子天真，无欲无累，易入道境。《庄子》云 : “虚室生白”，即此境界。
　　其时童真，非必有心求道，未知“道”为何物。但其心境，当与深入法海、博通道要之大道高真，一般干净无异，所谓“天下无二道”是也。入此层次，智慧乃生，对于童年智慧开发，有着不可思议之效力。
　　上述内景，非有心为之而可得。一旦入此妙景，定然数度证验。纵是睡卧之中，恍惚即现。由其似梦非梦，似觉非觉，正合童子纯然之天真本性，于是心与道合。其时虽不知言“道”，但是已属道中境界，自性与道性为一，又何必定要知其为“道”耶 ? 《庄子》曰：“目击而道存矣，亦不可以容声矣。”是得其“实”而未知其“名”，妙在无后天“强名”之搅扰。《老子》曰 : “同于道者，道亦得之。”《清静经》曰 : “虽名得道，实无所得。”验之于童真境界，如车合辙，其义自见。而且对于以后丹修，无意之中已经获得了坚深根基。
二、性光圆满
　　全真祖师王重阳所倡北宗性命双修之法，讲究性功为先，以性功入手，且以性功了手。性功之重要，犹如夜行有灯，行去无差，否则性功差迟，便似盲人瞎马，难以进步。故此性功层次之修证，直接关联命功之成败，同时也决定了丹功全程之安危，乃丹道之总枢。性功澄彻，则命功水到渠成，纯出天然，不假人为。往古今来多少修真之士，都因性功失误，导致迟滞、退步，甚至炉倾鼎覆，致有前人“修道者如牛毛，得道者如麟角”之叹 !
　　若在童子之体，性命无亏，从性地下功，则易见光。要在夜深人静之时，塞兑垂帘，盘膝坐起，两手抱诀，收心止念，入静聚性。先由身心不动，渐入忘我境界，于是空空洞洞，杳冥恍惚，混沌之中，玄之又玄。忽焉性光发现，圆似满月，聚而复散，散而复聚。光至圆满，皆可印证，圆者状其形，满者无瑕疵。王重阳祖师当初教导马丹阳、孙不二夫妇，谓为“圆陀陀，光灼灼”，即是性光圆满之义，必须实修可知。既入此境，善于保持，则性光常见，而心地常明。功修亦随之而进，断无魔境干扰。
三、性住炁自回
　　性功修持圆满，随之而来便是命功上的一系列进步。性功即精神、心、意、神、德之修炼，命功即肉体、肾、气、形、功之修炼。性与命原不可分，性功圆满，自然命功精进。既能坚持“行立坐卧，不离这个”，念兹在兹，勤而行之，一念纯真，内外贞白，若在童体，自可以性摄命，神返炁回。其间内景效验，自有兆象，试列三条于下：
　　1. 阴　至丹田一路，似有一条虚线相通，平常之间，不时有炁流沿之上升，如同水中冒起的小气泡一般。
　　2. 熟卧之际，炁盈欲解，才撼阳关，猛然倒回，一路抖缩，如同传电，炁流激荡，妙不可言。仍在卧中，醒后自知。
　　3. 平时无意之间，忽焉肾根回缩，连连数度，类似枪膛之伸缩。非为外动，其形依然，内有机关牵动，其势不在人为。此时身心自静，不由自主，但顺其自然，感觉亦妙。
　　上述效验，皆可印证。炁动之时，必行阳关，若在人道，本为常理。道家修炼，此处正是玄机，掌握得当，自有妙境。丹道认为精原为炁， 炁化成精，由此分为清水源与浊水源，而有仙凡之别。 炁，乃无形清阳之炁，属于先天，又名元炁或元精，丹经谓之“清水源”，因之可成仙道 ; 精，乃有形浊阴之质，落入后天，又名阴精或浊精，丹经谓之浊水源，因之可成人道。中老年已漏之体，其炁成形为精，精走阳关为熟路，已入后天，故须勒之以采炼，从后天提取先天，要知后天乃先天所化，后天之中亦有先天也，其理在此。青少年童真之体，其精未形为炁， 炁归元海亦熟路，本属先天，惟要修性以保炁，以虚无合先天，如此先天自返虚无，自然再无堕入后天之忧，其义甚显。二者用功之不同，可谓径渭分明。当然根据各人素质，二者并不绝对。
　　尤其对于纯乾未破、命宝完固的童体来说，性功之修炼极为关键，要知炼性即保炁也，即保自身之道也。纯阳吕祖曰 : “不迷性自住，性住炁自回，炁回丹自结”， “自住”、“自回”、“自结”，皆系于性功之“不迷” !
四、还精补脑
　　丹书云 : “若要不老，还精补脑。”实指中枢神经之炁充盈流通，乃是人体养生长寿之保障。还精补脑，是道家修炼之所长，道教主张修今世，肉体成仙。这步工夫，因为青少年和中老年体质不同，所用方法也不尽相同。中老年人精炁已衰，脑髓转枯，先须炼精化炁, 炁满自可补脑填髓，恢复青春活力，返还乾健之体。青少年未婚者则不然，因其阳炁充盈，毋须炼精化炁，只须存神养性，降下心火，熏蒸肾水，促使肾炁上腾，飞上泥丸，自可填髓补脑。
　　此中证验，皆有真景。似上行功，月余之间，丹田之炁充实温暖，津液愈加甘美精醇。做功之际，神息归根，身心两忘，渐而周身太虚，浑若无物。忽焉甘露自上而降，由泥丸流出天池，叠叠滚落而下，其露甚暖，其势甚冲，不由自主，引颈吞之，如同雀卵，甘甜软美。随之一排暖流，由巅顶布向脚心，融畅之炁，传遍周身毛孔。其苏绵美畅之情景，妙难尽述。
　　自上所降之甘露，非常人之唾液，乃由先天真炁资生而来，实为长生大药。丹经又谓之“神水”，曹文逸真人《灵源大道歌》云 : “神水难言识者稀，资生一节由真炁。”此神水为先天真炁所生，非得真炁充足，上升脑府，凝化为露，方有还精补脑之效。以此神水，灌溉脏腑，坚固形体，可以接命延龄，保持青春，胜过世间药物。道家把“药”字写成“　”——“自家水”，即寓义于此也。最好的药物在人身上，最大的病患在人心中，倘能炼心合道，自得长生妙“　”。
五、炁足开关
　　上述神水降落，已是真炁充足，炁足自动，皆在自然而然。数日之后，炁可上拱脊背，虚极静笃之中，入于混沌。真炁大动，不由自主，忽焉背部发紧，真炁上涌，一路轰鸣，直冲夹脊，腾发之势，畅通无碍。复上奔玉枕，撞之难开。玉枕又称“铁关”，为三关之中最紧密处。其炁乖乖缩回，蓄势待发，似乎消失，隐于无形。稍停复显，吼叫上冲，再撞玉枕，一撞而通。升上昆仑，轰鸣如雷，脑门发紧，震动甚巨，舌拄上腭，接下此炁。但觉流粗壮，直穿脊髓，所到之处拱动剧烈，犹如钻机通过一般。丹经所述“大药过关”时“六根震动”之景，即丹田火炽、两肾汤煎、眼吐金光、耳后风生、身涌鼻搐、脑后鹫鸣等“六候”，此时均有发生。当此景象之际，周身四肢如石，六神无主，完全无力抗拒，丝毫不能主事。所以然者，炁足开关之景，乃是先天真炁之发动，由脊髓而通关，打开任督二脉。丹道又谓之“小周天”。与彼气功家以后天识神导引之皮下气通，断然不可同日而语。
　　先天真炁之发动，必得到达先天境界而后可，这就需要性功圆满，合乎先天，然后才是元神主事，识神退位，身心无主炁是主，神行而非我行。丹经有云 : “十分好汉，到此无一分主张。”若非亲身体验，难解其中滋味，盖有主张，即入后天也。此时一念不动，识神全泯，倘若炼意不净，再动念心，先天一堕后天，真炁便成浊质矣，所谓“一念之差，天堂地狱”。吕祖《百字碑》云 : “不迷性自住”，此中所言之“不迷”，即不可迷恋于外物，扰性乱心，惑乱元神，故必性功到处，一丝不挂，万缘俱空，而后方能性住，性住方得命功有成，性命双修之功，方得精进无差。
　　《玉皇心印妙经》曰 : “百日功灵”，意谓需要一百日，此则行功两月，便得开关之景。总结速效之原因 : 一者在于童体炁足，二者在于性功精纯。此时用功，所读丹经尚少，在脑中尚未形成系统的丹道观念，仅仅得到一点下手功夫，便去兴工实践，故此内景发生之后，还不明白怎么回事，后来明白此属“炁足开关”之景。正因为脑中尚未形成系统的丹道观念，头脑干净，没有负担，无心而求之，合乎先天，反而易得，得之亦真。相反，倘若事先知道有此内景，先有模式，充塞头脑，有心而求之，落入后天，反而难得，得之易假。所谓求之而不得，不求而自得，相信修证内丹的道友，也有类似的感受。
六、阳炁开八脉
　　任督二脉打通之后，其他八脉之阳维、阴维、阳　、阴　、冲、带等脉，渐次冲开，遂至八脉俱通。皆从自然无为中来，丝毫不假后天意念。每逢自身阳炁旺足之时，乃入混沌，其炁便发，自动冲脉。炁动之景，不可抗拒。冲开之后，归于平常。
　　任督二脉，为全身百脉之总纲，任督打通，百脉皆通。张紫阳《八脉经》云 : “凡人有此八脉，俱属阴神，闭而不开，惟神仙以阳炁冲开，故能得道。”又云 : “八脉者，先天大道之根，一炁之祖。”盖常人所行者十二经络，修士所循者奇经八脉，此人道、仙道径路之不同也。
七、男子修成不漏精
　　此后每遇阳炁盛满之候，活子时出现，混沌之中，阳炁发动，不行阳关，自升昆仑。皆自然而然，毋须人为追求。其中景象颇多，兹举一例 : 某年月日晚，修士静定之际，浑然无我。忽焉真炁腾发，直撞漕溪，犹如飞龙，吼吟而起，隆隆作响，升上巅顶。与此同时，忽现三颗光珠，接连穿过右、左双眼，颜色内黄外红，晶光莹澈，色彩夺目。但觉此炁升腾不已，飞上昆仑。其时身心无主，从其自然而已。旋顷自停，复归于无物。平时睡卧，内炁自交，阳升之际，时至神知，炁动意随，即有真炁隆隆，飞升巅顶，两耳轰鸣，如雷行天。从此自无道解之患，而独得清修之乐。
　　人身内部之阳炁，总在不断地生发，以维持人体之生机。生发到一定程度，盈满而溢，就会出现动机，丹家谓之活子时。红日初升之际，未露太阳，先见阳光，人体之阳生亦然，故阳炁发动伴有光的出现。至于此三颗光珠之景象，非在拟议。
　　关于采取活子时，此中又有中老年与青少年用功之不同：中老年已漏之体，活子时到来，易行人道而走失，惟用有为之法将其勒回，加以采炼封止之功，去矿留金，若用无为之法，必走熟路而漏失。青少年未破之体，活子时到来，外行阳关非熟路，八脉已通无阴气阻塞，清阳之炁自然归内，沿督上升，不必用后天有为之法也，所谓“不采之采，乃是真采” ( 此乃重阳祖师丹诀 ) 。从此之后，炁行任督，阳关止闭，而无道解之患。《黄庭经讲义》引曰 : “男子修成不漏精”是也，全真北宗清修之真义在此。惟须平时注意保持养性，神合先天，方保无失，大道就在平常日用之间也。
　　中老年人修道，需要炼精化炁，补满三宝，筑起丹基，方能返还童体，与青少年纯乾之体相匹。青少年命宝未丧，三元丰满，本身素质已较中老年人大为优厚，倘知下手兴工，可以省去炼精化炁一着，而直接从炼炁化神入手，获得较高层次的修为。道之易求，可知矣，年轻道友岂不悟耶 !
八、大道甚夷
　　修丹途中，可能会遇到一些令人费解的奇怪现象。有的属于道之变化，出现神异 ; 有的属于意识恍惚，便成幻觉 ; 有的属于境界不同，异于凡间 ; 有的属于人心未定，念动生魔 ; 有的属于炼意不净，识神作祟……种种不一。
　　例如，道友修习站功，每日坚持四个小时，从不间断。以之印证内修，感觉相符。六个月之后，某晚八点，身在观内楼上习站，面向东方，似看非看，独立守神，肌肉若一，虚灵挺拔，松和自然，津液时涌，咽纳不已。过了约有一个小时，东方慢慢冒出光华，知其乃为新月初升，并未在意。但见此光升起，并非月亮，乃是好几个碎块，大小不一，悬在空中，极为明亮。依旧独立，视同不见。碎块在相对静止中慢慢上升，却又变成一颗亮星，继续上升，旁边又出现一颗小星，亮度稍弱。此时怀疑是否出现幻觉，用手揉揉眼睛，再看还是。仍然独立不改，静以处之。只见两颗星慢慢上升，复变为长方形的块状物，很亮。接着上升，才是一轮月牙。既在平常看到，仍以平常视之，依然独立下去，直至四个小时之后收功。此景出现之后三日，更有不可思议之景象发生，但修道非淑诡幻怪之事，兹略不谈。
　　举出这样一件实例，并非说明道人的功修如何特别，更不是为了惊世骇俗，乃是因事说法。修炼之中的异常现象，只要不是肉体感到痛苦的，一般来说都不是坏事，可以不去管他。这是唯一的标准，不必多加联想。同时这也正是验证自己道心的试金石，《老子》曰 : “大道本夷”，只可视为平常，不可以为怪异。如此，自可增长道力，否则的话，就是桎梏，就是魔障 ! 只可借“假”修“真”，岂可弄“假”成“真” ! 张紫阳《悟真篇》云 : “顶后有光犹是幻，云生足下未为仙”是也。真假之辨，幻仙之别，判乎明矣。
九、大妙中黄贵
　　中黄，又称黄中、黄道、中黄正脉、玄窍、总持门等。非医家之冲脉，亦非气功家之中脉。其脉处于人体之正中，内系五脏，旁通五心，达于四肢。
　　炁通中黄之景，非初学可测，八脉通后，方开此脉。戊辰年 正月十九日 ( 丘祖圣诞日 ) ，道友在崂山太清宫，临窗养静，浑然无我。先觉顶额如有蚁爬，似知非知之中，忽焉脑后如雷轰然，一股粗劲炁流，自天而降，直冲黄道，透体下行，全身震颤，不由自主。其炁一路啸声，直达海底，流溢腾发，散向四肢。但觉浑身舒畅，妙不可言，复归于混沌，其后面部犹热。
　　中黄一道，因其联络七门，即天门泥丸、地门尾闾、中门夹脊、前门明堂、后门玉枕、楼门咽喉、房门绛宫等，五脏六腑皆背禀焉，故此修炼家多重此脉，且视为不传之秘。丹经之中多有隐指，往往为人忽视。如《周易参同契》 : “黄中渐通理，润泽达肌肤。”《太上黄庭内景经》 : “中有真人巾金巾，负甲持符开七门( 即上述七门 ) ，此非枝叶却是根，昼夜思之可长存。”吕祖《敲爻歌》 : “玉炉之中文火烁，十二时中惟守一，此时黄道会阴阳，三性元宫无漏泄。”张紫阳《悟真篇》 : “先把乾坤为鼎器，次传乌兔药来烹，既驱二物归黄道，争得金丹不解生。”丘祖《龙门派薪传百字颂》更是一语破的 : “大妙中黄贵。”《丘祖秘传大丹直指》曰 :“惟此一窍，乃内呼吸之根蒂，先天元炁实游于此，天地正炁实从此入。”
　　此脉修通，心肾二炁即无间隔，浑身关窍俱开，一炁遍达周身，且可内外相通。“先天一炁，自虚无中来”，即在此一道路。丹道又谓之“大周天”。
　　以上所谈，主要是积精累炁、小大周天之景，尚有天人合一、道法自然之景象，于后详述。
