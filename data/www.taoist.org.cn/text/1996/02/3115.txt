　　钟吕坪系太白国家森林公园七景区。位于陕西眉县东南五十华里的汤峪口太白国家森林公园西端八公里 处。东距西安古城一百一十公里，西距宝鸡山城九十公里 。
　　坪高海拔一千二百米，广袤各为三华里。
　　钟离坪原自东华 ( 东华帝君王玄甫 ) 演教传授钟吕二仙，故有仙境之称，闻名遐迩。
　　据传，正阳 ( 钟离权道号 ) 山有坪三处，乃昔汉钟离辅政吕洞宾得道之所，故名钟离坪。
　　据明代《眉县志》记载 : 钟吕坪“山不甚高而顶宽平，俗传钟离权、吕洞宾、韩湘子得道成仙于此”，故此易名“钟吕坪”。
　　历代骚人墨客、高士乡贤曾到此寻幽访奇，遨游小憩。
　　民间传说韩愈从此登上太白山，被玉皇大帝封为太白山社神。
　　明末清初，关中有三位著名学者 : 周至李禹 ( 中孚 ) 、眉县李柏 ( 雪木 ) 、富平李因笃( 子德或天生 ) ，世称“关中三李”。李雪木三登钟吕坪小憩，并赋诗三首曰 :
武陵久已绝人烟，惟许渔郎一往还。
二漳中分琼玉观，三泉侧涌紫金天。
长安日远浮云蔽，阊阖门高白云连。
桃花流水依旧在，不知何处遇秦贤。
 
一径通幽岛，山花处处芬。
穿岚鸟影没，傍岸水声闻。
丹灶明松火，菜畦绕鹿群。
道人横铁笛，吹断岭头云。
 
万木阴台榭，十年卧雪霜。
画眉双绕足，燕子一楼颡。
衣惹烟霞气，剑喷斗牛芒。
重来茹蕨处，可是旧云房。
 
　　雪木先生携眷隐居太白山中的大雪崖岩洞多年，常穿着道家服饰，安贫乐道，坚贞不屈的性格一如继往。宁为山林雅士，不为满清效劳，实乃秦中高士，先生所著《槲叶集》尤为后世所推崇。
　　钟吕坪顶连太白，足蹬渭水，了望云城，俯视秦川，左右两峰，形似二龙，钟吕坪环抱其中。
　　钟吕坪分东、中、西三坪。
　　东坪坪巅坦荡，突兀特峙，王母紫府。南跳诸峰峻峙，则重峦叠嶂，梦壑枝峰，青葱螺黛，翠连千山 ; 北眺则渭流如带，锦波雪浪，浮光耀晃，朗川原极目东西，则绿野平畴，村落鳞集，星罗棋布，郁郁苍苍，行连深岩，遂谷而岚光霞霭，变幻迷目，亦一隅胜景也。山行六七里，渐闻水声，潺潺而出，出于两峰之间者清泉也。峰回路转，有竹蕃然临于泉上者东坪也。
　　绿竹漪漪环绕于周围，若夫日出而林霏开，云归而岩瞑穴，晦明变化者山间朝暮也。牡丹花而尤香，佳木秀而繁荫，风霜高洁，水落而石出者，山间之四时也。朝日往暮而归，四时之景不同，乐亦无穷也。
　　东坪有韩湘子亲手栽植的“八爪盘龙柏”( 此树上为棉柏，下为刺柏，属全国稀有珍贵佳木 ) ，又名“湘子床”，形似雨伞，别具一格，为韩湘子宿卧之处。
　　中坪原有“二仙授书台”、“湘子藏经柜”，湘子洞旁至今尚有的“玉皇泉”和东坪“华泉”，相传为东华演教时道士所挖。味甘如醴，源远流长。老君洞的“镜泉”，碧泉玉液，潺潺不息。
　　中坪玉皇殿后有所谓湘子“早种晚收的三分田”，每日收获粮食可供湘子一日所食，日种日收，周而复始，取之不尽，用之不竭。
　　殿后还有吕祖由山东蓬莱运来亲手载植的绕果树，树高三丈，环抱不拢。相传五百年开花结果一次，花为黑色，果型条状。“文革”动乱年间遭砍。今逢盛世，枯木回春。
　　西坪宽敞，高峰峻峙，面崖靠山，塑钟吕二公像，碧泉接涧，橡林阴森，幽曲茂密，硖石堆云，槲叶秋红满坡，沟下清泉潺潺。坪之山腰橡林之中，有一巨石，高约一丈，人称湘子石，传说韩湘子成仙于此，又名湘子“坐功石”，俗传“卧眺长安”处。
　　正阳山钟吕坪，久有清流激湍，映带左右，诚然佳境。故眉县八景有“秦霭翠眉，雨云玉带”之佳称。清代举人张执中在《题眉八景图》中有诗道 : “川原雨霁月新升，烟村迷离水气蒸，恰似美人才浴罢，晚妆翠黛拂眉棱”。
　　冬寒季节，中坪之侧的峡谷之水，在飞流直下时凝结成冰柱，长达数里，直垂山下，故有“玉柱擎三坪，玉屏映终南”之美誉。
　　尝思有开于先者乃有继后，有作于古者乃有述于今，凡事类然，况在八仙了道凭依佳地。
　　古代营造有斗姥宫、王母宫、老君洞、吕祖洞、药王洞、太白殿、八仙殿、玉皇殿、救苦殿、山神殿、关帝庙、龙王庙等道宫殿宇。早在兵荒马乱年间，毁于兵燹。清咸丰七年、八年和光绪十八年，山下六堡整修正阳山东坪前殿三间，内供 关圣帝君、送子娘娘菩萨塑像，王母殿三间。西坪主持道士吴元真、张元廷、陈亨林重修了斗姥宫。
　　东、中、西坪重建庙殿，民国时期年馑饥荒，贼寇作乱，残遭覆圮，香烟停熄。八十年代，党和政府拨乱反正，贯彻宗教信仰自由政策。道士主持宫观，信众好善乐施，聚米积资，复修庙殿。
　　东坪与中坪之间原有玄珠古洞八处，相传为“八仙”所居，五百年洞开一回，紫气腾腾，祥光万道。
　　曾有龙门第二十二代弟子赵理清兴建老君洞于此，年久失修，难容站立，幸有龙门第三十代弟子王兴理 ( 政协宝鸡市委员会委员、政协眉县委员会委员 ) ，龙门第三十一代弟子牛世仙不忍坐视，苦自采药收入，更得信众好善乐施，于1985 年主持复修仙洞三孔，复名“老君洞”，中洞老君圣颜，左侧吕仙，右侧药王。
　　西坪龙门第二十一代弟子刘至鹏、龙门弟子张崇祥复修起太白殿、斗姥宫，增修西厢房六间。
　　中坪俗徒王桂莲主持信众乐助复修了玉皇殿。
　　东坪主持人张崇玄、坤道惠理真及罗坪先后相继复修了王母宫、玉皇阁、湘子庙，增修厢房九间，并由山神沟压管引水二华里至东坪，解决了数百年来香客游人饮水之困。
　　钟吕坪营造之殿宇庙洞，巧妙的利用自然地形地貌，以山壮其势，以林掩其幽，以水秀其姿，与大自然环境和谐地融为一体。景色奇特，引人入胜，到此登游，顿觉有超尘脱俗处身世外之感。
　　视野无堤，心旷神怡，令人陶醉，流连忘返。
