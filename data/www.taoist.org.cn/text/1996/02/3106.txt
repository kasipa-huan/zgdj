一、保护动物的规定
　　道教有“三戒”、“五戒”、“九戒”、“十戒”、“二十七戒”、“一百八十戒”乃至“三百大戒”，内容庞杂，牵涉广泛，但无不以“戒杀生”为主要大戒。下面略举几例 :
　　六朝时期道经《太上洞玄灵宝智慧定志通微经》中“十戒”的第一戒，就是“不杀，当念众生”。②刘宋陆修静在其《洞玄灵宝斋说光烛戒罚灯祝愿仪》中，把“守仁不杀，悯济群生，慈爱广救，润及一切”作为“十戒”的第二戒。③其所撰《受持八戒斋文》，更以“不得杀生以自活”作为八戒清斋的第一戒。④此外，《升玄内教经》卷九所述“升玄九戒”中的第四戒说 : “手不得杀害众生，　行蠕动含血之属，皆不得杀。”⑤《老君说一百八十戒》中的第九十五条规定 : “不得冬天发掘地中蛰藏虫物。”第九十七条规定 : “不得妄上树探巢破卵。”第九十八条规定 : “不得笼罩鸟兽。”⑥可见戒律的制定者对保护动物想得非常周到。
　　除了“不杀生”外，道教还反对惊吓、虐待动物。《老君说一百八十戒》中的第一百三十二条规定 : “不得惊鸟兽。”⑦《三百大戒》中亦规定 : “不得惊但鸟兽，蹴以穷地。”⑧此外，《老君说一百八十戒》中的第四十九条规定 : “不得以足踏六畜。”第一百二十九条规定 : “不得妄鞭打六畜群众。”⑨
　　道教并不以消极地“不伤害”动物为满足，它还呼吁积极救助动物。《六度生戒》中的第三条说 : “含血之类，有急投人，能为开度，济其死厄，见 ( 同“现” ) 世康强，不遭横恶。”第四条说 : “施惠鸟兽有生之类，割口 ( 省下自己的口粮 ) 饲之，无所爱惜，世世饱满，常在福地。”第五条说 : “度诸蠢动一切众生，咸使成就，无有天伤，见 ( 同“现” ) 世兴盛，不履众横。”第六条说 : “常行慈心，愍济一切，放生度死，其功甚重，令人见 ( 同“现”) 世居危得安，居疾得康，居贫得富，举向从心。”⑩总之，救助动物者必得福报。
二、保护植物的规定
　　《老君说一百八十戒》中的第十四条说 :“不得焚烧野田山林。”第十八条说 : “不得妄伐树木。”第十九条说 : “不得妄摘草花。” ⑾《妙林经二十七戒》中也有 : “不得烧野山林”的规定。⑿ 《三百大戒》中同样规定“不得以火烧田野山林”，“不得无故摘众草之花”，“不得无故伐树木。⒀
三、保护土地及水资源的规定
　　《老君说一百八十戒》中的第三十六条说 :“不得以毒药投渊池江海中。”第四十八条说 :“不得妄凿地毁山川。”第五十三条说 : “不得竭水泽。”第一百条说 : “不得以秽污之物投井中。”和一百三十四条说 ; “不得妄开决陂湖。”⒁《三百大戒》中也规定 : “不得以毒药投渊池江海中”，“不得竭陂池”，“不得塞井及沟池。”⒂顺便说一句，早在《太平经》中就已反对人们乱穿凿土地，称之为伤害大地母亲的恶劣行为。
　　把道教对待自然中界( 包括生物 ) 的态度与本世纪中叶兴起的生态伦理学的观点作一番比较，也许就能清楚地看出道教上述思想的重要价值。生态伦理学的创始人之一，英国科学家莱奥波尔德 (AldoLeopold) 在《大地伦理学》一书中提出了生态伦理学的几个重要观点 :（1） 人类并非自然界的主人、统治者，而是自然界中极普通的一员 ;(2) 必须树立生态整体思想，只追求经济功利而不关心生态平衡的做法是极端错误的 ;（3）要把权利这一概念从人类延伸到自然界的一切实体和过程。花草树木、飞禽走兽都有生存繁殖的权利，人类没有权利去践踏它们的这些权利。⒃通过比较我们就会发现，道教的主张与生态伦理学在基本精神方面是颇有一致之处的。道教的“天人合一”观念正是一种生态整体思想，它反对污染土地和水资料，更是保护生态环境的具体规定。而它反对杀害动物、虐待动物，反对破坏花草树木，正是尊重自然界中所有生物的生存权的表现。我们还必须注意到这样一个事实 : 在道教产生的时代乃至整个中世纪，自然环境的破坏问题并不是问题，因此很少引起思想家甚至自然科学家的关注。然而，在道教中却有着如此成熟的生态意识，这不能不让人为之惊叹。
