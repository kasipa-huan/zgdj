　　北京白云观早坛功课经、晚坛功课经为住观道士每日早晚，上殿诵唱的必修课程。是修持练性，步入玄门所必学的功课。是敬神祈福，解厄保安的每日必做的科仪道场。《功课经》序记有 : “功课者，课功也。课自己之功者，修自身之道也。”“金书玉笈，为入道之门墙，讽经诵咒，乃修仙之径路。《玉皇心印妙经》记有“诵持万遍，妙理自明。”
　　北京白云观为全真派的祖庭，有“天下第一丛林”之称。金代王重阳祖师 ( 原名中孚，字允卿，宋徽宗政和二年 ( 公元 1112 年 ) 生于终南刘蒋村。 ) 创立了全真道，建立了丛林制度，规定了日诵功课。《全真教祖碑》记载 ; “先生劝人诵《道德经》、《清静经》、《般若心经》、《孝经》，云可以修正。”上殿诵经时间为 : 早功课在每日天刚亮之时，早饭以前唪诵。晚功课在晚饭之后，天刚黑一点时唪诵。由于季节的变化，太阳日出日落时间的改变，而上殿诵功课时间也略有改变。逢阴历天干为“戊”之日为“鬼哭日”不能上殿诵经。东晋葛洪 ( 公元 284-364 年 )《抱朴子》记有 : “六戊者，是戊子、戊戌、戊午、戊申、戊寅、戊辰是也。此六日乃天地造化之期，独儒家之忌辰。天地逢戊则迁、出军逢戊则伤，蛇逢戊不进、燕逢戊不衔泥。”《女青律》云“若法官道士焚香诵经，不禁六戊，钟鼓齐鸣，进表上章、关奏天曹者罪加一倍，禁戊不犯者功德无量。”教规要求道众上殿前要 : 衣冠端正、斋食洁身、清心未一，进入心诚忘我的境界。《孟子·离娄》云 : “斋戒沐浴则可以事上帝。”进入殿堂后，道众先要烧香供水、礼拜祖师。完后再开始唪诵功课。
　　全国道教全真派所用早晚功课经的经文版本都是一样的，为《全真正韵》。早期为直诵，到北魏 ( 公元 420-534 年 ) 嵩山道士寇谦之吸收儒家有音乐伴奏的礼法仪式改为乐诵。《魏书·释老志》写有 : 嵩岳道上士，上谷寇谦之，立身直理，行合自然，才任规范，首处师位。太上老君授其“天师之位”，赐《云中音诵新科之诫》二十卷，号曰《前进言》。寇谦之所著《云中音诵新科之诫》中即有《华夏赞》、《步虚辞》的记载。
　　全国全真派所用音乐韵腔多为《全真正韵》亦称《十方韵》。在北京、陕西、甘肃、河南、山东、江苏、浙江、湖北、四川等十方丛林宫观大都用此韵。全真派中还有 : 东北韵、北京韵、崂山韵、子孙韵、武当韵、广成韵等地方性的经韵。而正一派的道韵受地区性和当地民间民俗音乐的影响，不同地区的道韵都有着明显的地方音乐的特性。
　　唪诵早晚功课的作用在于 :1. 设坛祭典、拜神祈祥为道教斋醮科仪的重要宗教活动。 2.道士修持必学功课，以求修身养性，达到清静无为，道法自然的上乘境界。 3. 祈祥延生，赐福消灾，解厄去难，普度幽魂。 4. 提高道士的道德修养，遵守道规戒律，敬神崇德，脱俗皈道。 5. 宣传道教，宏扬教义，感化民众。
一、早晚功课经音乐的特性
　　1. 宗教性 :
　　任何音乐都有着它的属性。道教音乐是为道教服务的，有着强烈的宗教色彩。每当经师身着法衣站在神像前习诵功课。道众以及观看的信民无一不被那神圣的气氛所感染。庄严的殿堂，道祖的神威，素雅的音乐，在经文和韵曲的导引下，使人松静自然，修德养性，脱离尘俗，情操得以升华。
　　由于道教教理所决定，所以早晚功课音乐具有以下特点 : ①严肃虔诚，功课经因是敬神修身的音乐，所以音乐中没有浮夸、华丽之旋律。音乐是庄严是发自内心的倾诉。②节奏平缓，起伏不大。音乐大为缓慢以使道众进入清静忘我的境界。③音乐典雅旋律优美动听。得道之人，道德高尚。所以道教音乐中不可能溶进尘乱之音。④宁静的音乐气氛以利于道众松心、松神、松身进入空静状态，达到修持的目的。⑤飘逸，静听功课经音乐使人心旷神怡进入虚无飘渺的境界。
　　2. 古代性 :
　　中国道教音乐已有一千五百多年历史。道教早期源于巫祝。《周礼·春官宗伯》 : “司巫掌群巫之政令，若国大旱则帅巫而午雩。”《尚书·伊训》 : “敢有恒舞于宫，酣歌于室，时谓巫风。”早在春秋战国时期的祭祀活动中已有祈神音乐出现。在东汉《太平经》写有“音声者，即是乐之语谈也。”“夫音，非空也，以致真事，以致虚实，以无形身召有形身之法也。夫乐乃以音响召事，比若人开口出声，有好有恶，善者致吉，恶者致凶。”“乐，小具小得其意者，以乐人 ; 中具中得其意者，以乐治 ; 上具上得其意者，以乐天地。”东晋葛洪《抱朴子·道意篇》写有“撞金伐革，讴歌踊跃。”北魏道士寇谦之改直诵为乐诵。在他的《云中音诵新科之诫》一书中已有《华夏赞》《步虚辞》的记载。金代王重阳创立道教全真派，建立了丛林制度，规定要日诵功课。
　　从以上记载不难看出在道教的宗教活动中与音乐的紧密联系。在春秋、战国的巫的祭祀中已有乐和舞，在北魏寇谦之时使用了诵唱的道曲。到金代王重阳时已将习诵功课规定为道士每日必修之功课。音乐更加强化了修持的作用。
　　北京白云观建于唐玄宗时，至今有一千二百多年的历史。身为道教全真龙门派的祖庭，全真道派，教规戒律严格，道众深居宫观，较少外出做道场。道场多在宫观内举行。斋醮科仪所用经韵由师父口传心授给徒弟，世代相传，延留至今，具有延续性。又加道韵为敬神和修持所用，是神圣、严肃的韵曲，是不能任意更改的，具有保守性。所以现在延用的白云观道教音乐具有古代性。
　　3. 宫廷性 :
　　历代皇室对宗教都极为重视，有的是重佛抑道，有的是重道抑佛。唐高宗时，据《册府元龟》记载 : “ ( 天宝十年 ) 四月，帝于内道场亲教诸道士步虚声韵。”《新唐书·礼乐志》记有 : “帝( 玄宗 ) 方寝喜神仙之事，诏道士司马承祯制《玄真道曲》，茅山道士李会元制《大罗天曲》，工部侍郎贺知章制《紫清上圣道曲》，太清宫成，太常卿韦绦制《景云》、《九真》、《紫极》、《小长寿》、《承天》、《顺天乐》六曲。”宋代斋醮科仪已十分盛行，日逐规范。据宋代张商英《金箓斋醮科仪序》中记有 : “斋醮之兴，……陆修静行之于前，杜光庭集之于后，神宗讲兴废典。”宋代《玉音法事》为最早的记录成谱的道教音乐文献。收集了宋真宗、徽宗时所用赞颂等道韵，用曲线符号记写成谱。明代也崇信道教，明成祖时用工尺谱记录成书《大明御制玄教乐章》，有道韵十四首。明嘉靖中江永年《茅山志后编》中记有皇室道场的规模。“唱念二十二名 : 其中知馨四名、止仪一名、表白四名、清道一名、宣读一名、词忏二名、引揖二名、手鼎二名、知钟一名、知鼓一名、侍职二名，内坛奏乐一十五名 : 其中云锣一名、笙四名、管二名、笛二名、札二名、板二名、鼓二名，外坛奏乐一十五名 : 其中云锣二名、笙二名、管二名、札二名、笛二名、板二名、鼓二名。”由以上记载不难看出历代皇室对道教崇奉的程度。皇宫家族举办道场规模的宏大，具有宫廷性的高雅道韵为皇家所赞赏。
　　4. 民间性 :
　　历代以来，道教都脱离不开民众基础。在敬神、修持的科仪中除了庄严、神圣的一面，为了宣传介绍道教，在斋醮科仪中又有它们民俗性一面。既要好看又要好听。所以在道场中也吸收了当时的一些好的表演形式，和流行于民间的为民众所喜爱的地方民间音乐。在唐末五代时，张若海的《玄坛刊谱论》中描述道教音乐“广陈杂乐，巴歌渝舞，悉参其间。”现使用的北京白云观早晚坛功课经音乐中民间民俗性还不为明显。而北京白云观《萨祖铁罐施食》下卷中的《叹文》吸收了浙江地方戏曲的音调。接近“施食”尾声的《二十四类》中则吸收了民歌音调如“凤阳花鼓”等。

三、早晚功课经的养生作用
　　道教提倡炼丹修身。道士日常的修身是与宗教活动紧密的结合一起进行的。笔者认为早晚功课经为上乘的养生功法。而在道教内部则是强调它的宗教作用，健身作用为副。气功不外乎通过动功静功的锻炼，调神、调心、调息、调身而健身。而气功高手则进一步入静入定，达到松静自然上乘境界。
　　早晚功课经的养生功法程序 :
　　1. 特殊的神圣、宁静的气场 : 举办功课科仪的殿堂，为历代道众所举办道场，修持的场所。它的环境与气感是其它练功场所无法相比的。
　　2. 《澄清韵》《步虚韵》的导入 : 平缓、优雅的音乐，使人进入“天无氛秽，地无妖魔。冥慧洞清，大量玄玄。” ( 摘自《澄清韵》 ) 的境界。
　　3. 放松 : 通过诵念《净心神咒》、《净口神咒》使心、神、身得以放松。开始功法的习练。
　　4. 站桩 : 诵念《清静经》《玉皇心印妙经》《救苦拔罪妙经》..…等。练息入静，进入“恍恍惚惚”的境界。
　　5. 动功 : 站桩与跪拜的结合。诵念唱《宝诰》多次下跪，礼拜祖师诸神。
　　6. 诵念《神咒》《宝诰》通神祈福。此法为气功的特殊功法。
　　7. 《三皈依》的导出 : 祥和的音乐使人回归世人大自然的世界。功法结束。
　　早晚功课习练时间短者半小时，长者一个小时余。在习练中，要站、跪拜、念、唱。可见功法的运动量之大，形式的多样。习练此功可达到 : “上药三品，神与气精。恍恍惚惚，杳杳冥冥。存无守有，顷刻而成。回风混合，百日功灵。” ( 摘自《玉皇心印妙经》) “杳杳冥冥清静道，昏昏默默太虚空。体性湛然无所位，色心都寂一真空。”( 摘自《升天得道真经》 ) “腹养黄芽，鼎飞白雪。体静潜而藏天地，心寂默而转乾坤。” ( 摘自《邱祖宝诰》 ) 所以有的老道长面润而体健，这也与常年修持功课经有关。
　　早晚功课经为认识研究道教的重要文献，而功课经的音乐为古代音乐、传统音乐、宗教音乐的一个重要组成部分。为有这样一颗光辉夺目的明珠而自豪。
