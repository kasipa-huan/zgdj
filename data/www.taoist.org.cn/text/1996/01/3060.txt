　　《太一宫记》原载清王昶编《金石萃编》卷一二三。今人陈垣先生编《道家金石略》亦收入，列在宋代碑刻类。据《金石萃编》所言，碑高六尺六寸，广三尺一寸，二十九行，行六十四字，正书。碑额题 : “华山陈抟撰。乡贡进士宋复书。乡贡进士陈羲箓额。”碑尾题 : “大宋建隆三年岁次壬戌正月一日住持焚修上清三景大洞法师崇玄大师赐紫道士王若海立石。大宋政和元年岁次辛卯庚寅朔十月乙亥太一观主兼太一湫事赐紫道士王希美重立石。”其后尚载有刘希文、皇甫希旦、秘希孟等十三名道士之称。碑文内容颇为丰富，约 1600 余字，记述了太一宫的由来及演变，尤其是题名“华山陈抟撰”，更是引人注目。
　　然编者王昶认为，旧时长安有元君庙、泰一坛，咸宁有太乙元君行宫，但均“非碑所称太一宫” ; 又谓碑文言后晋桑中令“镇雍时岁旱祷雨之事，盖皆荒邈难稽矣。”他的结论是 : “文多夸诞，事无可征，似非希夷手笔，恐亦道流所托也。”
　　那么，此碑记究竞真伪如何 ? 是否陈抟所撰 ? 下面试加辨析，以求正于学人大德。
一、太一宫的历史考辨
　　顾名思义，太一宫是专祭太一尊神的庙宇。
　　武帝元光二年 ( 前 133 年 ) ，依方士缪忌言，朝庭正式设立太一祠。《史记·考武本纪》曰 :“毫人缪忌，奏祠太一方。曰 : 天神贵者太一，太一佐曰五帝。古者天子以春秋祭太一东南郊，用太牢具，七日，为坛开八通之鬼道。于是天子令太祝立其祠长安东南郊，常奉祠如忌方。”
　　元鼎五年 ( 前 112 年 ) ，武帝又听方士公孙卿言，于雍县五　处增设太一　，完善五帝陪祭太一制度。元马端临《文献通考》卷六九载 :“上遂郊雍，至陇西，登崆峒，幸甘泉，令祠官宽舒等具太一祠坛。”武帝亲临祠坛，口诵赞词 :“天始以宝鼎，神策授皇帝，朔而又朔，终而复始，皇帝敬拜，见焉而衣。”时“有司云祠上有光，公卿言皇帝始郊，见太一云阳。有司奉　玉嘉牲荐享。是夜有美光，及昼黄气上属天。太史令谈、祠官宽舒等曰 ; 神灵之休，佑福兆祥，宜因此地光域，立太　坛以明应。”武帝准奏，即于雍郊设立太一　。令太祝领秋及腊闲祠，天子三年一祭。
　　至元封二年 ( 前 109 年 ) ，武帝亲行雍郊，主持太一祭典，其赞词曰 : “德星昭衍，厥维休祥，寿星仍出，渊耀光明，信星昭见，皇帝敬拜，太祝之享。”
　　据上所言，武帝时所建太一祠应有二座。其一在长安东南郊，即后来的太一坛。《古今图书集成·职方典》卷 504 载 : “太一坛，元光二年从毫人缪忌奏，立太一坛於长安东南郊。”其二在雍县东南二十里处，这里原有秦时、汉初所建造的祠庙三百余所。《汉书地理志补注》卷三曰 :“雍，秦德公都之。有五　、太昊、黄帝以下祠三百三所。”其注引《史记·封禅书》曰：“秦德公即立，卜居雍。后子孙饮马于河，遂都雍。雍之诸祠自此兴。”“雍有日月、参辰、南北斗、荧惑、太白、岁星、填星、辰星、二十八宿、风伯、雨师、四海、九臣、十四臣、诸布、诸严、诸逑之属，百有余庙。”武帝遂于此处新设一座“太一　”。
　　考汉时之雍县，即今凤翔县。它与周至、武功等，同属汉时的扶风郡，终南山即绵延郡内。《古今图书集成·职方典》卷 493 《西安府山川考》曰 : “太一山，在府城西南八十里，连瓦秀特，上插云霄，汉武帝祀太一于此。……按《长安志》、《五经要义》 : 终南一名太一，太一者犹曰天下第一山也。”在此山中即建有太一宫。《职方典》卷 505 曰 : “太一宫，在府南太一山中。汉元封初建。”此之太一宫，即《太一宫记》中所言。
　　《太一宫记》曰 : “终南山太一宫者，即太一降临之地也。因其受封，故得而名焉。于古雍之东南，玉案之西北，东接蓝水福地，西连太白洞天，左有千龄观，右有万年宫，　镐两间，杜陵之首，山水秀绝，灵迹环绕，实万古神仙之宅也。”这里明白指出太一宫位于“古雍之东南”，毫无疑问即是由汉武帝时所建的太一　演变而来，亦汉班固《终南山赋》中所言的“神坛”。
　　《太一宫记》继之又曰 : “自汉室元封初 ( 前110年 ) 武帝所建也。帝一夕焚香于别殿，忽觉满庭辉朗，神告浮空，帝惕然潜听曰 : ‘将中秋日，太一玄君降于终南山，与国为福，当建宫殿以俟奉安。'语已寂无所闻。帝恭默诚谢。……由是敕下起宫于此，奉安三昧太一玄君，并十神太一。至期，武帝銮舆躬临，清衷虔祷，果如神告。忽然山川震动，箫鼓鸣空，云鹤荡逸，天花散坠于宫南巨谷间，摧峰裂岫，万仞绝壁，下化为池沼，绀波云动，水色霞辉，谷之西北，数峰巍耸，上现楼台金碧，烂光紫烟。帝悸然敬谢，寻封其山曰太一，池曰澄源，修太一玄君祠于池之侧。祠成，其化宫失矣，惟神沼存焉。遂分上下宫，皆锡羽流焚修。”
　　从前面所引武帝崇拜太一事迹可证，《太一宫记》中这段记述是有所本来，只是渲染的词句与程度差别而已。汉班固《终南山赋》亦告诉我们，汉之帝王经常至终南山太一祠祭神祷福 :“伊彼终南，岿截嶙力　，　青宫，触紫宸。……碧玉挺其阿，密房留其巅，翔风哀鸣集其上，清水泌流注其前，彭祖宅以蝉脱，安期飨以延年，唯至德之为美。我皇应福以来，臻扫神坛以靠诚，荐珍馨以祈仙，嗟兹介福，永终亿年。”
　　《太一宫记》的第二部分是述晋代事迹 : “于光熙中，奉敕重加完整，沼三洞法司梁谌主其宫事。法师道业高迈，德行崇显，内守真玄，外专真素，每奉征命，祈晴祷雨，却厉储祥，无不感应。至太兴元年戊寅岁上升。晋元帝闻而异之，遣中使就山设普天大醮，封玄君祠为金华洞天，并上下宫额，悉皆御书。乃谥法师为升玄天师，以发辉至道，旌显玄风，俾万世修真之士知仙可学而得焉。兹后三元五腊，八节四时，请福祈恩，壤壤如市。”
　　碑记中这些言辞，亦是有本可证的。光熙为西晋惠帝所用年号，仅有一年 (306 年 ) 。梁谌为晋代高道，隐修终南山，为道教楼观派重要传人。《终南山说经台历代真仙碑记》曰 : “履道有年，志尚高邈，精忱遐感，以晋惠帝永兴二年（305 年 ）遇太和真人降其庭，授日月黄华上经、水石丹法，并授本起内传。三年丹成，身轻如羽，颜若童儿，目见地中，耳闻霄汉。”《历世真仙体道通鉴》卷三十曰 : “道士梁谌者，字考成。京兆扶风人也。博通经史，虽阴阳占候之术，靡不精究，外声利，薄滋味，唯以安闲自适。”“咸熙初（264 年 ）事郑法师于楼观时，年十七。”“后隐于终南山，食气吞符，大尽其妙，又广索丹砂，还而为饵。凡辞气之出，则音韻清澈，自是钦奉多。”于东晋元帝大兴元年 (318 年 ) 羽化，享年 71（247-318 年 ) 。这些记载与《太一宫记》所言完全一致。
　　碑记第三段是讲述隋唐故事 : “逮隋开皇中，兴崇尤盛，继有道士杨景通住持。景通真浑沦仙客，高道奇人也，善胎息辟谷，炼气出神，咒水飞符，役使六丁，召龙致雨，救旱苏危，即有岁时。至唐高祖始定天下，闻而惊喜曰 : ‘太一者五福之主，万乘敬仰之庭，真人圣贤，间世而出，实天下太平之兆也。'遣中贵降手诏，修崇殿宇，一概鼎新。选戒洁道流三十员以奉香火，及赐土田绕宫周广五十余里，以为斋给之费。于景龙三年（709 年 ）上元日，杨景通升堂鸣鼓，召门弟子曰 : ‘吾将朝元始上帝，汝辈修真，无令自惰，人身一失，永劫不复。'俄有瑞云覆地，鸾鹤翔天，金童玉女，迎师浮空，再蒋而去，异香清气，余月不歇 ( 师于真观间已三百岁矣 ) 。时中宗皇帝闻而倍加称叹，　不得夙授真教，北面以师礼之。御撰赞文，遣上卿降香及敕，修澄源阁于池之北，创升仙堂于下宫。至开元中，玄宗皇帝銮驾诣宫，登坛虔祷，顷忽祥风异香，烂霞轻霭，　覆坛上，中有神仙隐隐而见，天衣羽服，鹤驾霓旌。帝拜首俯伏，请问国之休咎，仙一一垂谕。少选，云色高举，鹤驾辽天，上稽首祈谢，以仙语书其册，自为驾鹤篇，即修驾鹤轩于宫之西北隅，帝亲文其碑以彰神异。銮舆归国，颁宣中使继踵不绝。自僖宗皇帝入蜀之后，兵火数至，道流潜遁，宫宇隳灭，所赐常住，悉为众有，存惟两宫基址、尊像而已，其诸盛事，废失之尽。上宫传呼为澄源夫人之庙，下宫为太一之观。其澄源者，即当时所封池阁之名也 ; 夫人昔，乃太一玄君也。盖岁月浸久，后人不知其源，相习之讹。嗟乎 ! 玄宫真府，半为樵叟之家 ; 仙圃芝田，尽作茺榛之野，修崇废久，灵沼仍存，风雨顺时，尚为民福。”
　　考隋唐之际，终南山确实道业鼎盛，高道辈出，如王廷、于章、岐晖、严达、巨国珍、田仕文、李淳风、尹文操、岐平定等。隋文帝崇信道教，开皇初 (581 年 ) 到终南山中，“沐芳礼谒，获闻休徵” ; 并下令重修宫观，“精择羽流，累致墨词，以祈景福，于是朝野宗奉焉”( 见欧阳询《大唐宗圣观记》、《历世真仙体道通鉴》卷三十 ) 。
　　隋大业十三年 (617 年 ) ，李渊入关，“尝幸观庭，命建醮，有瑞应”，遂封楼观主持岐晖为金紫光禄大夫。 ( 见《终南山说经台历代真仙碑记》) 适时李淳风亦在山中，便假老君之命劝进李渊。《混元圣纪》卷八曰 : “大业十三年丁丑，老君降于终南山，语山人李淳风曰 : ‘唐公当受天命'。淳风由是归唐。”
　　唐朝初立，李渊为皇号高祖。于武德二年(619 年 ) 诏令重修终南山各大道观，并赐田 10顷，以道士岐平定主观事。武德三年 (620 年 ) ，诏令改终南山楼观为宗圣观，并赐米、帛以供其用，李渊亲临终南山，谒宗圣观、老君殿等。武德七年 (624 年 ) 冬十月，李渊又幸终南山，谒崇圣观、老子殿。
　　至高宗时，宠信终南山道士尹文操，并亲行终南山、老子庙，出现了老君显灵的圣迹。《古楼观紫云衍庆集》卷上《大唐宗圣观主银青光禄大夫天水尹尊师碑》曰 : “上及皇后诸王公主等，同见老君乘白马，左右神物，莫得名言，腾空而来，降于坛所。”
　　玄宗时李唐崇祖尊老的活动达到了最高潮。他利用各种机会大肆颂扬其玄祖玄元皇帝——老子，一再诏令天下诸州普遍建立玄元皇帝庙。由于楼观道士李玄　 , 颜无待、傅承说的参与，又出现了老君显迹终南山的神话。《册府元龟》卷五三详载其事 : 开元二十九年 (741 年 ) 四月，玄宗对牛仙客、李林甫说，十日前于梦中见一真客云 : “吾是汝远祖，吾之形象可三尺余，在京城西南一百余里，时人都不知年代之数，汝但遣人寻求，吾自应见。汝当庆流万叶，享柞祚无穷。吾自度其时，合吾与汝兴庆中相见，汝当大庆，吾犹未及言。”觉后记忆尤新，“联即命使兼令诸道士相随于京都西南求诸，果于周至县楼观东、南山阜间，乃遇真容一，昨迎到便于兴庆宫大同殿安置瞻睹，与梦中无异，卿等可入观之。”于是仙客、林甫俱拜贺，玄宗仍令有司写真容，分送天下宫观，遂大赦天下。
　　天宝元年 (742 年 ) 七月十五日，又于楼观立《大唐圣祖玄元皇帝灵应碑》，由戴璇撰序，刘同升作颂。颂曰 : “终南之北洞真境，关令尹真宅兹岭。阴阴松柏造华顶，草结华楼龙护井，灵仙之窟肃而静，惟皇夜梦真人来。神光赫赫金银台，瑶容绰约冰雪开。霓裳羽驾纷裴回，前圣后圣相感哉。帝心虔求斋玉京，王公百辟咸致诚。云旗彩仗森出迎，日月晏温颢气晶。真容来兮受天庆，真容来兮圣人寿，千春无涯百福有。真容来兮宝历昌，远郊却马雄四方，紫殿敷座烟云香。拜手稽首天地长，玄元之祚万斯唐。”对老君显圣终南山作了渲染。
　　太一宫位置楼观的北面。《大唐宗圣观记曰 : “始皇建庙于楼南，汉武立宫于观北。崇台虚朗，招徕云水之仙 ; 闲馆错落，宾友松乔之侣。秦汉庙户，相继不绝 ;晋宋渴谒版，于今尚存，实神仙之奥区，列真之会府。”众多道观仙宇，星罗终南山中，羽流高道隐修其中，帝王将相谒拜出入，理应鼎盛。由之可见，《太一宫记》所言隋唐故事，大多有所由来，并非王昶所说的“事无可征”。至十碑记中所载唐末后晋桑中令“镇雍祷雨”之事，一时难以确证。但仅此一点，亦不可作为判断碑记真伪的证据。
　　太一宫为终南古观，历汉唐宋元，时兴时衰，至明清时大多遗迹尚存。《三才图会·终南山图考》曰 : 澄源池， “一名太一湫，其上环以群山，雄伟秀特，势迫霄汉，水广可数丈，深丈许，锦鳞浮，游人莫敢触，鳞之大有三二尺者。自昔祷雨，咸在於是。其南即太一殿，左有三官、雷神二洞。所谓金华洞者，在山之最高处。”注意，此之“太一湫”名称，亦见于《太一宫记》碑尾 : “太一观主兼管句太一湫事赐紫道士王希美”。
　　总结以上所言，笔者认为《太一宫记》所述事迹大多有史可证。从道教的立场上看，其文并非“夸诞”、“荒邈”，只是遵照许多道书的通例，对神迹灵徽做了一番渲染而已。因此，在缺乏确凿证据以前，不可武断判此记为伪作。应该尊重原碑记的题记，承认此《太一宫记》为“华山陈抟撰”。
二、陈抟与太一宫之关系考辨
　　从五代至宋初，陈抟主要隐居、往来于华山、终南山，其师友钟离权、吕洞宾、麻衣道者、谭峭等亦活动在这一地区。
　　在后周世宗时，陈抟即出入终南山，与高道羽流相交。《终南山说经台历代真仙碑记》曰 : 显德中梁筌主持楼观庙务，“为观宗主，时陈希夷居仙游宫，与真人密迩往来，为林下友”。”此“仙游宫”亦在周至县境，与楼观相邻。《古今图书集成·职方典》卷 512 曰 : “仙游宫，在县南三十五里。隋文帝避暑於此。”又　南县 ( 今陕西洛南县 ) 有藏修洞、希夷洞，皆为陈抟修真圣迹。“藏修洞，在县东郊外，陈图南藏修于此，后乃之华山。”“希夷洞，宋陈希夷藏修于此，后徙华山，有石床石枕古迹。”此间曾应世宗之沼，于显德三年 (956 年 ) 入阙。《五代史补》说 : “世宗之在位也，以四方未服，思欲牢笼英杰，且以抟曾践场屋，不得志而隐，必有奇才远略，于是召到阙下，拜左拾遗。抟不就，坚乞归山。”至赵匡胤兵变自立，陈抟便正式入道，定居华山。《邵氏闻见前录》卷七曰 : “闻艺祖登极，大笑坠骡，曰 : 天下于是定矣。遂入华山为道士，茸唐云台观居之。”其后羽化于华山张超谷希夷峡。
　　与陈抟交往甚深的吕洞宾，后周时也隐居终南山、周至县。《古今图书集成·职方典》卷512曰 : “‘吕公洞，在城东南三十五里，唐吕岩隐居之洞，洞前有亭。”其师钟离权亦修炼于此。“迎阳洞，在赤谷南五里，相传钟离权修炼处。” 《佛祖统纪》卷四二亦曰 : “钟离权，字云房。自称汉时遇王玄甫，得长生之道，避乱入终南山，于石壁间得《灵宝经》，悟阴中有阳，阳中有阴，为天地升降之宜 ; 气中生水，水中生气，即心肾交合之理。乃静坐内观，遂能身外有身。唐品岩，字洞宾，三举进士不第，于长安酒肆遇云房，将洞宾入终南山，授《灵宝毕注》十二科。”
　　陈抟的好友谭峭，显德年间也在终南山。《佛祖统纪》卷四二曰 : “隐士谭景升，居终南山，与陈抟为之友，著《化书》百十篇，穷括化原。”陈抟亦说 : “吾师友谭景升始于终南山著《化书》。”(《历世真仙体道通鉴》卷三九 )
　　此外，刘海蟾弃官学道，“后遇吕洞宾得金丹之秘旨，自此往来终南、泰化间，复结张无梦、种放访陈希夷先生，为方外友。“ (《历世真仙体道通鉴》卷四九 ) 张无梦、种放皆为陈抟弟子，宋真宗时曾入对，深受真宗敬重。“有黄白术秘而不言。居琼台又十余年，复隐于终南鹤池，久之游嵩山，泛湘汉，抵金陵、保宁、寿宁佛舍，杜门不出。” ( 《历世真仙体道通鉴》卷四八 ) 种放则长年隐居终南，以讲习为业。《国老谈苑》曰 : “种放隐终南山，至老不娶，养母非力耕之粒不馈，四方从学者几百人，由此被召。”
　　陈抟及其师友、弟子均经常出入终南，使终南山成为宋元时期道教最重要的圣地，不仅陈抟一派直袭终南老子、文始之道脉，同时金元全真派更以终南山为其祖庭。理所当然，陈抟本人是非常熟习山中的古观名宇的。考《太一宫记》碑尾记载此碑立于大宋建隆下年壬戌正月一日，即 962 年。其时陈抟正隐居华山，但亦时往来终南。如曾亲自在终南山为种诩 ( 种放之父 )选择葬地“于豹林谷下不定穴”，“既葬，希夷见之，言地固佳安穴，稍后世世当出名将。” (《太华希夷志》卷下 ) 因此，陈转应终南山太一宫道士之请，为其撰写碑记，是合情合理、完全可能之事。因此，笔者认为，此记当依原碑所题，定为“华山陈抟撰”。
　　陈抟在道教、道家学术史上，实是一个传微迷绝、既往开来的关键人物。他生前著述甚富，但其后却散佚难觅。现存的一些题名陈抟的著述却往往被人轻易判为伪撰，这对研究陈抟的思想以及梳理唐宋道教学术源流脉络发展十分不利。今天，我们应当审慎地考辨题名陈抟所撰均每一篇文、每一首诗、每一部著作，既不轻易肯定，更不随意否定，为解开道教思想史研究、中国哲学史研究中的这一难题做努力。这正是本文撰写的初衷。
