　　道教作为一种文化形态在我国广为流传，它的社会教化作用极为普遍和深刻。道教史上不乏有出色的教育家，他们的教育思想、教育方法和教育实践活动，无疑是我国教育宝库中的一枚瑰宝。本文就清代乾嘉时期活跃在晋、陕、宁、甘、青一带的全真龙门派的主要代表人物刘一明的道教教育思想作一浅析，就教于大道方家。
一、教育内容的进取性
　　刘一明，号悟元子，又号素朴散人、被褐散人，原籍山西平阳府曲沃县 ( 今闻喜县东北 ) 。他是西北地区全真道龙门派的第十一代宗师。他一生潜心修道，在晋、陕、宁、甘、青一带设坛传教，著书立说，成为清代最引入注目的道教内丹炼师和学者，且通医术。他的著作《道书十二种》在内丹学术发展史上有重要的地位，是清代全真道教理论著作中最优秀的文献之一。刘一明对道教的研究及其教育活动颇受时人及后人的赞赏。张鸿汀辑《陇右经学之传授》便记载：刘一明精研“《周易》、《参同》、《悟真》之理”，尤“好言《易》，……当时人多从问《易》。”可见了其道学研究和教育活动在当时已颇有影响。刘一明的道教教育内容有积极进取精神，这是他道教教育成就卓著的关键所在。
　　重视人们的主观能动性，主张“造命在我”。刘一明认为人是可以挽救的，在追求道教仙境方面，只要自己通过长久不懈的习善、行善就能保持，恢复“天赋之性”。他说：“ ( 上德之人 ) 当此之时若遇明师指点，行尤为自然之道，……便为无漏真人矣。·····一阴虽微，为祸最烈。金丹有为之道正为中、下人垢后而设” ( 《孔易单真》卷下 ) 。这种“造命在我”的教育思想，不仅在道教修炼方面是积极进取的，而且在社会生活中也有积极的社会效益。它鼓励重视发挥主观能动性，人们通过主观努力来改变自己的命运，实现白己的理想。
　　刘一明的道教教育“教人去妄存诚明善”。这对个人修养和社会教化都是有益的。刘一明的道教修炼有两个阶段、四个步骤。即有为变化之道和无为自然之道两个阶段，每个阶段各有两个步骤，有为变化之道阶段分为“炼己筑基”和“凝结圣胎” ; 以后天修炼，恢复先天阴阳五行之气 ; 后行“无为自然之道”。先行“有为变化之道”，以后天修炼，恢复先天阴阳五行之气 ; 后行“无为自然之道”，借助先天真阴真阳，以成就后天修持之功”。具体办法是“谨言慎行，非礼不履，非礼不行，非道不处 ; 不为酒色财气所迷，不为富贵功名所诱，不为尘缘世情所染 ; ……闭邪存诚，去妄归真，惩忿室欲，烦恼不生，慎恨不起。伐三尸，除六贼，扫七情，净八识，消灭人心，振发道心，戒慎恐惧，无丝毫妄念” ( 《悟道录·日照月临》 ) 。他说：“金丹有为之道教人去妄存诚明善复初者，即采药以医病也。医病者即复初也。医病须药物，复初须明善。……所明者何善 ? 善即仁、义、礼、智、信也。仁、义、礼、智、信非身外之物，乃我本来天命之性所固有者” ( 《会心集·采集药物论》 ) ”。他还强调：“礼者，动静之节。进退之序，为修身应世之规范。”这就是说，刘一明的金丹有为之道，实质是使先天五行，先天真阴真阳都合而为一，凝结圣胎，把后天的“气质之性”炼掉，使人心复归道心，人性复归天理。有为之道，与当时社会盛行的纲常伦理相一致，相吻合。但正是这种将自我融入封建社会整体中的妙论，使全真龙门派在社会上有广泛的群众基础，从而在封建统治下得以发展。圣胎凝结之后，修真之功便由有为之道进入无为之道。无为之道强调内心修养，他说“无为之道，即无欲观妙之功，”“无为者抱元守一之道。……须要百念俱息 . 诸尘不染，身心大定，方能济事” ( 《参同契直指笺注》中篇 ) 。“修性之道，乃炼神还虚之一着。……内以养己，安静虚无，盖养性之道，须要安身于虚，静心于无，身心两忘” ( 《参同契直指经文解》中篇 )从而进入清净虚空之境即无限之中去。“气足神全，霹雷一声，金蝉脱壳，身外有身，功世名遂，朝北阙而驾翔莺，白日飞升，为纯阳不死之天仙，岂不快哉 ! ” ( 《悟真直指解》卷一 ) 。我们且不说刘一明的虚幻之梦肯定是要破灭的，但就其个人修炼的标准和要求来看大有君子之风度，这种“去妄存诚明善”的道风，对社会个人素质的提高和风气的改良起着辅助教化的作用。
　　刘一明的道教教育还十分重视“行”的作用。他在“知”“性命之理”和“行”“性命之功”的认识上，虽然主张“知”先而“行”后，即先“学道”以穷究“性命之理”，而后“修道”以了却“性命之功”，但他并不轻视“行”的作用。他认为，“学道”固是先着，“修道”绝不是可有可无。仅“学道”而不“修道”，即“知”“性命之理”，亦不能成圣成仙成佛。因此，“知道”尚须“行道，“行”是“知”的目的。刘一明的道教“彼岸世界”是虚幻的。但他的“知道”是为了“行道”却是明明确确实实在在的目的。这种“知道”、“行道”的教育，不仅引导众徒在修炼上狠下功夫，而且启示社会上的一切人都要有生活的目的，追求的理想，并为之不断努力奋斗。
　　刘一明以道为本原的宇宙观有着明显的合理因素。他认为字宙是无限的。说：“天体至高，虚园不测，廓大无边，无物不覆，无物不容”( 《悟道录》卷上 ) 。还说：“察乎天地，其大无外，其小无内” ( 《参同契直指笺注》上篇 ) 。用宗教的语言解释了宇宙的浩瀚无垠。 另外，刘一明的赏善罚恶教育思想，事物运动变化的教育思想和对立面双方的互相渗透、三教同源相通等教育思想，都有着他的许多合理性、积极性和进取性。这些教育内容上的特点赋予了他道教教育的旺盛生命人，使全真龙门派为众多群众所接受，在西北地区蓬蓬勃勃地发展起来。
二、教育方法的通俗性
　　刘一明的道教教育之所以在当时有较大的影响，与他深入浅出通俗易懂的教育方法不无关系。当时人杨灿评价其《易理阐真》曾说：“述伯阳之遗义，参神仙之余蕴。亲取其龙蛇铅汞之法，验以交系图象之说，别其纷歧，归于宗主。其辞约，其义繁 ; 其文浅，其旨深。取类极迩，而见道甚远。其身心性命之功，殆不谬乎吾儒之为之者。呜呼，若道人才，遮可与言《易》哉 ! ” ( 《易理阐真·杨序》 ) 对他言简意骇，深入浅出的阐述教义的方法做出了很好的概括。刘一明精研“周易、参同、悟真之理”，他的解易著作是其思想根本。所以杨灿的话具有一定的代表性。刘还自己总结说：“略譬象而就实义，去奥语而取常言。” ( 《易理阐真·序》 ) 也总结了通俗的特点。
　　刘一明教育方法有以下几个特点：一是用人们熟悉的自然变化现象的和农业生长规律为例证来阐述深奥莫测的道教哲理，他宣传道教“造物者”赏善罚恶的因果关系说：“正阳翁曰：种麻即得麻，种黍即得黍。此皆言有因必有果……盖天地间一切大小事物皆有因果。” ( 《通关文》卷下 ) 。又如他借助魏伯阳和虞翻使用过的“月体盈亏”说来注释三《易》，说：“日中则必昃 ，月盈则必亏，此阴阳盈虚消长循环一定不易之理。” ( 《悟道录》卷下 ) 都是借用自然规律来论证的，易于理解和接受。刘单述修道成仙的“炼己筑基”、“炼神还虚”等都运用了司空见惯的日常事物来比喻的方法。他说即如“起屋必先筑基，基地坚固，木料砖瓦由人做作，无不负载也。” ( 《象言破疑·炼己筑基》 ) 对“炼神还虚”比喻说：对景忘情，有无不立，万物皆空，如面于壁而一无所见之义。”( 《参同契直指经文解》下篇 ) 他解释“金丹有为之道教人去妄存诚明善复初者”时，比喻为“采药以医病也。”“医病者即复实也。医病须药物，复初须明善。善即药，明善即采药。明善复初，采药医病，同一理也。” ( 《会心内集·采集药物论》 ) 刘在讲述“穷究性命之理”时，通俗地解释了“性命” ( 金丹 ) 到底是什么。他说 : “有的以为金石锻炼而成，有的以为男女血气而结，有的以为血肾相交而凝，有的以为精神相聚而有，有的以为在丹田气海，有的以为在黄庭泥丸，有的以为在明堂王枕，有的以为在两肾中间，……诸如此类，不一而足，都不过是砖弄瓦，认假为真。”这也难怪世上学道者如牛毛，而成道者如麟角。因此，世间“迷人”“只在皮裹上强扭捏，自以为修持性命”，其结果，“非修持性命，乃伤性命也。性命且不知，妄想长生，岂不遇哉 ? ” ( 《易理阐真·金丹图》) 他认为所谓性命，是指先天阴阳五行之气的凝结而言。“命”是先天至健的阳气凝结而成，“性”则是先天至顺的阴气凝聚而成。“性命者，阴阳之体 ; 阴阳者，性命之用。” ( 《修真辨难》卷上 ) 因此，所谓性命之理，就是阴阳之道。如何“修行丹道”，他简单地比喻“还我娘生本来面目”，解释了深奥莫测“有为变化之道”。刘一明这种借用自然规律日常现象宣传教义的方法，使人易于理解和接受，拉近了群众与宗教的距离。
　　二是借重古人，推崇名著。他“述伯阳之遗义，参神仙之余蕴”。借助魏伯阳、虞翻等人使用过的“月体盈亏”说来注释三《易》。他还继承北宋陈传“逆以成丹”思想传统，讲解如何修行丹道。还借重张伯端、王重阳和邱处机等论述“三教一家”、“三教同源”之说。他将“圣道”归结为“仙道”，从而把《周易》道教化，又将“丹道”归结为《易》道”，从而把内丹道学易学化。
　　他还推崇名著，利用《周易》、《西游记》等著作为其道教教育服务。刘一明认为《西游记》为神仙之书，“其书阐三教一家之理，传性命双修之道。俗语常言中暗藏天机，戏谑笑谈外显露密法。” ( 《西游原旨序》 ) 还认为 ( 《西游记》 ) “贯穿三教一家之理，在释为金刚法华，在儒则为《河洛》、《周易》，在道则为《参同》、《悟真》，乃丹经中第一部奇书”，全书显尽了性了命之学 ( 《西游原旨读法诗结上卷 ) 。
　　借重古人和推崇名著提高了刘一明教育的知名度和可信度，这在道教衰微的清代提高了道教宣传的力度，扩大了全真龙门派的影响。
　　三是“绘图画象，细分是非”法。刘一明在《象言破疑》二卷自序 : “知是在《 ( 易理 )阐真》、《( 参悟 ) 直指》、《 ( 西游 ) 原旨》诸书刊行后，恐学者难于贯通，所以著此书，“绘图画象，细分是非，扫旁门而指正道，息邪说而辨真宗”。此书以说统图，解说金丹之象；卷上十四图，从《胎中而目》图至《纯阳元阳》图，表顺则生人之象，从《炼己筑基》图到《太空虚无》图表逆行造化修仙之道。卷下则专就内丹修炼中诸紧要关节处绘出《堰月炉》、《朱砂鼎》、《玄北之门》、《火候》、《十月胎圆》、《婴儿出现》等图十七幅。并辑《破迷诗》五十首，专驳斥纠正各种错误理解，再以《元牡真窍说》、《修真要诀》作结。此书于内丹各象解释分明，实为图释内丹，释正与破邪并重，对于内丹理论的阐发颇有贡献。
　　另外，刘一明还采用师生问答的方法，为修道者辨难释疑阐发道、阴阳、内外药等要义。
　　刘一明这种由浅入深、通俗易懂、行之有效的教育方法，拉近了群众与道教的距离，从而也使道教超出了仅在道教中传播的范围。这种被刘一明通俗化了的道教，尤其是作为一种炼养术的内丹术传入社会，甚至在儒士中有很深的影响。而一种教育、信仰的生命力必须根植于群众之中，使之具有广泛性、大众化的特点。刘一明通俗化才教育方法把道教进一步推向群众，使全真龙门派道教的影响风靡我国西北地区。
三、教育理论辩证性
　　刘一明的道教教育理论有较高的思辨性，可称独树一帜。他的辩证法更多的表现在他对事物的存在、运动或发展的看法和他的丹道思想两个方面。刘一明认为宇宙间天地万物存在的方式是运动变化的。其思想中包含了许多辩证法的因素。他认为事物都是对立统一、互相依存和渗透，而且是可以转化的。他认为宇宙间的事物存在着互相矛盾的对立的两个方面，如阴阳、刚柔、强弱、真假、动静、逆顺等等。它们又都是互相依存、互相联系、不可分离的。他说：“阴阳同居” ( 《周易阐真·河图》 ) ，“动不离静。静不离动，一动一静，互为其根。”“有生必有死，有死必有生。是生以死为根，死以生为根也。有恩必有害，有害必有恩”。( 《周易阐真·先天方园图》 ) “同中有异，异中有同。” ( 《悟真性宗真指》卷四 ) 刘一明指出，对立双方在发展到极点时就会向相反的方向转化。他说：“动极而静，静极而动 ; 阴极生阳，阳极生阴。”“高极必下，明极必暗。” ( 《指南针·阴符经注》 ) 基于宇宙万物对立统一的思想，刘一明认为，万物“变化无穷”，“生生不已”，“造之化之，生之成之 ; 造而又化，化而又造 ; 生而又成，成而又生 ; 消息盈虚，无有停息” ( 《周易阐真·文王后天八卦》) 。
　　刘一明是著名的道教内丹练师，在他的丹道思想中也不乏有丰富的辩证法因素。他认为，一切事物的存在、变化和发展，都是由“阴阳混沌未分”的“先天真一之气”所主宰。这个“气”，“是生物之祖气，是鸿朦未判之始气，是混沌初分之灵根” ( 《修真辩难》卷下 ) 如人有生之初，“阴阳一气，渐生渐长” ( 同上 ) 。此时，“阴恋阳，而阳恋阴，阴阳同气。” ( 《周易阐真·泰卦》 ) “阴阳同居”、“浑然一体” ( 同上书《河图》 ) 。然而，“乃至二八，元阳气足，满而必溢，极而必返。阳极则阴生，阴极则阳消” ( 《修真后辨》卷下 ) 。此时，阴阳涣散，睽离不交，“彼此戕害”。“如《书》 ( 指《洛书》) 阴水克阳火，阴火克阳金，阴金克阳木……”，如此“阴剥阳”，“阳陷阴中”，“阴伤其阳，本来天真失矣。”于此，则“阴阳调和之功不可缺” ( 《周易阐真》) 。
　　那么如何调和阴阳呢 ? 刘一明也用辩证的思想方法来分析。他认为“调和阴阳”，首先必须懂得“‘阴阳互为其根”的道理。“夫金丹者，太极也。太极中含一气，一气动而生阳，静而生阴。动极则必静，静极则必动，阳极则必阴，阴极则必阳，阴阳互为其根。” ( 《修真后辩》卷下 ) 还须懂得“先天己亏，百病俱发，纵得师诀，不得直下纯一，必须由渐而顿，由勉而安。”只有“到的了命之后，与上德 ( 指本源未亏，灵窍未闭 ) 者同归一辙，始可以无为矣。” ( 《修真辩难》卷上 ) 只有通过一番“由渐而顿”，扶阳抑阴”的艰苦努力，从而使“阴者可以化而为阳” ( 《周易阐真·履卦》) ，“阴不陷阳而顺阳” ( 同上书《比卦》) “阴为宾而阳为主”，“以阳统阴” ( 同上书《先天方圆图》) ，阴阳相济，刚柔相应，至此，“金丹自虚无中结就，取而服之。长生不死” ( 《修真九要》) 。
　　刘一明丹道思想中的辩证法显而易见。刘一明道教教育的思辩理论性主要来源于两个方面，一是继承并发展了全真龙门派的传统教义思想。二是汲取并发挥了易学的辩证法，加之个人在研究和实践中的进一步认识。
　　刘一明作为为道教神学服务的教育活动家，在唯心主义的宗教世界里，他的教育思想的闪光点极为可贵。虽然他最后以生命的悲剧揭露了道教神学的虚妄梦幻，但他一生的教育活动却留给后世较多的思辩哲学意味。
　　综上所述，刘一明的道教教育思想无疑是我国教育宝库中的一枚瑰宝，他积极进取的教育内容，深入浅出的教育方法，朴素辩证的教育理论，都为我国的道教教育思想增添了丰富的内容。
