　　近来，政坛 ( 指马来西亚 ) 最热门的话题，首推禁酒，自数年前，丹州政府禁售酒后，今年联邦政府也实行禁止酒后驾车。酒——这穿肠的毒剑，果为一大害，政府禁之有理。
　　在华人宗教界里，佛教是禁止饮酒的，华人的主要宗教——道教，是否也禁酒 ? 从华人庵庙庆祝神诞，人们大肆饮酒，啤酒的广告布条及海报，悬挂、张贴于庙宇的四周来看，仿佛道教是不禁酒的。又于神庙的犒赏五营神兵 ( 包括茹素的九皇庙 ) 、民间节日如天公诞及祭祖祀神活动中，祭品里头少不了酒来探讨，道教给人们的印象是不戒酒的。
　　事实上道教的戒律是禁止信徒饮酒的，这一点道教徒似乎已忘了。
　　道教的第一部经典——后汉的《太平经》云：“凡人一饮酒令醉，狂脉便作，买卖失职，更相斗死，或伤贼 ; 或早到市，反宜乃归 ; 或为奸人所得，或缘高坠，或为车马所克贼。推酒之害万端，不可胜记。念四海之内，有几何市，一月之间，消五谷数亿万斗斛，又无故杀伤人，日日有之，或孤独因以绝嗣，或结怨父母置害，或流灾子孙。……盖无故发民令作酒，损废五谷，复致如此之祸害。但使有德之君，有教敕明令，谓吏民言，从今已往，敢有市无故饮一斗者，笞三十，谪三日 ; 饮二斗者 ; 笞六十，谪六日 ; 饮三斗者，笞九十，谪九日。各随其酒斛为谪。酒家亦然，皆使修城郭道路官舍。……”
不持斋戒
　　《太平经》提出了禁酒的主张，指出酿酒浪费粮食，酒醉以后，容易产生种种祸患，政府应当明令禁止饮酒和酿酒，违者要按情节轻重给以不同的处分禁止和劳动教育，罚修城郭道路官舍。这些办法，张鲁在汉川时曾实行过。 ( 见《中国道教思想史纲》第一卷 )
　　左仙公云 : “学道不修斋戒亦徒劳山林矣”。道教徒学道莫不持戒精虔，益善止恶，渐而悟道，不戒诸恶行，道无由得。故道门的众多戒律，是道教徒严格律己的修道方法。道教的许多戒律亦提到戒酒。
　　积功归根五戒之第二戒 : 不得荤酒。按 : 《老君戒经》曰 : 戒酒者，非身病，非法礼，皆不得饮。身病，谓已身疾，应以酒也。法礼者，明非世俗撰会及鬼神之炸 ; 若尊卑之礼，真灵之飨，则不至于失之者也。
　　“初真十戒”之第七戒：不得饮酒食肉，犯律违禁。当调和气性，专务清虚。按：《初真十戒文》解曰 : “不饮酒者，所以戒乱性也。盖酒本以行礼，浃洽和气，古人主宾百拜终日而不至醉，示有节也。后人嗜欲无厌，纵饮谋醉 ; 遂至形骸颠倒，礼法丧失，或以致病，或以构祸，罔觉其非。”
　　老君五戒 : 不杀生 ; 不偷盗 ( 也人不与取 );不邪淫 ; 不妄语 ; 不饮酒。按 : 《老君说五戒》解曰 : 酒戒者，南方火也，太阳之气，物以之成，而人好酒，则心受其毒。其他戒酒的尚有 : 女真九戒 ; 中极三百大戒 ; 智慧上品大戒 ; 二十四门戒等。 ( 以上见闵智亭著《道教仪范》 )
　　道教的禁酒，是因为酒能乱性，而且不生智慧，酒饮得多，则心性迟钝，渐而愚痴。道教只允许教徒在“身病”及“法礼”情况下饮洒。倘若因身上的疾病须治以药酒，那是可以喝的 ; 法礼者，古代规定的礼仪，现在的新时代，已不受古礼约束，这方面道教徒不可借故乱破酒戒。
　　民间的道教活动中，我们可看到正月初九天公诞，善信祭玉皇以茶及酒，三瓯或九瓯的茗在前，酒在后，茶是供奉玉帝，他是三界的最高执政者，当然不受荤酒，以酒祭天是古代之礼 ;再看神庙的犒赏五营大兵神将，这是模仿凡间的军事活功，这五营天兵神将，专供神明遣用，祭以酒是古时之传统，从明清小说中，可看到将士出征时受赐酒三杯，大胜回营又设酒宴大飨三军。
　　道教以酒祭祀天地社稷、鬼神祖先，那是“无酒不成礼仪”的古礼，与禁酒戒律相悖而行。其实这是中华文化民俗，多为道教所吸收，所以不了解道教，就很难全面地了解中国的传统文化及其历史。 鲁迅先生曾说 : “中国根柢全在道教，以此读史。有多种问题可以迎刃而解。”
九皇爷诞
　　一些闽籍老人家告诉我，如果家中无酒或无钱买酒情况下，可以酒瓯盛清水，内放几粒米以代“米酒”祭神。拜神是以诚意的，不是区别于酒的贵贱。设使道教徒能够了解这一点，以“米酒”代酒祭祀天地神明，不仅可以不犯教戒，亦可保存传统古礼 ; 两全其美之方法，何乐而不为 ?
　　道教徒须戒酒， 9 月上旬的九皇爷斋戒更戒酒，但马来西亚的神庙庆神诞及九皇大帝诞，啤酒在庙宇内售卖，信徒大肆狂饮，实已严重犯了酒戒，冀望筹备中的马来西亚道教总会及九皇大帝总会能关注此事。
　　严守酒戒，信徒在心理及生理两方面才能健康，才能步入长生之门，而且可提高素质，利于道教之发展，慎之，慎之 !
