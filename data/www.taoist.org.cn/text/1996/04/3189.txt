　　汉代安顺之际，天灾纷至、生灵涂炭。秦汉以来的信仰危机、伦理危机、社会政治危机全面爆发，清明、太平、宽松的政治环境自然成为人心所向。这时，《太平经》 ( 又名《太平清领书》 ) 应时而生。该书提出了一些清明政治的设想，勾画了一个理想政治社会的蓝图。本文试以《太平经合校》 ( 王明，中华书局， 1960年 2 月版 ) 为据，对其中的政治伦理思想作一粗浅分析。
( 一 )
　　《太平经》的政治伦理思想是以其宇宙生成观为起点展开的。对宇宙生成的秩序，《老子》认为“道生一，一生二，二生三” ( 《老子》 42章 ) ，《太平经》继承了这一思想，用“三气一元论”来解释。它认为“夫物始于元气” (254页 ) ，“夫天地中和凡三气，内相与共为一家，反共治生，共养万物” (113 页 ) ，“元气乃包裹天地八方，莫不受其气而生” ( 78 页 ) 。故而，天地万物产生的根源是气，“元气与自然太和气相通，并力同心，时　　未有形也、三气凝，共生天地” ( 148 页 ) 。其具体的化生过程为“元气惚惚自然，共凝成一，名为天也 ; 分而生阴而成地，名为二也 ; 因为上天下地，阴阳相合施生人，名为三也，三统共生，长养万物” (305页 ) 。这样，经过气的化生，世间万物开始出现并按自己的规律运作，构成一个纷繁复杂的大千世界。在这里，它强调的是三个要素 ( 天、地、人 ) 共同齐心努力，才可使宇宙万物运作有序化。这一思想具体落实到政治伦理上，就成为“三合相通”之说：君臣民基于各自的政治伦理准则和道德规范，互相调和、协作，致成“太平”之世。
　　“天人感应”思想是中国传统文化中的重要思想内容，《太平经》也以此作为其政治伦理思想的基本理论依据之一。它强调君主政治与天地之间的感应关系，同时也把感应的范围扩大，使之照及一切众生。“天之照人，与境无异”(18 页 ) ，那么人的一切行动都由上天注视，并据此作出“天”的价值判断：或呈祥或显实。“王者行道，天地喜悦 ; 失道，天地为灾异”(17 页 ) 。且“夫天地之性，自古到今，善者致善，恶者致恶，正者致正，邪者致邪，此自然之术，无可怪也” (512 页 ) ，所以按它的逻辑，结论则是：信道者当奉天地，法天道，得天心，顺天意。尤其是贵为天子的帝王，应顺承天道，顺天地者，其治长久，否则当遭天罚。通过如此论述，《太平经》一 方面为君权的合法性涂抹上神圣的色彩，另一方面又把政治伦理的合理性标准放到自然界之变异灾祥之中。
( 二 )
　　《太平经》以“太平”二字命名，体现了作者在乱世对一种带宗教理想色彩的“太平世道”的向往。它对“太平气”的解释为：“太者，大也 ; 平者，正也 ; 气者，主养以通和也 ; 得此以活，太平而和且大正也” (148 页 ) 。可见，“太平气”代表着公平、正义、相爱相通，代表着最完善的政治伦理。在这一思想统率下，君臣民各自遵守一定的道德约束、伦理规范，三合其力，各司其职，各行其事，则“君导天气而下通，臣导地气而上通，民导中和气而上通”、“立平立乐，灾异除，不失铢分也。” (152页 ) 。
一、人分等级、各安其位。
　　为实现公平、快活的和睦社会，《太平经》提出天地之间从人类到万物，都有尊卑贵贱、君长臣民之分，这是上天的安排，是命中注定了的：“故人生各有命也，命者贵不能为贱，命贱者不能为贵也” (289 页 ) 。而现有的封建尊卑贵贱秩序是合乎“天道”的：“阳者为天、为男、为君、为父、为长、为师 ; 阴者为地、为女、为臣、为子、为民、为母” (271 页 ) 。按道教的逻辑“阳”总是尊而贵的，“阴”总是卑而贱的。故向君尊臣卑、官尊民卑、父尊子卑、男尊女卑等等，都是天经地义的。
　　为与封建伦理道德相合拍，它还强调了人分等级的说法。它认为“天生人凡有三等：第一天生，第二地生，第三人种类。受命天者为人君，受命地者为人臣，受命人者为民。君者应天而行，臣者应地而行，顺承其上 ; 为民者属臣，转相事。”(730页 ) 。这种人分三等的说法，无疑是为论证君王统治的合理性作依据。无独有偶，西方政治思想史上柏拉图在构建其实现正义的理想国中也曾提出“等级各异”说。柏拉图认为人的不同等级是由神用不同质料创造而成，质料是金质的为统治者等级，质料是银质的保卫者等级、质料是铜铁的则为生产者等级。一般说来，这种等级是世袭的。以这种等级划分而成的人各有其天性，每个等级由于天性不同，有着不同的美德，从而在国家中担负着不同的任务，有着不同的地位。第一等级的天性是理智，由此产生的美德是智慧，这部分人最少 ; 第二等级的天性是意志，美德是勇敢，这部分人也较少 ; 第三等级的天性是情欲，美德是节制，这部分人最多。根据“各人应做适合自己天性的工作”这一原理，柏拉图认为第一等级应当做统治者，第二等级为国家的保卫者，第三等级应充当生产者，为前两个等级提供物质上的需要。和《太平经》中的等级思想相对应，柏拉图所指的第一、二、三等级分别为君、臣、民。二者都是用神学色彩作外衣，同具人在天性上不平等的实质。
二、君臣民关系：“三合相通”。
　　基于“一分为三”、“三合为一”的哲学思维，在君臣民之间的关系上，《太平经》主张阶级调和。“君为父，象天 ; 臣为母，象地 ; 民为子，象和。……君者须臣，臣须民，民须臣，乃后成一事，不足一，使三不成也”，“故君臣民当应法天，三合相通，并力同心，共为一家也” ( 150 页 ) 。它看到君臣民同为政治体系中不可或缺的元素之一，三者互为依托、共同支撑着政治秩序的正常、良性运用。也就是“凡是三气共一治，然后能成功。故上之安者，其臣良也。臣职理者，其民顺常。民臣俱善，其君明，其治长。太平者以道行，三气悉善，合乎章，怀道德不相伤心” (730 页 ) 。可见，君臣民都遵循“三气悉善”的政治伦理道德，是太平长治的根本条件。同样，柏拉图在对等级进行划分之后，要求三个等级的人各司其职，各守其序，才能各得其所，使所有的琴弦都和谐一致，合奏一支交响曲。整个国家，也就是一个符合他所谓的“正义”概念的理想国家。反之，如果三个等级互相攘夺，互相干预，就是不正义的行为，会把国家毁了。
　　道教的《太平经》和柏拉图的“理想国”产生的时代属人之上升的不同时期，所处文化背景、社会制度也相异，却基于如此雷同的政治伦理思想，提出了如此相似的理想社会的蓝图，颇值得我们深思。
三、明君的政治伦理规范：道、德、仁。
　　基于道教伦理修养学说中“身治是国治也”的思想，《太平经》认为，帝王若能养其性，即能养其民。而帝王的“性”具体到政治伦理要求上，则是君贤、有德、仁慈爱民。
　　(1) 明君应效法天地，实行道德仁治。《太平经》说“为人君上者，当象天而行，乃以道、德、仁为行三统” (712 页 ) “君者，当以道德化万物，令各得其所也。不能变化万物，不能称君也” (20 页 ) 。至于“刑罚”，《太平经》也不反对。任何政治伦理学都是以人性伦为基础展开的。《太平经》基于阴阳二分伦，认识到人性中有善有恶，故而在具体的政治运作上，应刑仁并举，只不过它更主张重德轻刑：“阳者为生，阴者为杀 ; 阳者为道，阴者为刑。……故君贵道德、下刑罚，取法于此。 " (12 页 ) 。即使用刑，也要十分慎重，要用得适当，尽量避免重罚和扩大化的后果。真正的明君应实行以德治为主，刑治为辅的统治政策。
　　(2) 周穷救急思想。《太平经》认为，君主与富人都应当行仁好施，周穷救急，“赐饥者以食，寒者以衣” ( 230 页 ) 。若“积财亿万不肯救穷周急，使人饥寒而死，罪不除也” (44 页 ) 。这一思想提出的根据为“此财物乃天地中和之所有以共养人也” ( 246 页 ) ，和柏拉图主张一、二等级的人除掉必不可少的用度之外，不许有私有财产的极端思想相比，“周穷救急”温和得多、可行性也大得多。
　　(3) 以民为本的思想，这是对孟子民本思想的继承发挥。《太平经》要求明君实行“以民为本”的国策，说“君少民，通衣食不足，令常用心愁苦。故治国之道，乃以民为本也。无民，君与臣无可治，无可理也。是故古者大圣贤共治事，但旦夕专以民为大急，忧其民也。”(151 页 ) 。至于天下最急迫的事有三项：饮食、男女、衣服。治国家应当“守三急”，其余不清浮华奇伪之物“反多以致奸伪”、“君子失其政令，小人盗劫刺，皆由此不急之物为召之也。” (42—45 页 ) 。这是针对君主的穷奢极欲的反伦理行为提出的。
四、良臣、顺民的政治伦理规范：忠、孝。
　　紧跟儒家的三纲五常思想，《太平经》提出了臣、民的政治伦理道德规范。它说：“三纲六纪所以能长吉者，以其守道也，不失其治故长吉也” ( 27页 ) ，这无疑是证明了封建伦理道德的合理性。对臣、民具体的道德规范和行动准则的要求，则是“孝”、“顺”、“忠”。若有违犯，则“子不孝，弟不顺、臣不忠、罪皆不与赦”(406 页 ) 。至于上善的臣子道德规范为“夫上善之臣子民之属也，其为行也，常旦夕忧念其君王也。念欲安之心，正为其疾痛，常乐帝王垂拱而自治也” ( 132-133 页 ) 。即使君主衰微或有了过失的时候，臣民应当扶助他，而万万不可抛弃他。“臣见君父之衰，救之，使其更兴盛，是大功也 ; 深知其衰也，不救之，或反言而去，名为倡娇。罪不除也。 " (685 页 ) 。可见，《太平经》要求臣、民无论在何时都应谨守自己的伦理规范，无条件效忠君王。总之，“人生之时，为子当孝，为臣当忠，为弟子当顺。”
( 三 )
　　可见，《太平经》中的政治伦理思想秉承道家‘'清静”之风，构建了一个美好的政治神话，其具体主张中又不乏与西方柏拉图思想相似之处，让人惊叹。其合理的部分值得我们深思。
