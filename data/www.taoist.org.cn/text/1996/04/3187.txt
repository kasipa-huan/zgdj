　　道教作为我国土生土长的有着一千八百年悠久历史的宗教，在国内曾有着广泛的传播和影响。历代以来，道教的洞天福地、宫观文物等成了名山胜境，道侣期荫仙风栖止于此，游人仰慕名胜寻迹而至。道教景观以其特有的魅力吸引着古今中外的旅游者，成为人们游憩赏览的佳境之地。如何审美、鉴赏道教景观，这是一个值得研究、探讨的问题。
　　近年来，有些地方在旅游开发中，“拆了古的盖洋的，拆了真的盖假的”，其中包括盖了一些作招揽游客之用的神仙庙宇，它们的炮制，本来与宗教界无关，但其景点的设立，却利用了像道教这样历史悠久的宗教的影响，借用了宫观、神仙的外观。这种做法，受到了一部分有识之士的批评，有关“景点”开放以来，效果也并不好。与真正的道教名胜相比，它们远没有那么强烈的号召力、吸引力，究其原因，撇开此类景点常常粗制滥造不说，其中重要的原因，是它们没有真正的道教景观那种深沉的历史感，因而显得没有根基，在旅游者眼中，不耐看，缺少有厚度的鉴赏价值。
　　深沉的历史感，是道教景观的旅游鉴赏价值的重要内容。这种历史感，是这些景观与生俱来的，并非从外界赋予，也无法将之抹煞，因而弥足珍贵。本文就想围绕着它，稍述一得之见。
一、可直观的历史积淀
　　道教景观大多历史悠久。除港、台地区在近数十年来有较多新造宫观之外，大陆道教宫观中，要找出本世纪中新创建的宫观神庙，实在是一件难事。五百年历史的宫观和洞天胜景，只能算年轻后辈，七八百年乃至近二千年的景观尚历历可数。即如被联合国列入世界自然历史遗产的泰山，与道教的前身秦汉时的方仙道就有联系，武当山，至迟在唐代已有道教祠庙出现。因此，道教景观基本上都可列入历史遗存。这是道教景观的一个重要特点，也是形成它特具魅力的观赏价值的重要因素。
　　这种特具魅力的观赏价值，在于它将历史封固在自己的建筑、植被、洞天之中，使历史的积淀，呈现出可以眼见耳闻、亲历其境的感性外观。
　　旅游中的观赏，离不开种种赏心悦目的感性形象。无论是自然景观，或是人文景观，没有“好看”的形象，便失去了吸引人的基本条件。然而，“看”什么，不同类型的景观，却大有不同。道教景观，就其内容来说，可以归入人文景观。然而由于道教素来亲近自然，注意将自己与自然融为一体，所以道教景观，往往是人文景点和自然景点的有机结合。特别在洞天福地之中，更是如此。这些是构成其景色美的外在特征。然而，倘若要问，它们与一般的新建的绿荫深处隐红楼，宾馆、别墅星罗棋布于秀山流水的景观有什么不同，那么，一个重要的区别，就在于游览前者，人们在当下直接的景观中同时看到了历史；而后者则没有。这便使得道教景观比后者更耐看耐品，因为它们具有后者所不具有的深沉的历史感。它们不仅有空间上的广度，有空间结构上的精巧、和谐，而且有时间上的深度，有时间流逝中的遗留、积淀。与分布于眼下的空间中的景点一样，这种历史的积淀，也成了可视可触的感性存在。
　　所谓看到道教景观中的历史积淀，内容也有种种不同的外观和内涵。其中之一，是遗留千百年的历史的实物见证。陕西周至县的楼观台，据说，系二千五百余年前老子西出函谷关，遇关令尹喜挽留，于此写下了世界文化史上的名著《老子》，此书道门中尊为《道德真经》，老子被尊为道祖。约晋代，有道士在此建楼观，成立道派。此处山脚田野处有古柏数株，其中一棵系牛柏，据说是当年老子系青牛处。系牛柏遒劲地搏击二千五百年的风霜，仍然屹立于斯。从它的苍古，我们仿佛又回到了二千五百年前中国文化史上重要的一幕。泰山之巅，有古封天、告天处“古登封台”，后在其旁建玉皇殿。封泰山、禅梁父，是方仙道的重要主张。秦始皇、汉武帝都曾来此封禅告天，为道教前史中的重要事件 ; 后来的皇帝，也颇有涉足于此，唐玄宗、宋真宗等的登泰山，又是道教史上的大事。这古登封台，便是当年种种历史 事件的见证。其旁的玉皇殿，玉皇不作帝王冕旒之装，却似一位古代真人，大约也是想唤起人们对历史的回忆吧。这类历史见证，在道教景观中可以说到处都有。
　　在道教景观中的不少自然景观，都有久远的历史，它们往往在道教徒入其境前就存在，但由道教的神仙之说和宗教活动。赋予了它们活的灵魂。它们主要是作为历史活动的见证而存在，在道教景观中的楼台、殿阁、宫、观、庙、庵等等建筑物，则是古代祖先们的文化创造。游于斯，乐于斯，大量看到的便是这类历史上的创造，或者说，亲眼目睹着古代道教徒和工匠、民众的创造文化的历史。
　　在这些历史的创造中，不乏当时条件下的精品绝活。武当山天柱峰金顶和其中所奉的真武铜像，结构之精，形象之传神，已可作为文物精品，最奇的还在于每当东方第一缕阳光射向金顶，首先照着的是玄帝像的脸庞。这种独具匠心的设计，正表现了营造它的明代永乐年间，中国文化、技术在世界上的领先地位。苏州玄妙观中保留的唐吴道子画老子像刻石，线条流畅，见之犹可领略一代画圣“吴衣当风”的艺术风格。茅山道院珍藏的宋传“四宝”，其中的玉圭，有“飞龙在天”的天然图画不论，其玉质奇特，每当夏季其身便会“出汗”——有极薄水汽凝干上，扪之湿润 ; 其中“哈砚”，向之哈气，便有水珠凝结，可以调朱画符，这些珍宝的发现者、制作者不仅在当时要算一流高手，就是八九百年后，也令人叹为观止。在道教景观中，无论是建筑、造像、碑刻、珍宝，数不胜数，观赏它们，犹如读着历史，看到当时历史条件下的创造和成就，感受到祖先们的创造性仍然活在身边。
二、耐品味的历史内涵
　　上面说的，是道教景点历史感的一个侧面：可以直接观看到的历史积淀。实际上道教景观中蕴含的历史，不仅有直接可以观赏的、物化了的表现，也有更多的还是保留在其中的口头的和文献的历史。这是更耐品味的历史内涵。
　　每一个洞天，每一座宫观，都有自己形成、兴废和发展的历史，它们本来就是道教历史的一个侧面。同时，在它们当中，曾经发生过、经历过中国历史，特别是文化史上的一些书件。
　　毫无疑问，道教景观中蕴含着的，首先是道教本身的历史。说到鹤鸣山、青城山，马上使人联想起张道陵创立正一盟威道教的活动 ; 游历陕西户县的重阳宫，道长能在带你游完大殿后再去赏读后边的碑林，并且向你娓娓道出王重阳创立全真道的历史 ; 迈入江西西山万寿宫，可以探访道教净明派的由来。至于龙虎山、茅山等道教诸派宗坛中，更隐藏着不知多少道教史上的重大事件。
　　不仅如此，由于道教本来就是中国文化独特的一支，道教文化在中国文化史上有着其他文化支脉不可替代的地位。因此，在道教景点中，蕴含着的尚有中国乃至世界文化史上的重要事件。比如，在各景点中常常可以见到的炼丹遗址，便连系着世界科技史。杭州的葛岭、广东罗浮山冲虚古观前，以及江西三清山，都是传说为葛洪丹灶、丹井等等遗址，有的，只是后人的纪念性建筑。但从中，我们却可以透视出曾经在这些地方及其他地方发生过的历代道士的炼丹活动。如众所周知，道教中承传千年的炼丹术，是世界实验化学的先驱。炼丹术经过阿拉伯人的中介，传入欧洲，才导致了近代化学的诞生。而在炼丹中的副产品——黑火药，则是中世纪的三大发明之一，在人类的文明史上有着不可估量的巨大作用。道教景观中这类遗迹，深焉者可以引来化学史的专家的研究，浅焉者也可打动匆匆来访的游客的探奇。除化学史外，在文学、艺术等等文化领域中，也有大量的与道教景观有关的事件，以及传世佳作。人们游览这些景观时，也同时回顾起这些历史，品味出当时历史条件下人们的精神、情趣和创造的成果。
　　道教景观的历史内涵多与文化史有关，但也常常越出文化史，在更加广泛的时空领域产生影响。试举北京白云观为例，作为全真龙门派的祖庭，在全真道的发展史上有崇高地位。同时，它又使人们回想起中外交通史上的一件大事，邱处机远赴大雪山 ( 今阿富汗境内 ) 弘道。白云观，其前身为唐天长观，因邱处机自雪山回来后，被迎居于此，后谕旨改为长春宫，邱祖羽化后，弟子在长春宫修白云观，内有处顺堂，安置邱祖的遗蜕。元末长春宫渐颓，明代起以处顺堂为中心渐加扩充，成为今白云观。白云观的历史，与邱长春的远赴雪山分不开。当年成吉思汗远征西域，慕邱长春之名，派专使到山东迎诣，邱长春率领弟子十八人于 1220 年正月起程，跋涉二年，历经大漠和雪山之险，方在 1222 年 3 月抵达今阿富汗境内库什山的大汗行营。他的弘道活动，不仅促进了国内各民族的文化交融，而且远在七百年前就将道教的行迹带到了国外，在中西交通史上留下了印记。对这一事件的了解、述说，便成了游观时兴味盎然的探奇。值得指出的是，长春子的学生李志常曾编写了一本《长春真人西游记》记述其事。由于道门少于世事，这本著作一直隐于宫观，外人只是闻其名，而未见其面。明末，当一本当时没有具名的、描写唐僧取经的《西游记》流传时，人们便将之当成邱长春的著作，一些评注者甚至还以全真内丹之学详细注释了《西游记》。这些文学史上的轶事，更增加了品味历史的兴趣。
　　道教与山有不解之缘。不少道教景点，深隐于崇山峻岭之中。因其中独特的地理位置，某些历史事件，比如军事活动就会与之发生联系。武当山紫霄殿中，至今仍保留着红三军指挥部的旧址，原来当年贺龙同志率部于此周围建立根据地。贺龙与山上道长关系融洽，其部下郭凡( 柳直洵) 还曾拜道长为师学习武艺。上清宗坛茅山，曾是陈毅同志领导的新四军的重要根据地，小茅峰深处的乾元观，正是他的指挥部。道友们支持新四军的抗日活动，因而茅山的宫观曾遭日寇的炮火轰击，不少道友牺牲在日寇的屠刀下，他们英勇不屈的斗争，为茅山增添了光辉。广东的罗浮山，曾经是党领导的抗日武装东江纵队的根据地，所有这些，都使道教景观的历史内涵更为丰富，更耐品味。
三、富有情趣的历史影响
　　游历道教景观，除了上面说的以外，还可感受到它们深远的历史影响。这种历史影响，有时会以可以感触、可以直接鉴赏的面貌表现出来，特别是它们在一方民俗中的印痕，更是富有情趣。
　　道教的洞天福地、宫观名胜的历史影响，当然是多方面的，在道教史上，中国文学史、艺术史上，乃至在王朝更迭的政治史上，都留下了它们的印痕，不过论其广泛和稳固，莫过于它在民间的影响。因为这种影响，已经深深地扎在民众生活的土壤中，已经成为一方一地的乡风民俗，在民众世代的传承中，稳固地保留着。单说其中与旅游有关的话题。
　　顾颉刚先生在《妙峰山》一书中说道 : “承受香火的佛道教庙宇是各地方都有的。例如我们苏州，有玄妙观、北寺、蛇王庙、七子山、穹窿山、上方山、观音山……许多地方。但这种地方的势力并不大，不过受到百里以内的香火。势力大的，如浙江的西湖和普陀，山东的泰山，安徽的九华，山西的五台，四川的峨嵋，广东的罗浮，江苏的栖霞和茅山……，他们可以吸致千里以内甚至于数千里以内的香火。所以然之故，只因他们的风景是特别好，能给予进香者以满足的美感，因此便在他们的意想中更加增神灵的美妙的仪态。” ( 上海文艺出版社， 1988 年影印本，第13 页 ) 按顾先生的说法，谈及风景与朝山进香的关系，确有独见，然而他的话只说了问题的一半。因为风景秀美之处到处都有，为什么这些唯独佛道教的名山能吸引那么多香客兼游客 ? 其实，使这些地区得到开放的真正动力，是它们在佛道教中的地位。佛山不论，仅以道教名山宫观而言，它们最大的吸引力是获得民众的信仰。它们或者是道教某些派别的宗坛、祖庭，或者是其中供的“神仙”被认为灵验非常，于是声名远播，吸引着广大的信众，香客在信众的心目中，“灵验”是第一位的，还愿表示着自己信仰的虔诚，进香为的是带福还家。上茅山的香客，“顶宫一住香，印宫一颗印” ( 上大茅峰顶的九霄万福宫进香，到元符宫讨取“九老仙都君印”的印文 ) ，是活动的主要内容，游是附带的、第二位的。就信仰与旅游的关系而言，在现代旅游业没有形成以前，是信带动了游。就道教宗教活动场所与自然风景的关系而言，是前者推动了后者的开发，使之声名远播。“山不在高，有仙则名”，是一个历史的事实。
　　道教景观对民众的这种吸引力，是历史造成的。古代信息传递不方便，更没有广告，只有通过身临其境者的述说、诗文的传抄，去扩大那些景观的知名度，更重要的是通过香会、庙会一类活动稳固地扩展它们的影响。
　　这种历史造成的吸引力，在当代的旅游活动中，又会转变为新的景观，新的观赏内容。当你在武当山的小道上攀登时，迎面遇上打着旗帜的朝香团时，你不觉得是种情趣 ? 当你在茅山香期的熙攘人群中看到上山者的虔诚目光和进香后下山者喜悦而安详的脸色，你不觉得是摸到了民众心态的某一脉搏 ? 在这里，道教的历史影响，获得了富有情趣的活生生的表现形式。
　　道教名山宫观的这种历史影响，已经渗透到了民间的风俗中，成为民间信仰风俗的一部分，它又常与民间的生产、集市等经济习俗，更多地是与戏剧、杂耍、歌舞、灯彩等娱乐结合在一起。这样，那些名山、宫观的香期、会期便成为民俗风情的大展示，人们置身其中，既是一种参与，又是一种鉴赏。比如陕西耀县药王山，以纪念唐代高道、药王孙思邈而出名，千余年来，每属药王生日，周围群众都要举行纪念性的庙会。这是乡村，就城市而言，也是如此。例如旧时的北京人，有所谓逛庙的习惯，《清稗类钞》记述说，“ ( 北京 ) 各庙，辄有市集，百货充盈，游人纷沓，俗谓之‘逛庙'。逛者游也。”据该书所记，逛庙有定期，差不多每个月都有某些庙有固定的会期，这些庙，有属于佛道二教的，也有属于民间神祀的，大抵说来，以属于道教及归属于道教的民间神祀为多，如财神庙、白云观、蟠桃宫、妙高峰、北顶 ( 皆为碧霞元君庙 ) 、都城隍庙、城隍庙、关帝庙、灶君庙等等。到了当代，这些庙的庙会在民间的影响程度有过很大变化，但其历史影响犹在，在某些庙中，这类民俗风情的大展示，仍然能吸引大批的游客，最为典型的就是白云观的民俗庙会。本来，每年的 正月十九日游白云观，是旧北京的一项传统民俗活动，当地称这一天为燕九节，又称阉九，又称会神仙，近几年来，北京白云观举办的民俗风情庙会，正是对它的继承，又加入了新的内容。据《中国道教》的报道，这些庙会年年都很旺盛，至期游人如织，热闹非凡。这类庙会中的民俗风情，是道教的历史影响深入民间的典型表现，从旅游开发的角度说，它本身就是一种资源，从鉴赏的角度言，它使旅游者领略到民众生活中质朴、热烈而充满向往的心态，以及由这种心态外化而成的俚俗的、色彩鲜明的美 ; 看到中国民众将历史影响溶解在当下的精神享受中的传统。
