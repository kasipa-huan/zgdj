　　“万古长生，不用餐霞求秘诀 ; 一言止杀，始知济世有奇功。”这是清代乾隆皇帝给予邱祖一生的高度评价。
　　邱祖讳处机，字通密，号长春子。南宋高宗绍兴十八年 (1148) 生于登州栖霞 ( 今山东栖霞市 ) 。宋理宗宝庆三年 (1227) 七月九日，逝世于燕京长春宫，弟子们在长春宫东建白云观，于观中专设处顺堂以安邱祖仙枢。其后白云观逐渐代替原长春宫的地位，历元明清至今，成为道教全真派“三大祖庭”之一，而以“全真天下第一丛林”的美誉驰名天下。今北京白云观中邱祖殿前有长春真人事实碑，殿内有长春真人雪山应聘图、长春真人像，另有一木制巨瓢瘿钵，传为邱祖遗物。瘿钵础石之下，即埋藏着邱祖的遗蜕。
　　邱长春出生显族，却自幼好学道。《金莲正宗记》卷四说 : “子家世栖霞，最为名族。敏而强记，博而高才，眉宇闲旷，举措详雅，善相者言足下有龟文，必为帝王师。年未弱冠，酷慕玄风，非长生久视之说不道也，非骖鸾跨凤之语不咏也。”
　　金熙宗大定七年 (1167), 闻全真教主王重阳在昆嵛山烟霞洞讲道，于是登门请教。重阳见而爱之，与语终夕，玄机契合，故赠诗曰：“细密金鳞戏碧流，能寻香饵会吞钩。被予缓缓收纶线，拽入蓬莱永自由。”从此，邱长春拜师重阳，旦夕亲侍左右，先后辅助王重阳，四处布道。于大定八年 (1168) 八月从烟霞洞入文登县姜氏庵，建立“三教七宝会” ; 次年四月，又应周伯通之请，迁居宁海，创建“三教金莲会” ; 又赴福山，建“三教三光会”；然后再到登州，建立“三教平等会”。一年之间，王重阳在山东半岛沿海地区，便建立了五个全真教的会社，其中邱长春与马丹阳、谭处端、刘处玄四子所起的作用是相当大的。正因如此，邱长春深得重阳的器重，史称丘祖从师，“东别海山，南游汴梁，岁月既久，志气弥坚，师资之道莫过于胶漆，玄关妙键，多所启明。”
　　大定九年 (1169) 秋，邱长春与马、谭、刘三人同随王重阳返还陕西，不料途经开封时，重阳染病于旅舍。大定十年 (1170) 年 正月初四 ，王重阳召四子于榻前，说 : “丹阳已得道，长真已知道，吾无虑矣。处机所学，一听丹阳、处玄、长真，当管领之。又顾处机曰 : 此子异日地位非常，必大开教门者也。”①言毕书诗一首，奄然而逝。即将传教的大任委托于马丹阳、邱处机。
　　邱处机与马、谭、刘护送王重阳遗体回陕西终南山刘蒋村安葬，庐墓近三年。至大定十四年 (1174) 八月，与马、谭、刘在刘蒋村分手，乐秦、陇之风，西游凤翔，居磻溪六年，龙门山七年，乞食饮以度日，声名大显。《甘水仙源录》卷二曰 : “师乃入磻溪，穴居，日乞一食，行则一蓑，虽箪瓢不置也，人谓之蓑衣先生，昼夜不寐者六年。既而隐陇州龙门山七年，如在磻溪时，其志道如此。既成，远方学者咸依之。”前后十三年，是邱处机潜心修道、读书著述时期。他成道于陇州龙门山，故其门徒以此名派，称邱祖一宗为全真道龙门派。
　　大定二十六年 (1186) ，应京兆统军夹谷公疏请，邱祖返还终南，修构了王重阳旧居，取名祖庵 ( 或称祖庭、祖堂 ) ，并居庵中宣道，玄风愈振。大定二十八年 (1188) 春，金世宗闻处机之名，遣使召人京师，多次问以至道，对答称旨。史称：“剖析天人之理，演明道德之宗，甚惬上意。”同年八月，返归终南祖庭，世宗赐钱十万，辞而不受。
　　至章宗明昌二年 (1191) ，邱祖东归故里栖霞，大建琳宫，章宗敕赐其额曰“太虚观”。此观颇为壮丽，“气象雄伟，为东方道林之冠”。邱祖以太虚观为基地，致力传道宣教，与众多达官贵人和当代名臣多有往来，社会影响渐渐扩大。泰和七年 (1207) ，元妃遥拜邱祖为师，并赠《玄都宝藏》一部六千余卷，驿送栖霞太虚观，作为镇观之宝。贞佑二年 (1214) ，山东杨安儿等起义，登州、宁海皆动荡不安，朝廷乃请邱处机进行抚谕。邱祖受命招安，“所至皆投戈拜命，二州遂定。”（2）名声大振。兴定三年 (1219) ，邱祖迁居莱州昊天观。金、南宋先后遣使来召，各欲为其所用，但皆不应诏。州牧问其原因，邱祖曰：“吾之出处，非若辈所可知，他日恐不能留耳。” （3）
　　同年五月，元太祖成吉思汗自乃蛮国 ( 今蒙古境内 ) 派使臣刘仲禄召清，邱祖权衡时局，毅然许行。事实证明，邱处机的这种政治选择，不仅对于他个人，而且对于整个全真道的命运都有重大的意义。这是邱处机之能成为亨誉道坛，一代宗师的转折点，也是全真道获得大发展的基点。
　　兴定四年 (1220) 正月，邱处机率领随行弟子十八人，自莱州启程北上。二月抵燕京，馆于玉虚观。四月，设醮于燕京太极宫，亲登宝玄堂传戒。醮毕，离燕出居庸关，五月至德关府 ( 今河北涿鹿 ) ，居龙阳观，并于中元节在观中设醮传戒〔期间曾作诗以寄燕京道友，表示了他济世救民的宏愿。诗云 : “十年兵火万民愁，千万中无一二留。去岁幸逢慈诏下，今春须索昌寒游。不辞岭北三千里，仍念山东二百州。穷急漏诛残喘在，早教身命得消忧。”八月抵宣德州( 今河北宣化 ) ，居朝元观，又于观中构建圣殿及祖堂。次年 (1221) 二月出张家口，五月抵陆局河，沿途受到士庶奉香火拜迎，王公大人也争相进诗勉志，使全真道在河北的影响非常广泛。七月至阿不罕山，因前有大山广泽，无法行车，留弟子宋道安等九人于当地立栖霞观，自率刘志坚等九人轻骑而往。中秋日抵金山，八月越阿尔泰山，继入新疆，于十一月抵塔什干 ( 前苏联境 ) ， 十二月至撒马尔干 ( 前苏联境 )。 到第二年 (1222) 四月五日，终于到达目的地——设于阿姆河南岸 ( 阿富汗北境 ) 的成古思汗的军营。前后历时二年半载，行程万里，爬山涉川，跨戈壁，渡荒漠，“不辞暴露于风霜，自愿跋涉于沙渍”，这对于一位年过七十余岁的老人而言，其辛劳是可想而知的。（4）
　　对此，成吉思汗十分赏识。因此在邱处机到达之后，立即便予召见，并对他说 : “他国征聘皆不应，今远逾万里而来。朕甚嘉焉。”（4）并约日后正式召见，向处机问道。据《元史·释老志》所载 : “太祖时方西征，日事攻战。处机每言 : 欲一天下者，必在乎不嗜杀人。及问为治之方，则对以敬天爱民为本，问长生久视之道，则告以清心寡欲为要。太祖深契其言，命左右书之，且以训诸子焉。”对于邱处机的应答，成吉思汗十分满意，谈话中对处机不称其名，但尊称“神仙”。召对之后，又集而合太子诸王大臣曰：“汉人尊重神仙，犹汝等敬天，我今愈信真大人也，乃以师前后奏对语谕之。且曰：天俾神仙为联言此，汝辈当铭诸心。”⑤
　　元太祖十八年 (1223) ，处机请求东归承准，成吉思汗授以虎头金牌及玺书，命其掌管天下道教，诏免道门赋役。《七真年谱》说 : “三月七日得旨东还，赐号神仙，俾掌管天下道门大小事务，一听神仙处置，他人无得干预。宫观差役尽行蠲免，所在官司常切卫护。”得旨之当月，邱处机率随行弟子启程，经蒙古南道，自科布多至金山，偕门人宋道安等人东返。成吉思汗仍派宣差阿里鲜等人护送，沿途受到蒙古官员们的敬奉及百姓的欢迎。经过一年的跋涉，于十九年(1224) 三月还抵燕京。
　　处机一回燕京，遂应燕京行省金紫石抹公等人之请，入住天长观 ( 即后来的长春宫、白云观 ) 。从此，该观便成了全真道的中心。“厥后道侣云集，玄教日兴。乃建八会，曰平等，曰长春，曰灵宝，曰长生，曰明真，曰平安，曰消灾，曰万莲。会各有百人，以良日设斋供奉上真。”（6）《长春真人西游记》卷下说 : “自师复来，诸方道侣云集，邪说日寝，京人翕然归慕，若户晓家喻，教门四辟，百倍往昔。”商挺《大都清逸观碑》亦云 : 邱祖谓道众曰 : “今大兵之后，人民涂炭，居无室，行无食者，皆是也，立观度人，时不可失，此修行之先务，人人当铭诸心。长春既居燕，士庶之托迹，四方道侣之来归依者，不啻千数，宫中为之填咽。”（7）
　　正是为了救民众于刀兵之中，邱处机四处营造道观，传戒度民、如在燕京地区，他以长春宫为中心，令其弟子们广建观宇，遂使这个地区成为全真道观和道士最集中的地方。据《顺天府志》所载，燕京及附近地区共有百余所宫观，其中大都建于邱处机至其徒李志常掌教期间。如李守征建有固本观，潘德冲建有清逸观，宋道安建有长生观，宋德方建清都观，王志明建洞神观，何志邈建兴真观，何守夷建清真观，李志方建真元观，霍志融建崇元观，陈慧端建玉华观，梁慧真建玉真观，陈守玄建冲微观，夏宗道建玄禧观，孟道宣建昭明观，皆先后建造于这段时间。
　　在河北、河南地区，众多全真道徒亦遵邱祖之训，“立观度人，将迎往来道众为务。”如李志柔格遵玄训，于漳州建长春观，于大名路建奉天观、栖真观，其余的神霄观、大同观、洞真观等大小宫观二三百余所，遍布河北。刘志源也在大名路建了许多宫观，著名的有太清观、万寿观、洞玄观、天庆观、参天观、洞真观、重阳观、明真观、清真观、东华观等。《重修缔造南山上清太平宫记》谓其“建宫立观，亦二百余所”，“度门弟子三千余人”。此处，赵志渊亦在河北建观授徒。《甘水仙源录》卷八谓他主持神霄万寿宫，在“大名、磁、相之间度学者凡数百人，立庵观十有余所”。
　　陕西是全真道的发祥地。邱处机于大定二十六年，即在终南山建立祖庭。在他主持燕京长春宫、掌管天下道教后，其门徒很快便在其地大建宫观。如尹志平曾在陕西营建二观四宫，度弟子千余人。李志常改终南灵虚观 ( 即祖庭 )为重阳宫，并敕于善庆为重阳宫住持，主领陕右道门事，《终南山祖庭仙真内传》卷下曰 : “不十载间，雄宫杰观，星罗云布于三秦之分矣。”此外，綦志远在主持重阳宫期间，亦度门弟子数百人，建立宫观二十余所 ; 冯志亨也在燕秦一带或修或建宫观百余所，《甘水仙源录》卷六称 : “自燕至秦三千余里，凡经过道家宫观，废者兴之，缺者完之，到百余所”
　　山东为全真道的基地，北七真皆出自山东，北七真弟子亦多山东人，故全真道在山东的宫观亦相当多。除邱处机所建栖霞太虚观外，其弟子亦很快在各地建造了一大批宫观。如张志渊在郓城建白云观，度弟子千余人 ; 周道全在济州创白云观，度弟子数百人，“悉立庵观于齐鲁之间”。据《古今图书集成·职方典》山东各府部记载，邱处机所在的金元之际，先后在山东建造了全真道观百余处所。如登州府有紫府洞、云光洞、积金观、修真观、祝圣道院、炼阳观、丹阳观、朝元观、神清观、玉虚观、元都宫、东华观、契遇观，莱州府有迎祥观、先天观、万寿宫、崇德宫、盘石观，云山观、古现观、德真观、玉清宫、天仙宫、玉真宫、栖真观、集元观、太微观、修真庵、通真宫、寓仙宫、华楼宫、清虚观、通真观，青州府有龙华观、修真观、天齐观、全真观、修真宫、遇仙观、太清观、栖灵观、女真观、神清观、龙泉观、云溪观、迎仙观、无为宫、朝元观、栖真观，衮州府有万寿宫、元真观、朝元观、玉泉观、龙泉观、重阳观、清真观、太清观、常清观、龙门观、三清观、万春宫、神游观、白鹤观、玉霄观、瑞云宫、弘阳观、玉清观、通仙观、神霄观、清神观、青龙观、通仙观、全真观、万寿宫、迎祥观、崇真观、太虚观、南天观、天平观、保真观、灵泉院、香严院、洞渊观、长春观、龙兴观，济南府有长春观、升元观、玉清观、洞真观、高真观、白云观、太微观、丹阳观、永兴观、栖霞观、修真观、明德观、洞明宫、逸真观、佑德观，如此众多的全真道观遍布山东，说明全真道在该地区的影响甚大。
　　此外，邱处机的弟子们亦在山西建造了一批宫观。如潘德冲在芮城县永乐镇创建了有名的纯阳万寿宫 ( 俗称永乐宫 ) ，其后成为全真道三大祖庭之一。宋德方在平阳主持编纂《元道藏》期间，“犹假余力，建立宫观，自燕齐及秦晋，接汉沔，星分棋布，凡百余区”（8）
　　概括而言，自邱处机进见太祖，主持燕京长春宫的期间，他的众多弟子散布北方，进行了大规模的建观收徒活动。全真道三大祖庭，即户县重阳万寿宫、永乐纯阳万寿宫和北京长春宫 ( 后扩建为白云观 ) ，皆创建于此时，从而使全真道的发展达到了鼎盛时期，不仅远远超过了王重阳、马丹阳主教的时代，而且也超过了同在北方的真大道教和太一道，这就奠定了邱处机在道教发展史上相当显赫的历史地位。
　　另一方面，为了制止元代统治者的野蛮杀戮，邱处机一方面劝阻成吉思汗少杀少诛，一方面利用道观，广发度牒，其数多达数万。元姚牧庵《长春宫碑铭》曰： “长春癸未至燕，年七十六矣。而河之南北已残，首鼠未平，鼎鱼方急，乃大辟玄门，遣人招求于战伐之际，或一戴黄冠而持其置碟，奴者必民，死赖以生者，无虑二、三巨万人。”《元史·释老志》亦说：“时国兵践蹂中原，河南北尤甚，民惧俘戮，无所逃命，处机还燕，使其徒持牒招求于战伐之余，由是为人奴者得复为良，与滨死而得更生者，毋虑二、三万人，中州至今称道。”这种独特的“救民”措施，深受当时陷入战乱之中的群众欢迎，故世人给予 了高度的评价。
　　如元陈时可曰 : “师诚明慈俭，凡将帅来谒，必方便劝以不杀人，人有急必周之，士有俘于人者必援而出之，士马所至必以师之名脱欲兵之祸者甚众。度弟子皆视其才何如，高者挈以道，其次训以功行，又其次化次罪福，罔有遗者。故其生也，四方之门人丹青其像事之 ; 其殁也，近者号慕，远者骏奔，如考妣焉 ; 及其葬也，会者又万人，近世之高道福德兼备，未有如师者。”⑨秦至安曰 : “当蒙古之锐兵南来也，饮马则黄河欲竭，鸣镝而华岳将崩，玉石俱焚，贤愚并戮，尸山积而依稀犯斗，血海涨而仿佛弥天，赫威若雷，无赦如虎，幸我长春邱仙翁，应诏而见，一见而龙颜稍霁，再奏而天意渐回，诏顺命者不诛，许降城而免死，宥驱丁而得赎，放俘口以从良，四百州半获安生，数万里率皆受赐，所谓展臂拒摧峰之岳，横身遮溃岸之河，救生灵于鼎镬之中，夺性命于刀锯之下，不啻乎百千万亿，将逾于梯穰京垓，如此阴功，上通天意，固可以碧霄往返，白日飞升。”（10）
　　需要说明的是，邱处机不仅致力于全真道在组织上的发展，同时在完善全真道理论思想方面也颇多建树。他早在隐居磻溪、龙门时期，即潜心学习道经佛典、儒书诗文，有着很高的理论修为，如同元人陈时可所说 : “师于道经无所不读，儒书梵典亦历历上口，又喜属文赋诗，然未始起篙，大率以提唱玄要为意，虽不事雕镌而自然成文，有《磻溪》、《鸣道》二集行于世。”（11）
　　在内丹修炼方面，邱处机著述《青天歌》、《大丹直指》，系统地介绍全真丹法性命双修的奥秘，成为内丹派北宗的重要经典。其《青天歌》三十二句，乃按《度人经》所言三十二天运化之道，阐述修性复命功夫。“前十二句乃明修性之本体，中十二句为复命之工夫，末后八句形容性命混融、脱胎神化之妙也。”而被后人誉为“文简而理直，实修真之捷径，入道之梯阶。”（12）
　　其《大丹直指》上下二卷，内收炼丹行动图二十三幅，书中首先阐明内丹修炼的基本原理，认为人与天地禀受一同，人因父母二气交感而成胎，未生之前在母腹之中，受母气滋养，混沌纯一，是为先天之气 ; 降生之后，先天气散于九窍，呼吸从口鼻出入，是为后天之气，先天真气逐时耗散，以致病夭，要想健康长寿，就需修炼：“先使水、火二气上下相交，升降相接，用意勾引，脱出真精真气，混合于中宫，用神火烹炼，使气周流于一身，气满神壮，结成大丹。”
　　这接着，书中用图、诀、义详述了九种修炼功法，其中五行颠倒龙虎交媾法、五行颠倒周天火候法、三田返复肘后飞金精法，为小成法，炼之可以补虚益气，返老还童。三田返复金液还丹法、五气朝元太阳炼形法、神水交合三田既济法，为中成法，炼之可以延年益寿，长生不老。五气朝元炼神入顶法、内观起火炼神合道法、弃壳升仙超凡入圣法，为大成法，至此可显各种神通，获大自由大圆满，为丹法修炼的最高境界。三成九法，由浅入深，步步登堂证验，完整系统地阐述了北宗内丹法诀。
　　九种丹功，核心都是修炼性命。邱处机总结说 : “金丹之秘，在于一性一命而已。性者天也，常潜于顶 ; 命者地也，常潜于脐。顶者性根也，脐者命蒂也……只是性命二物，千经万论，只此是也。”
　　在北七真中，唯邱处机年寿最高，享年八十。这当与他精通丹法有关。自元以来，道教著名炼养家，不少出自龙门派法统。如伍守阳、柳华阳承邱祖之法脉，门庭极盛，而开龙门伍柳派，闵一得师龙门派高道高东篱、金怀怀、李蓬头等，又倡龙门派西竺心宗于世。明末清初的王常月，得清顺治、康熙的支持，入主北京白云观，说律传戒，广度弟子，成为振兴龙门的得力干将。在此基础之上，又相继分衍出许多龙门支派，如太微律院派、冠山派、云巢派、天柱观派、金鼓洞派、长春宫派、机神殿派、桐柏宫派、三元宫派、半持庵派、开化院派、斗姆宫派、碧洞宗派、千山派、霍山派、觉云派、南宫派、千峰派，成为道门中最具影响的一大宗派。
　　毫无疑问，这一切都是与邱处机的辛劳传教、大辟玄门之风的功绩密不可分的。正因为他于全真道的发展功劳甚大，道教中人往往把他和教主王重阳相提并论，誉之云 : “重阳再弘法教，专以性命之说普化三州，同归五会，以金莲居其首，东游海上，度者七人，经柔弱谦下为表，以清静虚无为内，以九还七返为实，以千变万化为权，更其名曰全真，易其衣而纳甲。逮我长春子邱神仙，受皇帝之宣，应阴山之聘，劝之以减酒色，戒之以少杀戮，一言恺切，万国生春，救亿兆于鼎镬马锯之间，人心归向者如百川赴海而莫之能御也。牧竖尧童，咸知稽道 ; 东夷西戎，皆泳步虚。家家谈道德之风，处处讲希夷之说，懒衣髯髻，云连乎道路之间 ; 琳宇瑶坛，星布乎山泽之下，自轩辕以来，教门弘盛，未有如今日者，是教也源于东华，流于重阳，派于长春，而今而后，滔滔溢溢，未可得而知其盛也。”（13）
