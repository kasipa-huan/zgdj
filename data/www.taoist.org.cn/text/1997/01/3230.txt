　　杜光庭 (850 — 933), 唐末五代时的著名道士、道教学者，著作丰富。他所撰的《虬髯传》传奇小说，更是脍炙人口，流传极广，至今不衰。
　　《虬髯客传》是一篇著名的神仙小说，说的是隋朝末年，天下大乱，群雄逐鹿，李靖以布衣向隋朝权臣杨素献策，当李施展辩才时，立于前的杨府歌妓红拂女张氏，慧眼识英雄，见李才华出众，遂产生爱慕愿托终身之情，毅然于当夜五更，化成男装，窃出“尸居余气”的杨素府门，私奔李靖住处，相约逃往太原，于途中灵石旅舍，又遇到豪侠虬髯客，三人遂结义，红拂女、李靖称虬髯客为张三兄，虬髯客称红拂为张一妹。虬髯客本想起兵夺取隋朝天下，后通过刘文静两次会晤李世民于太原，见他“神气扬扬，貌与常异”，“真天子也”。特别经道兄的指点 : “此世界非公世界，他方可也。”才心死认输说：“今既有主，住亦何为。”于是慨然将全部家财，赠于李、张夫妇，让李辅佐李世民兴唐，而自己与妻从一奴，远适扶余国，杀其主，自立为王。
　　由于故事情节曲折感人，文笔细腻生动，塑造的人物红拂、李靖、虬髯客栩栩如生，所以为当时及后世人所喜爱，称他们为“风尘三侠”。
　　现将此传奇故事历代流传情况及其对后世文学、戏曲、美术等方面产生的影响，分述如下：
　　一、历代流传情况
　　鲁迅先生说 : “唐代传奇之文……如柳　《上清传》、薛调《无双传》、皇甫枚《非烟传》、房千里《杨娼传》等，亦皆有造作。而杜光庭之《虬髯客传》流传乃独广”。 ( 《中国小说史略》 )
　　《虬髯客传》最早见于杜光庭《神仙感遇传》初名《虬须客》，文字简略。
　　宋李昉 (925 — 996) 等于宋太宗太平兴国二年 (977) 三月奉敕编纂的《太平广记》卷一九三收入杜光庭《虬髯客》，与现在流传于世的《虬髯客传》文字同，只少数语句稍有差异。
　　宋真宗大中祥符五年 (1012) 冬，张君房奉命主持编纂的《大宋天宫宝藏》，至天禧三年(1019) 春成书进之。君房又摘录书中精华近万条，名其书曰《云笈七签》，卷一一二收有杜光庭《神仙感遇传·虬须客》，又见于《道藏》恭字下。
　　写到这里，发现一个问题：为什么《太平广记》既已载有杜光庭文字详细的《虬髯客》，而君房的《云笈七签》和《道藏》均不收录，却采用了文字简略的《虬须客》呢 ?
　　据上海古籍出版社影印文渊阁《四库全书》本《太平广记·出版说明》称 : 此书于宋太宗太平兴国二年 (978) 完稿呈进，六年正月开雕，开雕后言者以为非学者所急，收版于太清楼。因此，宋、元年间罕有流传。我推测张君房有可能没见过早已收版于高阁的《太平广记》，因此，出现了上面提及的这一问题。
　　宋洪迈 (1123 — 1202) 《容斋随笔》“王珪李靖”条，称杜光庭有《虬须客传》。
　　元初官修《宋史·艺文志》子部小说类，始称“杜光庭《虬客传》一卷”
　　元末明初人陶宗仪辑《说郛》卷一百十二，载《虬髯客传》下题唐张说撰。清乾隆时陈莲塘辑《唐人说荟》 ( 坊间刻本题名《唐代丛书》 ) 收录《虬髯客传》也沿袭陶氏之说。查张说著《张燕公集》，不见有《虬髯客传》的记载，且自两宋以来，都认为此传奇是杜光庭之作，故“张说撰”之说，不足信。
　　杜氏本篇又见于明人顾元庆辑《顾氏文房小说》， 鲁迅先生以《顾氏文房小说》为基础，校以《太平广记》，将本篇收入其所编《唐宋传奇集》中。
　　今人叶桂刚、王贵元主编的《中国古代十大传奇赏析》一书，将杜氏本文列为中国古代十大传奇之一。
　　这篇传奇还散见于各种文言小说、古典文学选、文言文选中，并大都配有评析赞颂文章。
　　二、对后世文学、戏曲的影响
　　在戏曲方面，根据杜氏《虬髯客传》编成杂剧的，首先是明代戏曲作家张凤翼 ( 1527—1613) ，他改编的《红拂记》，既本杜氏原著，又吸收了唐代孟词　《本事诗》所记徐德言与乐昌公主“破镜重圆”的故事写成。为使读者阅读方便，顺便将此故事情节介绍如下：
　　乐昌公主是亡陈国主陈叔宝的妹妹，陈亡，落入杨素府中为婢妾，与红拂女同侍杨素，两人相处甚洽。当红拂私奔李靖，杨并未深究，现在又见乐昌公主常对半镜落泪，遂问缘由，乐昌便如实告曰：于国亡丧乱之际，与驸马徐德言离散时，以分镜为记，指望夫妻日后重聚。陈又与她约定：他日必以正月十五日，将半镜卖于都市，我当于是日至京往访。杨素听后，颇同情，遂于是日派一苍头持破镜到街上以高价呼卖，入皆笑之。俄顷，德言至，引苍头至其住处，出半镜以合之。苍头便引他至杨府与乐昌公上相见，夫妻庆幸“破镜重圆”，并得到杨素的资助，夫妻同归江南，竟以终老。
　　张风翼的《红拂记》，流传版本很多，有明毛晋编《六十种曲》本、《古本戏曲丛刊初集》本等。
　　张凤翼之后，是凌濛初 (1580—1644) 改编的《红拂三传》、凌字玄房，号初成，是明代著名的文学家、戏曲作家。曾根据杜氏《虬髯客传》故事改编一出名为《识英雄红拂莽择配》的杂剧，此剧又名《北红拂》。剧情与杜氏原著同，只是按戏剧惯例，在出场时增添了介绍自己身分的部分。剧本格局，以元杂剧为规范。作者又写了以李靖为主人公的杂剧，惜已失传。还写了《虬髯翁》 ( 又名《扶余国》 ) 以一事三传，故被称为《红拂三传》。当时的戏曲家、理论家祁彪佳 (1602 —1645) 在其所著《远山堂剧品》评《红拂三传》说 : “凌初成既传红拂，再传卫公矣，兹复传虬髯翁，岂非才思郁勃，故一传再传，至三而始畅乎 ! ”
　　《虬髯翁》杂剧唱词片段：
　　第一折欲夺天下，上场唱 : “俺则为四海烟霾，九重翠盖将倾败。捷足高材，此际可便对宁耐。”“看霎时把锦绣山河改。”
　　第二折见了李世民“真命天子”上场唱 : “长揖罢，吓得俺身躯退，瞥眼间，搅得俺心窝碎”。
　　第三折认输，赠家财，唱段略。
　　明人张太和，字幼于，号屏山。浙江钱塘人。约万历十年前后在世。善为曲，著有《红拂记》传奇一本。极为汤显祖所赏 ( 见《曲录》 ) 。
　　稍后于张太和的戏曲作家，名“近斋外翰”也著有《红拂记》 ( 《曲品》著录 ) 。
　　由“红拂”敷衍为“女丈夫”的戏曲作家，是明末刘方，他著有《女丈夫》传奇。
　　明末著名的文学家冯梦龙 (1574—1646) 长洲 ( 今苏州 ) 人。重视小说、戏曲，写有《女丈夫》传奇剧本。就是根据张凤翼《红拂记》和刘方《女丈夫》、凌濛初《虬翁》合并改编的，削去《红拂记》里乐昌公主和徐德言夫妻离合故事，情节较集中。
　　《红楼梦》也称“红拂”为“女丈夫”。曹雪芹对“红拂”勇于冲破封建势力的压制和其对时局的深刻见解，予以热情歌颂。如《红楼梦》第六十四回，通过林黛玉的赋诗赞美曰 : “长剑雄谈态自殊，美人巨眼识穷途。尸居余气杨公幕，岂得羁縻女丈夫。”
　　近代剧作家罗瘿公，名谆融，字　东，号瘿庵，晚号瘿公。广东顺德人，生于北京，与梅兰芳、王瑶卿等人熟识。为帮助程砚秋在倒嗓期间，不致因被迫演出而毁嗓，遂根据程自身条件，为他编写了《红拂传》 ( 又名《风尘三侠》 ) 京剧本，成为程砚秋终生的演出剧目。
　　红拂女唱段：
　　( 唱二黄慢板 ) 在相府每日里承欢侍宴，也不过与众女斗宠争妍。虽然是舞衫中常承宠眷，辜负了红拂女锦绣华年。对春光不由人芳心撩乱，想起了红颜老有谁怜。
　　( 唱二六 ) 见春花三月里百花开遍，撩人春色是今年，随风弱柳垂金线，灵和殿中学方眠，红襟紫领衔泥燕，飞来飞去把花穿。半空中又只见游丝百串， ( 摇板 ) 浑不觉　逗境坠花钿。
　　( 白 ) 呀 !( 唱 ) 在店中开妆镜青丝细挽，正对着明窗下自整云鬟。虽然是长途中征鞍不惯，幸得是风尘里未损容颜。我这里新妆罢镜中顾盼，等候了回来时好与郎看。
　　( 唱南梆子 ) 在筵前双手儿分开两剑，好一似双飞燕戏舞阶前，既不是化龙形空中百变 ; 又不是白猿女道法仙传 ; 也不是留仙裙回风自转；也不是汉宫中人柳之眠。多感他张三兄深恩不浅。这一别再相逢不知何年 ?
　　京剧“四大名旦”还有一段争演“四红”的佳话，附带说一下：20 年代以后，京剧“四大名旦”曾经开展一场编演新戏的竞争，使京剧舞台色彩缤纷。特别是他们争演的“四红”，在社会上广为传诵。所谓“四红”，是指以“红”字为首的剧目。梅兰芳率先推出《红线盗盒》，接着程砚秋创演了《红拂传》，尚小云创演了《红绡》，荀慧生创演了《红娘》。很快在社会上掀起一股“红”热。这次竞赛既促进了四大名旦艺术风格的形成和发展，也促进了京剧事业的繁荣昌盛。 ( 摘自《神州轶闻录·文化篇》 )
　　著名京剧女演员新艳秋加叙乐昌公主事，编、演了《二本红拂记》。
　　近现代许多地方戏曲都曾以“风尘三侠”为题材，编成剧本演出，使戏曲舞台百花齐放，异彩纷呈。
　　三、美术作品上的《风尘三侠》
　　以“风尘三侠”为题材的画家，当首推晚清画坛大师任伯年，他一生画了多幅《风尘三侠》，其中一幅是描绘李靖、红拂女于灵石旅舍见到虬髯客的刹那：“张氏以长发委地，立梳床前。公方刷马，忽有一人，中形，赤髯如虬，乘蹇驴而来。”三人各以惊奇的目光相对视，画中间画两棵树，近处一棵，树干上依附两条粗藤，寓“丝萝非独生，愿托乔木”红拂语意。另一幅画李、张夫妇送虬髯客的情景。虬髯客占据画面主要位置，骑头黑驴，身披斗篷，边走边回头，双手抱拳向李、张道别。李、张在画面右边，伫立门前，为他送行。虬髯客形象生动，胯下驴略小，更显得他的体魄魁梧，充满豪气。这幅画见《任伯年画册》。 .
　　此外三侠形象也广泛出现在雕塑、陶瓷等工艺美术作品上，近日翻阅北京著名的《面塑大师汤子博画搞》就载有《风尘三侠》作品。 综上所述，杜光庭的《虬髯客传》给予后代文学、戏曲、美术的发展以积极的影响，由此也可以看出道教神仙文化与我国传统文化相互依存之关系。
