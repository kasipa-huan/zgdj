　　“鲁字公旗，初，祖父陵，顺帝时客于蜀，学道鹤鸣山中。”为什么《后汉书》、《三国志》、《华阳国志》等史书都说张陵学道鹤鸣山中呢 ? 要弄清这个问题，就要先从道教产生的社会基础说起。
　　众多的道教经典和史书都公认，道教形成的社会基础有三 : 一是黄老道学 ; 二是神仙方士的炼丹长命之说 ; 三是民间世俗中的原始巫现鬼道。黄老道学是指黄帝和春秋思想家老子的思想体系，也就是把黄帝、老子的思想学说作为道教的思想理论和教旨教义，如清静无为，修身养性，“清静为宗，虚无为体，柔弱为用。”对于世界的认识“道生一，一生二，二生三，三生万物”。对于宇宙的认识则是“人法地，地法天，天法道，道法自然”。黄老道学原本就是封建帝王用似统治国家的一种方法，为古代统治者所操纵和利用。
　　神仙方术是方士们为了满足封建统治者追求长生不死，永远统治人间，永远享受人间的荣华富贵而臆造出的炼仙丹、求仙药，寻求长生不老、飞升成仙之术，并把阴阳五行之学说贯穿于神仙方术之中。
　　民间原始巫觋鬼道是古代人类对自然界缺乏认识，对一些自然现象不理解，对日月星辰、风雨雷电产生的畏俱心理或寄托的梦幻想法，逐步发展而来的。早在殷商时期，鬼神祟拜就已经十分盛行，《周易·观卦象辞》中有“圣人以神道设教，而天下服矣。”神道中，这些卜筮吉凶，祈福禳灾的女巫男觋，司掌了龟筮，认为能够沟通天神和人间，传达天意，又能代人表祝愿，向天神卜问凶吉，更能替人治病救灾，免除灾祸，古代广大民众都拥戴和信奉。
　　由于张陵“七岁即通道德经” (《 徐州志》 ) ，“于天文、地理、图书、谶纬之秘，咸贯通焉” (《汉天师世家》) ，“本太学书生，博通五经” (《太平广记》 ) 。说明张陵从小就已精通黄老道学和神仙方术，入蜀鹤鸣山后，又向当地“民夷”学习原始巫觋鬼道，然后将原始巫觋鬼道与黄老道学和神仙方术相结合，才“造作道书”，创立天师道五斗米教的。
　　张陵向“夷”学道，这个“夷”究竟是古代哪个少数民族呢 ? 有人说是羌人，有人说是氐人，本人翻查众多史料文献后认为，“夷”应是古代少数民族炁人。查《辞海》“炁，秦汉间，今四川、湖北一带少数民族交纳的赋税名称。交的钱币叫炁钱，交的布匹叫炁布，这一部分民族也因此叫炁人”。《华阳国志校注》中，刘琳老师对“夷”作了详细的解释。“夷指白虎夷，即炁人或板循蛮，民指汉民。《晋书·李特载记》‘汉末，张鲁居汉中，以鬼道教百姓，炁人敬信巫觋，多往奉之'。《北史》卷六六 :‘巴俗事道，尤重老子之术'。《蛮书》卷十 : ‘夷 ( 白虎夷 ) 事道，蛮事鬼'。皆可征炁人信奉鬼道五斗米教，至唐宋犹然，”刘琳老师引证众多史典，从多方面证实“夷”就是指古代少数民族炁人。王家佑老师的《道教论稿》中有“天师道是先进巴人从原始巴人巫鬼教中改造出来的。”“张陵天师道是吸收了巴族的原始巫术与地区传统民俗而创成的。”“可以肯定，张陵、张鲁的‘五斗米道'是在巴蜀民俗基础上建立起来的。”王老师还用大量篇幅阐述了夷就是巴夷、白虎夷，就是板循蛮，汉至唐时叫炁人的发展过程。《四川古代史稿》《中国通史》和卿希泰教授的《中国道教史》等书籍都用大量篇幅对炁人信奉巫觋鬼道的风俗习惯作了详细记载。
　　賨人由何而来，居住何处呢 ? 晋代史学家常璩的《华阳国志》中载有 : “周武王伐封，实得巴蜀之师。巴师勇锐，歌舞以凌殷人，前徒倒戈，故付称之曰 : ‘武王伐纣，前歌后舞'也。秦昭襄王时，白虎为患，自秦、蜀、巴、汉患之，秦王乃重募国中 : ‘有能杀虎者，邑万家，金帛称之。'于是夷朐忍、廖仲药、何射虎、秦精等乃作白竹弩于高楼上射虎，中头三节，白虎常从群虎，瞋恚，尽搏杀群虎，大（口＋句）而死。 秦王嘉之曰 : 虎历四郡，害千二百人，一朝患除，功莫大焉。欲如要，王嫌其夷人，乃刻石为盟，要复夷人，顷田不租，十妻不算，伤人者不论，杀人雇死倓钱，夷人安之。汉兴，亦从高祖 ，定秦有功，高祖因复之，专以射白虎之事，户岁出賨钱口四十，故世号‘白虎复夷'，一曰‘板循蛮'。汉高帝灭秦，为汉王，王巴蜀，阆中人范目，有恩信方略，知帝必定天下，说帝，为募发賨民，要与共定秦。秦地既定，封目为长安建章乡侯。帝将讨关东，賨民皆思归，帝嘉其功而难伤其意，遂听还巴，谓目曰 : “富贵不归故乡，如衣绣夜行耳。'徙封阆中慈 ( 凫) 乡侯。目固辞，乃封渡沔( 县 ) 侯。故世谓‘三秦亡，范三侯’也。 ( 目 ) 复除民罗、朴、昝、鄂、度、夕、龚七姓不供租赋。阆中有渝水，賨民多居水左右。天性劲勇，初为汉前锋，陷阵锐气，喜舞，帝善之，曰 : ‘此武王伐封之歌也。'乃令乐人习学之，今所谓‘巴渝舞'也。”《集解》中有郭璞的 : “巴西阆中俞水，獠人 ( 即賨人 )即其上，皆刚勇好舞，汉高祖募取以平三秦。后使乐府习之，因多巴俞舞也。”杜佑《通典》中载有板循蛮賨人信奉巫鬼的传统风俗和演变发展的详细记载 . 《华阳国志·李特雄期寿势志》中有 : “李特，字玄休，略阳临渭人也。祖世本巴西宕渠賨人，种党劲勇，俗好鬼巫。汉末，张鲁居汉中，以鬼道教百姓，賨人敬信。值天下大乱，自巴西宕渠移入汉中。魏武定汉中，祖父虎与杜、朴胡、袁约、杨车、李黑等移于略阳，北士复号曰‘巴人'。”以上史料说明，公元前十一世纪，周武王姬发在消灭殷纣王，推翻殷商王朝，建立西周时，就已因賨人具有劲勇的性格而重用了巴西賨人。公元前 306年左右，秦、蜀、巴、汉四郡为白虎所患，秦昭襄王悬重赏召募人杀白虎，巴西古賨人竹箭杀虎，秦昭襄王刻石为盟，重赏賨人。公元前 207 年，汉高祖刘邦召募巴西賨人，攻破秦王朝都城咸阳城，推翻秦朝统治后，让賨人回归巴西郡西间中一带故乡生活，并减轻赋税，故称为“賨人”，也称“白虎夷”或“板循蛮”。由于賨人世代都居住在巴西郡嘉陵江沿岸，有种尚劲勇的性格和世代好鬼巫的习俗，当张鲁在汉中建立政教合一的五斗米教政权时，賨人就从巴西迁移入汉中，魏武帝曹操平定汉中，賨人就又被迫迁居略阳、咸阳和中原各地，后又回归巴西。万绳楠所著《魏晋南北朝文化史》“我国道教的产生和发展”中，也用大量的篇幅叙述了我国古代著名少数民族賨人居住的地域，賨人活动和发展的情况，世代都信奉巫觋鬼道的传统习俗和后来信奉五斗米教的情况。晋元康初年，賨人李特流民起义，在迁居青城山的巴人范长生的援助下，取得“成汉”政权，这实际上也是宗教信仰和民族传统习俗的内在因素和力量的结果。
　　综上所述，由于賨人世代都有信奉巫觋鬼道之传统习俗，当张陵入蜀阆中 ( 《四川总志》 ) ，到鹤鸣山后，就向賨人学习巫现鬼道，并结合已学的黄老道学和神仙方术，著书立说，创立天师道五斗米教，又传授给当地汉民和賨人，因此，史书一致认定 : “张陵入蜀，学道鹤鸣山中。”
