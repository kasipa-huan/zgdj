　　1944 年 12 月 23 日，于四川成都四川联合大学，举行春秋战国二千多年来，最空前盛大的“道家道教与中国文化学术研讨会”。开幕典礼的礼堂内，聚集了 125 位名教授和名学者，在他们的热烈掌声中会议揭幕。会议经过四天百家争鸣而圆满结束 ; 以游鹤鸣山和青城山为其余兴。
　　作者试以诗叙记情怀。
参　加　研　讨　会　有　感
( 十二月二十三日 )
炎黄文化亿千年，道教道家根本连；
　联袂英豪蜀樱下，何曾青史见开先。
游　鹤　鸣　山
( 十二月二七日 )
( 一 )
失群弱羽堕村边， 老妇垂怜鹤足缠 ;
养复寻回比翼侣，常来哀唳主游仙。
( 二 )
溪水清清环抱流， 临风古柏意悠悠 ;
汉陵启教千年秀，久暮仙踪惜此游。
( 三 )
宝殿行墙残烛留， 山坡足印密稠稠 ;
应知盛典开光日，定是如潮香客酬。
游　青　城　山
( 十二月二十八日 )
( 一 )
建福山门古柏群，枝雄干挺叶芳芬 ;
若然不作青城客，树木参天只是闻。
( 二 )
久慕青城天下幽，有缘今日尽倩游；
石阶万级沿峰转，杠竹轻摇仙欲酬；
( 三 )
三十六峰高入天，雾云连接好登仙；
心开涤尽凡尘虑，长寿何须道有缘。
( 四 )
太君圣座最高峰， 紫气祥云来自东；
山雾轻涂牛鼻出，青城幽誉挂仙宫。
