　　道源洞的命名，无非是取得道逢源之意。是温岭市道教协会所在地。属长屿洞天旅游区内的一个景点。
　　从温岭市东行十五公里，到长屿车站下车，沿旅游公路南行，即可看到矗立在公路旁的导游图，各处景点，使你了然于目。通过隧道，公路两旁松竹掩映，竹径通幽。沿盘山公路前行，便可看到山上有八仙拱胜，欢迎游客。复前行，便进入群山环抱，岩壑深幽。到处是山花烂漫，野果挂彩，伸手可撷。大多数可供人品尝，味道鲜美，酸甜可口。其时，唯闻虫声鸟语，山花送香。你会吟出 : “蝉噪林逾静，鸟鸣山更幽”; “鸟飞林壑里，人在画图中”。至路旁有石梁桥处 ( 又称仙人桥 ) ，便可沿石级而上。两旁古柏森森，荫蔽天日 ; 只见松鼠攀枝，轻巧活泼 ; 透过树隙，道源洞府在半山腰上，隐约可见。正像孩子急欲扑进母怀，使你疲劳顿消。一入山门，有七间新建三层楼宇，拔地而起，仿古建筑，气势雄伟。首先映入眼帘的一副门联是 : “入真门秉真教参透真玄真自在，来妙境达妙理顿开妙道妙通神” ; 匾额是 :“道为纲纪”。作为一个修真悟道场所，此联是何其贴切。凌霄宝殿坐落在三层楼上，殿宇轩昂，宽敞明亮。玉皇坐像高一丈有余，神龛内还供有玉女及四值功曹，左右供奉四御尊神。
　　养性楼，建在玉皇殿后悬崖之上，傍山依壁，地势峻险，沿三折石梯，攀登而上。内既无神像烘托，也无陈设装演。清幽寂静，迥别尘寰，真是修真养性佳处。名曰 : 养性楼，不亦宜乎。
　　东边是仿原有建筑重建而成，殿分前后两幢 ( 前面是灵官殿，后面是三清殿 ) ，另有三层楼及斋堂等建筑。据碑文记载 : 创于清光绪十五年，是开山羽士林明江所建。在“文化大革命”期间，被破坏得片瓦无存。在十一届三中全会后，赖有本洞弟子蔡信德道长，挺身而出，苦心经营，不但恢复了原貌，而且扩建了七间三层楼及养性楼等殿宇，诚属道源洞中兴羽士 !
　　其原有建筑，皆依峭壁成楼，洞中有洞。正如殿前一联云 : “云从天出天然奇峰天生就，月照洞前洞中胜景洞内天”。内有吊桥鱼潭，潭水如碧，须发可鉴，久旱不涸，霖雨不溢 ; 潭鱼群舞，逗人住步 ; 峭壁上还有降龙潭之胜，出自天然 ; 并有骚客题诗，游人玩意。今录其一 : “绦鱼迎游子，暮鼓促行人。幽草道源洞，奇花峭壁春。慢言蓬岛地，此可涤风尘。翘首山容瘦，红霞捧玉轮。”穿过左侧石窍，峭壁环耸，环境清幽 ; 内有摩崖碑记，植有芭蕉香花，可供游人观尝。斋堂上一联云 : “来客有缘一盏香茗聊供奉，山居无物几碟野菜略充饥。”是衬托出道士的淡泊生涯，热情好客。三层楼上是纯阳殿，恰巧与前边山峦上的铁树遥遥相对，并有桃红柳绿相衬，以及珍稀之树——玉叶梧桐，新叶颜色乳白如玉，一转金黄，再转碧绿，堪称一绝 ! 这里是凭栏观赏、摄影留念的好地方。从纯阳殿穿过石洞，又回到了养性楼和凌霄殿上。在凌霄殿左右，还建有小屋与旧殿相通，把新旧殿宇构成一体。
　　道源洞之胜，还有道源十八洞之誉。再游览十八洞，却别有一种情趣。在三清殿下右侧，凿有小窍，仅供人匍匐而下。冬暖夏凉，气温宜人 ;曲经通幽，石洞层出，星罗棋布，千姿百态 ; 神奇深幽，峻险妙绝 ; 不但洞奇，水亦更幽。如果你从东往西，会渐渐听到人声鼎沸，岩石轰鸣，这里是本洞与俗家弟子合办的采石场，拥有工人百余。你可亲睹采石情景。这十八洞已列入旅游区开发计划，指日可待，那时洞景焕然一新。
　　殿前场地平旷，植有奇花异木，四季不凋。晨起空气愈觉清新，柔风拂拂，花香阵阵 ; 松鼠遍地，互相追逐 ; 山鸟轻歌，松风唱和 ; 近视则前山如屏，苍翠欲滴 ; 远眺则岗峦起伏，群山竞秀。身临其境，宠辱皆忘。道源洞不但景美，而且境幽，诚集旅游、朝圣、修真之胜地。
