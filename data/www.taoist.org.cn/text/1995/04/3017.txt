　　《九天生神经》全称《洞玄灵宝自然九天生神章经》，亦称《三宝大有金书》，简称《九天生神经》、《生神章》或《生神经》。敦煌遗经伯希和4659 号题曰《太上洞玄灵宝自然至真九天生神章》。收于《道藏》洞玄部本文类。《云笈七签》卷十六《三洞经教部》收有该经。此外《道藏》洞真部谱录类另收有《灵宝自然九天生神三宝大有金书》1 卷，为该经之节本，少《九天生神章》九章及《太极真人颂》二首。
　　相传《九天生神经》为唐末五代道士杜光庭所撰，朱熹常谓“此经亦杜光庭所撰” ( 见宋赵希弁《郡斋读书志附志》神仙类《生神经》 ) 。然据陆修静《元始旧经紫微金格目》( 敦煌卷子 ) 曰:“《九天生神章》一卷，已出。”足以证明该经在刘宋时已有。又据《广弘明集》卷九《笑道论·太上尊贵十六》云 : “今据《九天生神章》，太上住在玄都宫也。”和唐释法琳《辨正论·七箴篇》曰 : “按《灵宝九天生神章》云 : 气清高澄，积阳成天。气结凝滓，积滞成地。”以及唐孟安排《道教义枢》卷一所引《九天生神章经》，开元时之《要修科仪戒律钞》卷八所引《生神章》，均表明该经非杜光庭所撰。
　　同时魏晋时期，由于封建统治阶级对民间道教的限制和打击，致使道教向上层社会发展。一部分道教徒活跃于权门贵族之间，直接参与封建统治阶级的政治活动，而封建统治阶级中的一些豪门士族，也纷纷加入道教，成为它的信徒。民间道教逐渐走向贵族化，原先许多思想很不适应时代的发展，于是出现了贵族道教改造原始道教的现象。《九天生神经》即是这一时期的产物，而非唐末五代时产生的。并且经中列元始天尊为尊神，三宝君位在三官之上，劝人诵三宝章和九天生神章。这显然与原始道教列太上老君为尊神，以三官为大神，劝人诵《道德经》和首过，差别很大。表明六朝时对道教的改革是从各方面同时进行的。
　　《九天生神经》分为四个部分。第一部分称《三宝大有金书》。第一段叙三宝君 ( 天宝、灵宝、神宝 ) 名讳。又述玄元始三炁合生九炁，九炁生天地万物和人。三洞飞玄之炁结成九天生神章，劝人诵之以升仙。第二部分为五言韵语的宝章三篇，劝人修正一法，以期太平与成仙。第三部分即为九天生神章九篇，亦为五言韵语，描绘神仙世界，声称诵经即能成仙。第四部分为《太极真人颂》二首，亦是五言韵语，劝人诵玉章。经后还附有《诵经应验》，述道士李若冲诵《九天生神经》拔度蜀主孟昶爱妾张丽华幽魂之故事。
　　《九天生神经》的产生对后世影响很大。首先，它成为道教科仪特别是度亡科仪中经常提到的经典。经中宣称 : “鬼灵闻之 ( 指《生神经》)以升迁，凡夫闻之以长存，幽魂闻之以开度，枯朽闻之以发烟，婴孩闻之以能言，死骸闻之以还人”。并且“诵之一遍，声闻九天 ; 诵之两遍，天地设恭 ; 诵之三遍，三界礼房 ; 诵之四遍，天王降仙 ; 诵之五遍，五帝朝真 ; 诵之六遍，魔王束身 ;诵之七遍，星宿停闪 ; 诵之八遍，幽夜光明 ; 诵之九遍，诸天下临，一切神灵莫不卫轩”。可见其作用极大，常念能使生者长存，死者还生，故科仪中常以此经为生者祈祥，为逝者超生。如施食科仪中，高功称职 : 臣系太上无极大道……九天自然生神上帝，救苦会中无鞍数众，使得“枯骸朽骨，咸得光辉，拔度沉溺，不滞寒渊”。又如请圣科仪中，高功称职 : 臣系太上无极大道……九天自然升神上帝，西河救苦萨公真人……三界官属，合座威灵，悉仗真香，普同供养。此外，用及该级的还有送神科仪、上大表科仪等，表明《生神经》在科仪中的地位是十分重要的。后来宋董思靖对此进行诊解，加以发挥，著《灵宝自然九天生神章经解义》)4 卷，劝人诵经。
　　其次，它是一部养生学著作。经中认为人若能“爱其形，保其神，贵其气，固其根”则“终不死坏，而得神仙骨肉，同飞上登三清是舆”。指出形与神不可须臾离，形坚则神能久任，从而斥责了外形骸之说为愚者之说。宋王希巢撰《生神经解》曰 : “《生神章》大要有四 : 一曰爱其形，二曰保其神，三曰贵其气，四曰固其根。”提出“积阳生神，积阴生形”，主张精、气、形“三者混而为一，则神仙之道不难致”。他告戒修道者当爱形、保神、贵气。指出“气本生乎精”，精为气之根，“故《庄子》曰 : ‘无摇汝精，乃可长生。’形得神以往，神得气而灵，可以还原，可以成丹”。他总结说，上焉者可以升于九天，次焉者终于尸解，不知者则徒受一形，气竭而死，如此而已。
　　第三，它是早期丹鼎派经典之一。丹鼎派是以炼丹作为修道基本方式的道派。一些天然矿石经过合炼生成药，即称之为“丹”。其基本原料是铅汞，将此初炼而成“丹头”，再炼即可生成可以服食的“金丹”。这种合炼金丹的方术当源于先秦方士的炼丹术。当方士转化为道士以后，金丹术又与古老的导引服气术融汇贯通，从而形成以修炼人体内部的精、气、神为途径的内丹术。从《九天生神经》“爱形，保神、贵气，固根”来看，其即是一部反映早期内丹思想的经典。元华阳复撰《生神经注》，亦旨在发挥内丹术。华氏在阐明《生神经》之“尊气、贵形、保命、爱神”的基础上，强调“天人一贯之妙，天与人无二理，吾之心即天地之心，吾之气即天地之气，即秉一诚，此感则彼应”。见性闻道则能超脱生死，并曰 :“人有死生，性无死生，出生死之外者其闻道乎，其见性乎 ! ”故太和真人张守清赞其注曰 : “会粹诸说之长，混而为一，引《录宝》、《黄庭》发挥丹道，简而易见，”实为“学道之指南”。
　　总之，《九天生神经》在斋醮科仪和内丹养生学方面具有极其深远的意义。
