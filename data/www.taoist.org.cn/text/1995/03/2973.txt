　　有人说中国传统文化的乳汁，足够使几个地球的人类成人、成才——使他们成为一个道德高尚的人。在麦克·哈特的《人类百名人排座次》书中认为站在宇宙的高度去审视地球的往昔烟云时，只有这 100 位高大的身躯突出在烟云之上，其中有 8 位出生在中国。古人有孔子、蔡伦、秦始皇、成吉思汗、老子、隋文帝、孟子 ; 现代人中只有毛泽东。从地球存在 50 亿年以来，已出现过几百亿人，各类百科全书中，收录过 2万多名人，但有重大影响的是这 100 位。
　　排在第75位的是老子，他著的《道德经》，麦克·哈特认为，所有被介绍到中国境外的书中，《道德经》是最伟大的一本 ; 在世界范围内，对精神文明贡献最大的，《道德经》排在第二位( 第一位，他认为是《圣经》 ) 。虽然《道德经》只有五千字，但影响却大于孔子的著作，其光辉也闪烁在儒家思想中。
　　就我个人体会，《道德经》中最能代表老子为人和道德修养主张的是第八章，上善若水。他说 : “上善若水”，大意是最好的人应该像水那样 : “水善利万物而不争”。水只想有利于万物，从不争功、争名、争利。没水即没万物。没水也没生命，没见过水夸口，它总是默默地服务。有人赞扬过老黄牛精神，但老黄牛总还得吃些草，水连吃的都不要。水“处众人之所恶”，它反凡人所说的“人往高处走”而为之，它是“水往低处流”，这低处，是众人之所恶，但水泰然处之。老子说，这样的好人，才是“故几于道”，才仅仅接近一点最为高尚的道。接着，老子从多方面的说了最好的人应该其备的品德。他要“居，善地”，即人的心地要像水那样安于卑下，“心，善渊”，即人的心地要像水那样深沉，安静，大度，宽容 ;“与，善仁”，即与人交往要像水那样有仁爱之举“言，善信”，言必信，说话要像水那样实在，流到哪里就湿润到哪里，说到做到 : ‘'政，善治”，从政要善于治理，像水那样为老百姓服务 ; “事，善能，”办事。要像水那样无所不通，无所不能，善于因势利导 ; “动，善时”，即要随着形势需要而动，像水那样可行才行，善于审时度势。
　　在上述这七种行为中，都像水那样“夫唯不争，故无尤” 。即像水那样不争名、不争利、不居功、不图报的情况下，才不至于招来怨恨。试想，如果一个人居功自傲，自已天天向人数道过去的功劳，甚至有人让别人也都得记着他的功劳，那就会让人心烦；至于那些“恃功妄行”，“有恃无恐”者，人民就更不买往日有功之账了。他没有水的那种夫唯不争的德性，就难免走向反面了 。
　　看来时至今日，以至今后永远， “上善若水”的比喻，教人学水，仍有现实意义。最后再细细体悟一遍 : “上善若水。水善利万物而不争，处众人之所恶，故几于道。居，善地 ; 心，善渊 ; 与，善仁 ; 言，善信 ; 政，善治 ; 事，善能 ; 动，善时。夫唯不争，故无尤。”真是既形象，又深刻 ; 再呆的人也会明白，人民喜欢的，正是这种像水的人。
