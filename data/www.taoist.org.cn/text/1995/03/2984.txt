　　麦子山，位于南阳盆地西沿，伏牛山南麓，长寿山东端，地处余关、七里坪、马山口三个乡镇交界处。海拔 688 米，形状如麦粒，孤峰壁立，素有“北顶” ( 武当山为“南顶” ) 之称。历史悠久，山水俱佳，观光旅游者称 : “游玩伏牛山，不至牛心垛，不足以知其高 ; 不至老界岭，不足以知其险 ; 不至麦子山，不足以知其奇。”
　　麦子山自然风景诱人。桃树遍山，松柏成林，占地三四十公顷，郁郁葱葱，遮天蔽日。旧社会，“刀客”、土匪藏匿其间，打家劫舍，剪径路人，使该林地阴森恐怖，衍生出许多杂揉的江湖侠客故事。因而此林又素有“快活林”之称，已成为攀爬麦子山的第一客栈。在这林间小憩，呼吸松柏竹叶散发的芬芳，盘坐于林间草地之上，尽情玩耍或嬉戏，你便会自然而然地寻找到那种期盼已久的“归回大自然”的感觉。
　　出松柏林，步入茂林修竹，那奇异的景观——“龙吐琼浆”便映入眼帘，一条青石小龙静卧茂林修竹之间，碗口粗的一股泉水自嘴中喷涌而出，昼夜不息，滋润林木，养育万物民众。麦子山四周只此一泉，且含有丰富的矿物质呢。传说远古天有十日 ( 太阳 ) ，天下大旱，民不聊生，“弈射九日”，普救苍生，但地表干裂，作物一时难以复苏。东海龙王动恻隐之心，派一龙子穴居麦子山半腰，口吐“玉液琼浆”，从此万物复苏，人丁兴旺。徘徊其间，泉水叮咚，鸟语花香，人“龙”同处，留恋忘返。
　　告别石龙，沿青石台阶而上，抬头仰望，三条大型石龙在半山腰逶迤盘绕，这便是远近闻名的“三围石寨、九门九关”，为中原罕见景点。内寨建于唐朝，周长 600 多米 ; 中寨建于明朝，周长 6 华里 ; 外寨建于民国二年 (1913 年 ) ，周长 10华里，三寨分设九道关卡九道石门，俨然铜墙铁壁。麦子山石寨素有“小长城”之称。
　　步入内寨，沿羊肠小道蜿蜒前行，一座小山横断其间，转过山包，眼前豁然开朗，举目望去，可见麦子山顶峰，真是“山穷水尽疑无路，柳暗花明又一村”。但若上到峰顶，还要过“舍身崖”。此崖，左为悬崖峭壁，右为万丈深渊，必须胆大心细，攀爬挪步而行。
　　过“舍身崖”，登踏青石台阶而上，麦子山无极宫耸立眼前。庙门对联为 : “麦子山巅望紫气，无极宫中乐长生。”麦子山宫始建于唐贞观年间，分前宫、中宫、后宫三部分，建筑群落，从前宫到后宫上八里、下七里，布局严谨，气势恢宏，现为明清时建筑。“祖师宫”、“玉皇庙”也基本保持完好。殿堂飞檐斗拱，雕梁画栋，五脊六兽，殿内壁画龙飞凤舞，造像栩栩如生。如今，“南天门”、“北天门”、“太子洞”、“修仙洞”、“扳倒井”、“炼丹炉”、“金鼎”等景点也修缮一新。麦子山祖师宫历代香火不断，游人不绝。如今春秋佳节，特别是农厉三月初三和九月初九，附近善男信女纷纷云集麦子山顶。
　　站立麦子山顶，背对祖师宫，俯视山下四野，赤 ( 眉 ) 马 ( 山 ) 、灌 ( 涨 ) 二 ( 郎坪 ) 公路似两根玉带从山前后缠绕 ; 湍河、默河似银河流淌 ; 城市烟囱高耸入云，高楼鳞次栉比 ; 乡村沃野千顷，房舍整齐划一，恰似人间天堂。
