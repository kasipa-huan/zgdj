　　在明代剧坛上，江南才子汤显祖以他的一部《牡丹亭》赢得了众多佳人才子的倾倒，从而独领风骚数百年。《牡丹亭》作为一部杰出的戏剧作品，曾在社会上引起轰动，“几令《西厢》减价”，其影响之巨大与深远自不待言，它奠定了汤显祖作为一代剧坛巨匠在中国戏曲史上的地位。此外，汤显祖还以他的《玉茗堂四梦》和篇帙浩繁、体裁多样的诗文作品，雄峙明代文坛，为卓然一大家，流誉后世，为人称颂。就是这样一位伟大的文学家，在他的生平和创作过程中，也时时可见道教思想对他的深刻影响，从而成为有明一代道教盛行、道教思想从上层统治阶级波及下层人民以及士大夫知识分子阶层的一个佐证。探讨一下道教思想在汤显祖这位文坛巨匠头脑中所占的比重和影响之深度，对于今天的学术研讨也未尝不是一件很有意的事情。
　　汤显祖生于明嘉靖二十九年 ( 公元 1550年 ) ，出身在一个富有道教气息的下层知识分子家庭中，他的祖父汤懋昭由于仕途不顺利，故转而信奉道教。他“能阴骘”好神仙之术，并且积极热心地宣传道教。对孙儿汤显祖，他热心鼓励他学道，而汤显祖的父亲汤尚贤却始终以儒家积极入世思想灌输给汤显祖。在这二者之间，汤显祖是倾向于祖父的，他后来回忆说 : “家大父蚤综籍于精黉，晚言筌于道术，捐情末世，托契高云。家君恒督我以儒检，大父辄要我以仙游。”(《 和大父游域西魏夫人坛故址诗》序 ) 这种情形与明代后期儒、佛、道三教合一的社会风尚的影响是分不开的。另外，汤显祖的祖母与道教人物南岳魏夫人同姓，汤显祖常常自夸“我家便有魏夫人”。据说她生时梦见南岳夫人降世，死时人皆以为她尸解。况且，汤显祖少年时教他古文词的老师徐良傅，也“颇有怀仙之致”，他们的仙道思想都对汤显祖产生了深刻的影响。这一点，从汤显祖自号为“海若居士”上也可以反映出来。海若是《庄子·秋水篇》中的海神名，由此可以看出老庄思想在汤显祖心目中的地位。
　　汤显祖从少年就喜欢思考人生问题，他的思想接受了道教宇宙观、人性论、政治学说和道德观念。在老祖父的带领下，他经常出去游仙以及炼丹，以致他后来考取了秀才，入学以后，仍然表示对功名不屑一顾，对同学说 : “上法修童智，齐庄入老玄。何言来修业，遂与世营牵 ? ”(《入学示同舍生》 ) 宁愿去修炼仙道，也不愿为名利所羁绊，这种思想植根于少年汤显祖的头脑之中，对他后来的官场仕途生涯和文学创作产生了极深刻的影响。从他的《红泉逸草》诗集中，多处可以看出他的慕道厌世的思想端倪。《灵谷对客》一诗中说 : “厌世转寻丹臼诀，怀人空散白云谣。拼将海日窥岑寂，定有人吹紫玉箫。”对寻觅丹诀、访道问仙抱有坚定的信仰。在《和大父云盖怀仙之作》中他又说 : “第少仙童色，空承大父言。”认为自己无缘成为仙童，有负祖父的厚望。在《登西门城楼望云华诸山》一诗中，他写道 : “玄符方自知，密约采飞根”，认为山中一定藏着神仙的玄符，决心进山去采仙药。在《侍大父白云桥秋望》一诗中，他也流露了看到羊角洞天便有志于寻访天书的心情。《送人入蜀求道书》一诗则写出了他满怀憧憬地派人到四川去寻求道书。故乡临川的道教传统、道教文物和遗迹深深地影响着汤显祖。在《送吴道士还华山》一诗中，他歌颂吴道士“能传吞景法，冠绝步虚声”，即既有“吞景”的法术，又会步虚的经韵。
　　由于父亲的严厉督促，汤显祖终于背弃了祖父的厚望，走上了艰难的仕途。他二十一岁中举人，可谓少年得志，可自此以后，一连串的挫折使他灰心丧气：二十二岁会试落榜，二十五岁又榜上无名，二十八岁的会考依然名落孙山。原来设想的宏伟抱负，此时全成为泡影。本以为“吏隐郎潜非俊物”，急切追求建功立业，可是偏偏又“壮年不得与明时”。 ( 《别沈君典》) 在《郁金谣》一诗中，他慨叹“慷慨功名良可叹，长日长愁行路难”。一方面他认为“鬼谷神楼非我宜”，可是另一方面，他又摆脱不了道教对他的吸引。在《寄奉举主张公参政河南》一诗中，他写自己“玩银筒玉蕴之事，以自畅自息”，在《以诗代书寄举主张龙峰令弟崶都水》一诗中，他又写出了自己渴望过“丹余葛公鼎，花落女儿床”的仙游生活，希望食仙丹。在《红泉卧病怀罗浮邡衍曾》中，他对友人邡衍曾去过罗浮山表示了极大的羡慕，说“吾欲从安期，往结天人欢”，盼望着过上“海枣是朝食，沉瀣饱宵餐”的快乐生活。他这段时期的诗文中多次出现老庄、仙游的主题，幻想“忽见跨鹤人，言是斗中真。授我《九光经》，散我升天行” ( 《黄华坛上寄龙郡丞宗武大还一篇》) 。他甚至还亲自从事炼丹的活动，不仅炼“内丹”，也炼“外丹”。他还在自己的诗作中描述自己炼丹的过程 : “男儿暗精宝，况乃炼形身。中规原抱一，正气日生神”，讲的是炼“内丹” ; “佑取祝融威，炼土塞玄冥。煎熬谨昼夜，抽添不得停。”“由然三五一，方知两七并。白雪与云芽，翼我万椿龄”，讲的是炼“外丹”。他还劝龙宗武也来修炼，认为能炼出一种“大还丹”，从而达到长生明视，乃至“升天行”的目的。
　　仕途的不顺，使得汤显祖把全部感情投进了道教宽广的怀抱。在《占仙亭晚归》一诗中，他写道 : “偶从旰姥游，遂作麻姑客。”写自己在南城麻姑山中，“道馆息云装，闲亭振全策。”并且感慨自己“未获了银丹，且言驻金碧。”在《逢南都张觅玄麻姑山中，从余来华盖，便辞去游河关》一诗中，他向往道教圣地龙虎山，说“好我龙虎山，　来醮诸神。龙虎上圣窟，占者即真人。”甚至想和麻姑一起旅游 : “东南要麻姑，去看沧海尘。”并且想和魏夫人结伴 : “南问南岳君，乃是魏夫人。”在《寄饶仑》一诗中，他写自己在临川过着“空山响长啸，手中《真浩》文”的逍遥生活，他经常到城南的青云峰去，“朝登青云峰，暮宿青云馆”，并且“但问丹铅术”，向道士讨教炼丹的诀窍。 ( 见《峰下示采药客》一诗中 ) 在《友可便欲求仙去，次韵赏之》和《送谢廷谅往华盖寻师》等诗中，他对好朋友谢延谅热衷于寻仙也表示了赞赏的态度。
　　汤显祖不但积极地慕仙修炼，他同时也在思考着道家的哲学思想，对道家主张回复婴儿的观点表示了极大的赞同。说“善哉庄生 ! 人欲为婴儿，吾亦与之为婴儿”。 ( 《寄前太守胡公》序 ) 《七月四日天清，步出城西门，望红泉宝盖，折北而东，偕内戚一浚、子杬回翔沙际，见两具枯骨，凄然痊之，道歌一首》中写他自己偶血出城外散步，看到两具枯骨遗骸，他马上想到《庄子》的《至乐》篇，说“人世露栖草，人生风落花。欢养有同尽，贤圣讵能赊 ? ”“相逢那相主，谁问骷髅家 ? ”道家思想贯穿他的一生，是他用来调节理想与现实之间的矛盾，达到心理平衡的工具。
　　万历八年，汤显祖又一次会考落榜。万历十一年，汤显祖终于考中进士，进入仕途，第一年先在礼部观政实习，次年出任南京太常寺博士。和当时同时考取的新进士一样，他热忱地忠诚于皇帝，所谓“竞折葵心捧太阳”。虽然太常寺博士是一个闲职，几乎无事可干，但却给汤显祖提供了一个了 解社会、认识人生的契机。进入官场，汤显祖马上感到了仕途的艰险，官场的腐败黑暗，龙宗武等一批友人的相继遇难，以及特务的暴戾横行，令汤显祖感到深切的失望和寒心。汤显祖更加深入地思考着人生。他不需要八股文这块敲门砖了，把大量的时间和精力放在读书上，主要是看道书。在《送太常西署翁君归潮》一诗中，他说自己“稍迁清秩奉春祠，得侍玄都观寿羽”，并且结识了更多的道士。和太常寺有关的神乐观，藏有大量道书，汤显祖几乎无不涉猎，并认真作了读书笔记，在《送徐民部北奏归觐》中说 : “陵祠我当满，亦欲归著书。”他想等官期一满，就回家著述。汤显祖的道教思想集中地体现在他的《阴符经解》一文中，汤显祖接受了老子哲学中的清虚自守、节欲保躬、致虚守柔，以不争为宝的思想，并加以阐发，在《别沈太仆》中写道 : “名不可以多取，行不可以累危。虚以居之，可以待时”。在《答王宇泰太史》中说“遵时养晦，以存其真”。在《寄王弘阳周卿》中写道 :“列子、庄生，最喜天机。天机者，马之所以千里，而人之所以深深。机深则安，机浅则危。”在《与余节侯》中，他劝戒友人说“利器不可以示人”，要“善刀而藏 (《奉李渐庵司寇》) ，几乎都是用道教处世的箴言来奉劝人。
　　万历十六年全国大灾荒，更加剧了明王朝的衰败，借“星变”的机会，汤显祖上书皇帝，企图尽忠直言，以挽回明朝衰亡的局势。结果触怒龙颜，被贬为徐闻典史。遭此打击，汤显祖并未感到十分懊丧，甚至还有点“因祸得福”的喜悦，他精神上的主要支柱就是道教思想。他对朋友说 : 我一直想去游览浮丘、罗浮、擎雷、大蓬、葛洪丹井、马援铜柱这些道教名胜古迹，但苦于一直没有机会，现在正可以借做这县尉的机会，了此夙愿。他认为自己以前在南京过的是“吏隐”生活，现在正好到南方去做一回“仙尉”。在《伯父秋园晚宴有述四十韵》中他说：“谪迁方渺渺，抗疏失区区”，降职处分算不了什么，失去的并不多。道教的热情鼓励他走上征途。在《初发瑶湖次宿广溪》中他描述自己出了临川城，到达广溪的时候，心里想着“鸠视人难老，鹏扶尉欲仙”。在《入粤过别从姑诸友》中写到他路过从姑山，与老朋友作别，还要“欲过麻源问清浅，还从勾漏访丹砂。”在《衙冈望罗浮夜至朱明观》中写到自己经过罗浮山，游玩了朱明洞，觉得“江海亦保意，谪居欣居此”。在《答崔子玉明府朱明洞相迟不至二首》中写道 : “海上神仙气，能游仙尉来。”认为到徐闻当典史简直像做神仙。在《罗浮飞云岭》一诗中，他描写自己站在飞云岭上，感到无比舒畅和惬意，“勾漏沓无人，独自神仙尉。”在徐闻任职不久后，汤显祖调任遂昌知县，临走时，他还感叹自己没有当成“仙尉”，在《答李郴州乞雨苏仙有应》中说“古称仙尉我犹惭”。
　　遂昌上任后，汤显祖兼用儒家的“仁政”和道家的“清净”思想处理政务。在《与帅惟审》中他描写自己过着“仙令”般的生活 : “满堂溪谷风松，弦歌嗒尔，时忽忽有忘，对睡牛山齁一觉”。“得天下太平，吾属老下位何恨 ? ”汤显祖也和朋友们分享这种神仙般的快乐生活，屠隆来访，他作诗云 : “空谷逢人亦快哉，平昌一榻自仙才”。在其他诗作中，他还有“神仙县令如山鬼，白云青萝石泉泚”和“神仙曾作县，君子遂名堂”等诗句。由于豪强和矿使的胡作非为，汤显祖决计辞官归里。在《初归柬高太仆应芳、曾岳伯如春》中，他形容自己快活的心情说 : “云路试留仙会令　，泥丸初着远游冠”。在《遣梦》中又有“休官云卧散仙如”之句。卸任之后，汤显祖更加倾向于道教。他广泛结交道友，如在诗中所描写的那样 : “问路名津假，寻源道术真，香台时作主，丹室夜留宾。好相金轮净，阴符火候勾。”他又有“曲坛多道书，判扉且愉堰"、"仙人近住杏花楼”、“五云楼阁是仙家”等诗句，写自己借阅道书以自娱，陶冶性情。正如他自己所说 : “生死虚空一暮朝，由来得道始逍遥。”自此以后，他的一生再也没有离开过“丹坛”，再也没有离开过道教。
　　汤显祖“少年早抱长生之诀”，道教对他早年的影响大于儒释二教。而明代社会又恰恰是一个道释儒三教同一、三教合流的社会，从开国皇帝朱元璋、明武宗朱厚照、明世宗朱厚　到明神宗朱翊钧，都信奉道教，不但帝王如此，就连官僚士大夫也不例外。明代的一代名相张居正，在政治上是一个强有力的铁腕人物，他又是一个佛教徒，但对道教的斋醮以及房中术也很有兴趣，至于一般的士大夫对道教更是崇奉笃信。文学是社会的镜子，汤显祖在他的“临川四梦”( 包括未完成的《紫箫记》) 中，深刻地揭露了明代社会的现实生活。可以说“临川四梦”是汤显祖对明代社会所作的艺术性的批判。
　　《邯郸记》讲述的是一个道教故事，即吕洞宾度卢生成仙的故事。由于汤显祖有丰富的道教知识和长期的道教活动实践，所以写来颇不外行。吕洞宾的师父正阳子钟离权有飞升、黄白之术。吕洞宾的磁瓦枕头，“是黄婆土筑了基，放在偃月炉，封固的是七般泥，用坎离为药物。”扇风囊随鼓铸，磁汞料写流珠。烧的那粉红丹色样殊，全不见枕根头一线儿丝痕路”。“枕头上面的窟窿”，是按八风开地户，凭二曜透天枢。有甚的空笼样枕江山，早则是连环套通心腑。”这枕头有许多好处，“好消息在其中，但枕着都有个回心处。”由于汤显祖熟谙道教的“丹鼎派”烧炼之术，故而剧中的描写都十分真实可信，合情合理。除了丹鼎派的法术之外，《邯郸记》中还有道教“符箓派”的法术，吕洞宾见到卢生，就“噀诀”施法。唐明皇赐卢生仙音院女乐二十四人。以应二十四气，又有一个官儿希求进用，献了个采战之术。像这样的描写，都可以看出汤显祖深厚扎实的道教功底。
　　道教的天界神仙都是有品位的，没有官职的叫散仙。汤显祖的《邯郸记》也反映了这一点 : 天上造了一座蓬莱山门，门外有一片蟠桃林，花瓣落地，无人清扫。吕洞宾度了何仙姑专司此职，后来何仙姑被提拔了，张果老叫吕洞宾到赤县神州再度一人来扫花，于是卢生就成了神仙的接班人。得道成真是道教的最高追求。道教认为影响人成仙的有“三尸”、“五累”，卢生正是为这“三尸”、“五累”所困惑，最后在吕洞宾的点化下，他终于看透了功名富贵都是身外事，大彻大悟，断绝欲望，得道成真。
　　汤显祖除了在剧作和诗文中热情赞美过道教，流露了浓厚的道教思想情绪以外，在他的一些哲学论文中，也反映出他的道教思想，这在《阴符经解》中表现得尤为突出。《阴符经》是道教经典，它主张“天道自然”、“宇宙在手万化在身”，认为只要“观其时而合其符，察其机而应其事”。则可以运生煞之柄，神机而鬼藏”。汤显祖的《阴符经解》，是古今几十家注解中最为“玄畅”的一种。《阴符经》中提出了“天有五贼，见之者昌，五贼在心，施行于天”的观点，汤显祖对此作了进一步的阐释，认为天道由阴阳构成，包藏着五行 ; “五行之气，相变相胜，自然而相于生，生而相于杀，互相生克制杀，生为恩，杀为害”，这就是天道自然。此外，汤显祖对《阴符经》中所提出的“三才”说和“愚人以天地文理圣，我以时物文理哲”等观点也作了精彩的论述。可以说，在明代哲学家的行列里，汤显祖应占有席之地。
　　总观汤显祖一生的思想历程、文学创作和宗教实践，可以看出，汤氏对道教思想的弘扬是不遗余力的。当然，他有时也不免产生一些与道教思想不合的言行，反映了他世界观上的矛盾。但无论如何，汤显祖不失为一个热情的道教信仰者，也是热情地用文学创作来歌颂描写道教的一流作家。汤显祖不仅是可以与莎士比亚媲美的优秀剧作家，同时也是一位真正的政治家、渊博的道教思想家和积极的道教宣传家，他对道教的推波助澜的作用是值得今天的人们一肯定和研究的。
