　　丘长春真人，名处机，字通密，号长春子。公元 1148 年戊辰 ( 金皇统八年、南宋绍兴十八年 )正月十九日诞生于山东省登州栖霞县滨都乡。
　　幼亡双亲，跟兄嫂度过少年时代。自幼敏而好学，博闻强记，酷慕玄风，与人言谈“非长生久视之说不道，非骖鸾跨凤之语不咏。”
　　金大定七年丁亥 (1167 年 ) 丘长春十九岁弃俗入道，居昆嵛山。是年闻王重阳祖师在宁海全真庵传道，即往拜师，成为其高足弟子，即后来的“全真七子”之一。
　　王重阳祖师，名喆，字知明，号重阳子，尊称“全真教主”。公元 1112 年壬辰（宋徽宗政和二年）十二月二十二日诞生于陕西咸阳大魏村。其家乃当地富户豪门，早年修儒业而功名不显，金初曾应武举试，作过一般吏事。因生逢乱世，怀才不遇，看破世事，辞官归家。金正隆四年 (1159年己卯、在甘河镇酒肆中遇二仙人，传授金丹口诀，乃弃家入道，在南时村挖一穴室曰：活死人墓，居内三年修炼大法。后又迁刘蒋村占结庐而居，修行传道。金大定七年丁亥 (1167 年 ) 放火焚烧茅庵。东出潼关，沿途乞化，到山东传教布道。
　　王重阳来到山东，先后在宁海、文登、福山、登州、莱州等地建立 : 三教七宝会、三教金莲会、三教三光会、三教玉华会、三教平等会等五会。提倡三教圆融，独全其真。从教弟子甚众，其中最著名的有七位，即马丹阳、谭长真、刘长生、丘长春、王玉阳、郝大通、孙不二。后创七派，各为开山祖师。
　　金大定八年戊子（1168 年 ) 春，丘长春随王重阳住烟霞洞，秋冬居文登，长春亲侍重阳祖师，旦夕不离左右，甘愿尽洒扫之役。
　　大定九年己丑(1169 年 ) 十月，王重阳携马丹阳、谭长真、刘长生、丘长春四子，游河南开封，寓磁器王氏旅邸。第二年正月初四日，王重阳召马、潭、刘、丘于榻前曰：丹阳已得道，长真己知道，吾无虑矣！处机所学一听丹阳、处玄、长真管领之，吾今赴师真之约耳。言毕羽化，四子伤惑至极。
　　四子护重阳灵柩归葬祖庵，庐墓三年。
　　金大定十四年甲午 (1174年 ) 八月，秋高气爽，在陕西户县秦渡镇的真武庙中，马丹阳、谭长真、刘长生、丘长春四人正围坐一起，月夜言志，商讨今后修道的去向，这是最后一次聚会，明日将分手，各奔自己修道之路。 马丹 阳曰：牛贫；谭长真曰：斗是 ; 刘长生曰：斗志；丘长春曰：斗闲。
　　第二天清晨，各自启单上路，马丹阳以掌教身份去终南刘蒋村，谭长真去洛阳朝元宫，刘长生去洛阳土地庙，丘长春去宝鸡磻溪……
　　从 1174年至1180 年，丘长春“乞食于磻溪。战睡魔，除杂念，胁不沾席，一蓑一笠，日乞一食，虽寒暑不变，人呼为‘蓑衣先生'。如是六年”。这六年所受之苦，世间难寻。不但常人无法想象，难以理解，不可思议，就是道教后学也难以仿效，无法作到。现在不是倡导都像丘祖当年样，去吃那么大的苦，而是意在弘扬丘祖为修道表现出的那种百折不挠的吃苦精神。这种精神不但道教应视为楷模，整个社会都应当作为宝贵的精神财富，继承发扬和光大 !
　　全真道《功课经·七真浩》曰：磻溪六年，龙门七载，道功备而名闻时主，丹符锡而掌握神仙。”依此可知“磻溪六年”对丘长春来说是何其重要！
日乞一食
　　丘长春隐居磻溪，自言“所受艰难，亦与常人异耳。”直到晚年，他对这段不寻常的隐修生涯都感叹不已。在古稀之年，丘长春应成吉思汗之召，率十八弟子亲赴大雪山 ( 今阿富汗境内 ) 。在款留期间，丘长春闲暇时回首往事，感慨平生，赋诗道 : “东海西秦数十年，精思道德究重玄。日中一食那求饱，夜半三更强不眠。”后两句是“磻溪六年”最真实、最生动的写照。
　　丘长春初入磻溪，当时情景是“烟火俱无，箪瓢不置，日用何曾积。”生活必需的烧水做饭，照明、取暖以及锅碗瓢盆等日常用具，一无所有，人要活下去，饥餐渴饮是最基本的生存保障，总得解决。可初来乍到，人地两生，又别无供养，为了生存，只好“逐时村巷求觅”，靠化缘度日，一天只化一次，化来吃一点，化不来就挨饿。所以，至今道教界流传着丘祖大饿七十二，小饿无数的说法。化来之食还“选甚冷热残余，填肠塞肚，不假珍羞力。” ( 文中所引，除注明出处者外，皆引自丘长春《磻溪集》，下同。 )
　　他化缘与众不同，自言“北方一日，南方一日，共东西交日。”这样的四方化缘，有其深远的寓意，暗示着将来要化度四方。现在化缘度日不过是“且往来，游行销日。”最终目的是“待他年，道心开日”。
　　乞食生活，春夏秋三季尚可维持，隆冬季节又该如何度过 ? 三九寒天，丘长春“求饭朝入西村，临泉夹道，玉叶凌花结。冻手频呵……”寒天地冰，肚饥衣单，冻手频呵四字，已将当时饥寒交迫之状尽述无遗。在这种困境中，他没有怨天尤人，更没有灰心丧气，而是道心愈坚。他唯一抱怨的是“自憾浊骨凡胎为劣，昼夜参差，饥寒逼迫。”他坚信自己“早晚超生灭”，现在虽然苦不堪言，也要坚定信心，“须凭一志，撞开千古心月。”
　　四方乞食也非易事，什么情况也会遇到，有时“绕村觅饭充饥，拦门饿犬，撑突走跳如飞，张牙怒目。待操心，活龈人皮。”遇到饿犬狂吠，丘长春指犬道 : “教你看家，宁分善恶，不辨高低。可叹狺狺此物，盖多生乖劣，一性昏迷……”
　　丘长春过着“饥时只解巡门乞”的清贫生活，道友见访也无茶饭相待，他抱歉地说 : 道友相看唯莫怪，贫闲守拙无相待。有时达官贵人及名士来访也是“待客迎宾不点茶。”
　　磻溪地处秦川西部，此地民风淳朴，好善乐施，丘长春初来乍到，人生地不熟，缘分未接，多有不便，可时间一长彼此有所了解，他那崇高的人品，坚守苦节的毅力深深感动当地乡民，情况就大不一样，每逢他“逐疃巡村过处，儿童尽呼饭相留”。这里的人及灵山秀水使他留连往返，他说这儿“山秀水甜人义，遍村坊，各生和气，我来不忍轻归”。
　　古语云 : “君子谋道不谋食”，丘长春乞食而不苟活，他胸怀大志向，心抱大念头，吃饭不过是“壮贯皮囊”，而真正的目的是为修道，是为了超生灭。丘长春称得上是一位谋道不谋食的真君子。
凿长春洞
　　丘长春初到磻溪，并无现成的庙宇可居，他只好在“旷峪岩前幽间畔，高凿云龛栖迹。”自己挖一小洞穴作为存身之所。
　　洞址选在何处，关乎修道的成败。丘长春独具慧眼，将洞址选在一条南北走向的峻岭之下，一个平台之上。丘长春在诗中提到 : “台边水谷尤清旷，野外山家至寂寥。”台边水谷指的是离洞不远的东边，由南向北流去的磻溪河道。
　　洞周围既有瑞草又有灵禽，附近人家又稀少，可说是一个风水宝地。洞不大，住一人尚可“任曲肱展脚和衣寝”。白昼“侧身窥洞口，明目注山巅。遥望青峰雪，皑皑白映天。”丘长春居磻溪，六年未点过灯烛 : “碧洞经年无火烛，青山终日有烟霞。”
　　夏天 : “洞内不知门外暑，心闲唯觉腹中清。”小小洞穴是一个清凉、洁净、安宁的世界。“幸得清凉无垢地，栖真且放日高眠。”“唯有长春碧岩洞，清凉终日自安恬。”丘长春住在洞内“打叠精神，物物离心上，虚空帐，慧灯明放，坐待金鸡唱。”‘日午起，行了还坐，把旧习般般打破。”又说 : “尽日无他事，唯调虎龙交媾，坐盘石。”丘长春居磻溪，六年如一日，“烟霞洞府习真修”。
　　后来丘长春赋诗一首，题曰 : 《磻溪凿长春洞》，记述在磻溪修炼的洞穴 :
峨峨峻岭接云衢，
古柏参天一万株。
瑞草不容凡客见，
灵禽唯只道人呼。
凿开洞府群仙降，
炼就丹砂百怪诛。
福地名山何处有，
长春即是小蓬壶。
蓑衣先生
　　磻溪岩边，生长着很多叫莎草的植物，到秋天采集来，经过日晒雨淋，干后成丝，可以合成草绳。丘长春采集很多，闲时便合，“千条绳就，不假良工织，闲轩亲自，结成渔父装饰。”他身披自己编织的蓑衣，“时伴樵牧嬉游，青山绿水，带雨和烟，适妙绝堪珍，幽径晚，披雪冲开芦荻。”丘长春在大自然怀抱中优游，陶然忘机，有时竟忘却自己名姓，别人也少知他的名姓，所以大家皆呼“蓑衣先生”。“我本忘名，人皆易号，唤作蓑衣客”。蓑衣先生之名就这样传开了。
以诗化布
　　丘长春居磻溪六年之久，日复一日，岁月悠悠。前面谈到吃和住，那末穿衣问题是如何解决的呢 ? 他在一首《衲衣》词中写道 : “衲袄秋来，著身渐觉相称。霜天净，畅怀游兴，不怕西风劲。百片千条，上下穿联定。宽还正，外疏狂性，内放明珠莹。”
　　他穿着“百片千条，上下穿联定”的破衲衣，白日乞食，畅怀游兴，夜间打坐，修炼内丹功。初秋渐凉尚可维持，到了深秋和严冬还穿这补了又补，破烂支离的衲衣是无法抗御严寒的。虽然丘长春自言 : “我自饥餐并渴饮，布裘不羡披绫锦。”但起码也要有最低的生存保障，冬天能够御寒才行。
　　尽管“任垢面蓬头，纸袄麻衣，行满都抛，”可现实困难必须解决。于是丘长春写诗一首，题目曰《时八月间，令人持诗于县中，觅破布补衲衣。西虢也》，诗中写道 :
白露将残寒露洁，
山家冷淡观游绝。
树头黄叶坠千林，
身上麻袍联百结。
旧布重烦七里市，
衲衣复待三冬雪。
城中富豪各仁慈，
庶在磻溪长守拙。
　　由此可知，丘长春居磻溪，吃穿两大问题都靠自己化缘解决，较特殊的是以诗化布尚无先例。以诗化布估计是磻溪六年的中后期，丘长春已名声远播，人们既知道他是一位隐居苦修的高道，又崇拜他的道法文章，所以见到他的诗一定会施舍。
　　有些道友不详其情，反向地觅要败布和旧鞋，丘长春十分为难，他只好向道友述说苦衷 :“余在西虢六年，未尝新衣履，每至中秋唯完补褐衲耳。”他在诗中写道 :
秋风忽起雨天凉，
木叶萧疏草渐黄。
褐衲悬鹑唯补阙，
芒鞋伏免不能狂。
有身易着饥寒苦，
无福难逃日月长。
但愿诸公怀侧隐，
扶持同步入仙乡。
博览群书
　　丘长春居磻溪，以苦磨性，一心修真，看起来是避世闲居，其实这种闲绝非无所事事，从他的《磻溪集》中可以看出他一方面为完善自我、超越自我而苦修，另一方面胸怀济世度人之志和弘扬教门之愿而刻苦读书。所以，除了炼性修命外，还不忘博览群书，不断开拓知识领域，加固文化根底，使之日后能为教化之用。
　　虢县有一姓张的秀才家里藏书甚丰，因张氏系名门望族，书香富豪之家。丘长春称赞张氏家族“光辉代代生豪杰，讲论时时聚德星。”丘长春借阅其家的藏书，张秀才尽力满足他的要求。张秀才十分敬仰丘长春的道德文章，两人时有唱和。丘长春在一首赠诗中写道 : “顾我微才宏道晚，知君博学贯心灵。嘲吟不用多披览，续借闲书混杳冥。”
　　丘长春所看之书，除儒、释、道三教经典外，还涉猎秦文汉赋、唐诗宋词等历代文学名著，同时对《周易》也深究妙悟，下过一番功夫，“百尺孤松影下，独弄周天卦。”
　　丘长春博览群书，学识丰富，加上悟性超群，所以在修道之余也艺兼多样，既是诗人又是书法家，《磻溪集》中有题纸扇诗多首，说明求字的人很多。
　　丘长春博览群书不是为求取功名，而为实现“从师别后，更谁风范相继 ? ”以及“休怅怅，大器晚成，自有孤高栖凤质，能教倜傥化龙形，他日看超升”的凌云志而用知识充实白己，陶冶情操，扩展视野，壮阔胸怀。除参修道学之外还兼学别家学问，也符合全真道“三教圆融”的宗旨，丘长春提倡“修行大抵要聪明”，这聪明也包含着博学的内涵吧。
狂写诗篇
　　“工巧文章与词赋，多能碍却修行路。”古来修行家不主张精于文墨，所以一般人入道便废斯文而专事修养。
　　丘长春则不然，他主张修行吟诗两不误，“吟诗闲度日，观化临风前。”“独立红尘外，孤吟碧嶂前。”在磻溪六年，丘长春追求的是古代道家那种自由自在的逍遥游式的修行法。他在诗中写道 : “大智闲闲，放荡无拘，任其自然。寄雅怀幽兴，松间石上，高歌沉醉月下风前。”此种修行法，可以说既有儒之雅，又有道之逸。所以他“无烦恼，任开怀，纵笔狂写诗篇。”
　　丘长春以诗明道，教化世人，他的诗可说诗中有道，句句珠玑。他提醒不明道者“向词中明取一言”，只要吃透词中意，便可参悟出修道的真谛，所以“拍手笑归”。
　　丘长春作诗不事工巧，而是率性即兴，有感而发，功出自然，妙而合道。他说：“吾常志僻心颠，爱箪瓢淡薄，词翰嘲掀。幽洞小溪，开怀取兴。时成短句长篇。”诗如其人，从诗中可以体味到他的人品、情操与境界。
　　他写诗专注于情志，有时“任奋笔狂吟，走雾飞烟” ; 有时“策杖寻幽径，每日巡村转一遭，信步闲吟咏。”一部《磻溪集》尽述修道的艰辛历程，及当时的心境与体会，可说是长春真人修道过程的真实写照，也是启迪后学的最好教材。
筑清风台
　　丘长春在清风台上与来访者下棋，石棋盘至今犹在。清风台离长春洞不远，在磻溪宫内，据旧志记载 : 清风台系丘长春真人所筑，上有丘真人祠，原有卧牛石。异柏一株，枝干皆左扭，传说丘真人所植。东崖有一矮柏，枝垂而不扬，传为丘真人晾衣树。
　　据旧志记载 : “公讳处机，字通密……于是登西虢，览山川之秀气，访神仙之遗址……遂飞遁旷谷，筑一台，西依飞云之壁，东临漱玉之溪，北跨渭滨，南依山色，中引清风，故曰清风台。”最初只是一个土台，丘真人被封为国师后，人们为纪念他而辟为遗址，自元代始在清风台上建祠立庙，塑真人像，与长春洞一样成为香火之地。清康熙三十六年重修，系楼阁式建筑，砖砌台基，上为九脊歇山顶，本为一建筑群，今只留清风台及石棋盘一块，余皆毁。
　　丘长春有《无俗念》词十二首，其中之三“抨棋”说到其在清风台上与来访者下棋之事。同云 : “天赐我棋局，开颜销日。古柏岩前，清风台上，宛转晨餐毕，幽人来访，雅怀闲斗机密。”
　　丘长春筑清风台不单纯是为了在上面下棋，主要目的是想辟一修练外功的活动场地。这里四面来风，日照时间长，视野开阔，是一个理想的活动场所。上面有树，既可晾衣晒被，又能待客下棋，夏季夜静之时在这里吹箫，箫声宛转悠扬，别是一番情趣……
　　清风台在磻溪宫内，磻溪宫又叫清溪宫，旧志载，这里为丘长春成道处，本名长春观，丘入元为国师，元诏改“观”为“宫”。磻溪宫俯水面山，殿阁巍然，古时为宝鸡第一道场。宫内有八棱碑高丈余，刻老子《道德经》，是中国十三部《道德经》石刻本之一，世称为“磻溪本”。另有《丘长春内传碑》一通，古清风台一座 ; 宫外有银杏树一株高数丈，大数围，虬枝纷披，传为丘长春所植。宫内殿宇已毁。
　　除磻溪宫外，渭水以北还有南昌宫和磨性山，均为丘长春足迹所到过的遗址。现简述如下 : 南昌宫——在今宝鸡县西秦村，旧志载，一名玉泉观，一名小蓬壶。宫中有白鸡泉，水分两窍，交流成沼，松柏郁郁，仿佛蓬莱仙岛，故名小蓬壶。另有一说 : “闻海有神山，曰蓬莱，曰方壶，仙者之所居也。……周原之麓，有元长春丘真人昔曾居之地，以仙灵名非虚饰矣。” ( 见道光五年所立《重修小蓬壶记》石碑 ) 丘长春曾在这里居住过，他是真人，所以叫小蓬壶。南昌宫原有寥阳殿、通明阁、雷祖及三官诸殿，还有静宝、云馆、蔬圃、水磨并诸多碑碣，殿堂犹在，规模尚存，却已无往日风貌，尚有全真道士居住。
　　磨性山——在今宝鸡县杨家沟乡西高泉，丘长春曾在此磨性成真。旧志载 : “山背依周原，前临渭水。山下有左右二泉，下流汇为莲池，夏日莲叶田田，荷花朵朵。”
　　这里原有丘长春丹房，丹房后有混元洞，洞有二石大如斗，光滑圆润，传说当年丘长春每日晨起，滚此石于山下，再肩之山上，每日往复数次，从不间断，故称此石为磨性石。
　　原来洞舍颇多，绝顶石墙如城，洞辟四门。中为三清殿，东有太皇宫，西有菩萨殿。山腰有七个山洞，为老君洞、混元洞、药王洞、雷神洞等。明弘治十四年 (1501) 增修。现宝鸡县道教协会在此。
