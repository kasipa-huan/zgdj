　　功课经是道教徒日诵修持的重要经典。即所谓“功课者，课自己之功，明自己之性，修自身之道”，是道教徒修持寻真的门户。课功能使道教徒修心炼性，合助道力及保养元和。《祖天师宝浩》是功课经中主要宝浩之一。
　　“本来南土。上溯蜀都。先获黄帝九鼎之丹书。后侍老君两度于玉局。千轴得修真之要。一时成吐纳之功。法箓全成。受盟威品而结璘诀。正邪两辨。奇福庭治而化咸泉。得就大丹。道齐七政。大悲大愿。大圣大慈。三天扶教。辅玄体道大法天师。雷霆都省。太玄上相。都天大法主。正一冲玄神化静应显佑真君。六合无穷。高明大帝。降魔护道天尊。”
　　就此宝浩涉及的内容，加以概说祖天师修道证道过程。
　　祖天师张道陵是汉留侯张良的八世孙，原名陵 (34-156) ，字辅汉，东汉沛国丰人 ( 今江苏丰县 ) 。七岁时，就已精通《道德经》，曾入太学，通晓天文地理河洛图纬一类书中奥妙及五经学说。十六、七岁时，身高九尺二寸，浓眉毛大下巴，高鼻方口，朱顶绿睛，目有三角，相貌奇异魁伟。喜读黄老之书，信仰早期道教——黄老道。因蜀地多名山，且民风淳朴，善于教化，随后人四川修道传道。故经中谓“本来南土。上溯蜀都”。
　　张道陵悠然于摄气养神之道。曾闻“黄帝服九鼎神丹，遂以升仙 ; 黄帝陟王屋而受丹经，到鼎湖而飞流珠，登崆峒而问道于广成……”。张道陵受其黄帝升仙思想，追求神丹。为达到长生，寻访高人隐士，冀求至道。精诚所至，金丹亦得。据《仙鉴》、《汉天师世家》、《云笈七签》记载：“……复又与弟子王长从淮入江西都阳，溯流至云锦山( 龙虎山 ) 炼九天神丹，三年丹成而龙虎现，山因以名，时年六十余岁，饵之益壮。一日有神人自称赵公明骑黑虎至，曰 : 愿永护真人丹扃，道陵命其镇守玄坛，暨访西仙源得制命五岳摄召万灵及神虎秘文于壁鲁洞，继往嵩山，于石室得《三皇内文》 ; 《黄帝九鼎丹书》及《太清丹经》”。是谓经中“先获黄帝九鼎之丹书”。从此以后，张道陵修道有了很大转机，充实思想内容，丰富了修道理论，完善了证道依据。
　　张道陵受度于“太上老君”。汉顺帝壬午年元宵之夜。张道陵听得仙乐从天际隐隐传来。举目而望，见紫云满室 ; 一乘五条白龙拉着素车，··一神人手执五明宝扇，项负八景圆光，容若冰宝，神光四射，仪态万方，有使者飞降张道陵前曰 : “你不必惊慌，我主太上老君来也”。老君曰 : “近来蜀中有六大魔王、八部鬼帅在兴妖作怪，残害生民，十分可恶，你且替我好好整治他们”。传《正一盟威秘箓三清众经》九百三十卷，《符箓丹灶秘诀》七十二卷，雌雄剑一对，都功印一枚，冠衣、方裙、朱履各一副，授给张道陵。
　　张道陵再受“太上老君”所度，授受道书。据《云笈七签》卷六载：“汉末，有天师张道陵精思西山，‘太上'亲降，汉安二年 (143) 五月一日授以《三天正法》，命为天师 ; 又授《正一盟威妙经》、《三业六通之诀》重为天师正一真人”。卷二十八又说 : “谨按张天师《二十四治图》云 : ‘太上汉安二年 (143) 正月七日中下二十四治”。综上所述张道陵受度于“太上老君”，并成天师地位。是谓经中“老君两度於玉局”。其间留下大量文献资料。道陵以此宣行道化，更是天师兴道，发展教团组织的重要原因所在。设“祭酒”为教领，盛行“正一盟威之道”。为道教的形成在理论上奠定了基础。
　　张道陵发展道教经典开了先河，首创了向天、地、水三官祈祷的仪注。出现了最早经典《三官手书》。其皆意教人叩首思过、病者请祷。请祷之法，书病人姓字，说服罪之意，作三通，其一上之天，著山下 ; 其一埋之地，其一沉之水，谓之三官书。祖天师奉祀三官的思想，对后世道教经典很有影响，形成道教典籍《三官经》。总之人生病，是天、地、水神发怒所降，人若求神痊愈病体，显示神的威严，贯彻了“天人天威，心神合一”的教义思想，也是修真理论一方面的开端。
　　张道陵精神高于物外，归宿于纯粹的神仙思辨世界。解脱人生，寻其大道奥妙。早在东汉明帝时，曾以二十六岁“举贤方正极谏科”做巴郡江州 ( 今四川重庆 ) 令。后隐退北邝山 ( 今河南洛阳北 ) 。天缘巧合，遇魏伯阳祖师恳求妙诀，炼就长生。以真行得真功。故谓经中“千轴得修真之要。一时成吐纳之功”。后受到朝廷重视，皇帝多次征诏，皆称疾而不应。
　　张道陵在蜀创立道教，使人人有道德，向往天下太平的美好时代，以二十四治设立二十四个教区，教万民崇道。正在此时有邪妖残害人间，为使生民免受其害，张道陵作好了除妖降魔的准备。忽有人报，在西城、房陵之间，有白虎神常出饮人之血，道陵召之则训一，凶神从此匿迹。有毒蛇化出剧毒雾气沾人，沾之者则无有一人幸免。张道陵念咒书符，禁闭毒蛇。又慈憨化民，以符水咒法，为民治疾。
　　闻八部鬼帅领兵亿万，在人间到处流窜，散布各种疾病，为救其害，于是‘奉太上老君话命，佩《盟威秘箓》，以丹笔降服八帅，随后又降服六魔，将其“六魔、八帅”率众返回幽暗地下，阳阴人间不再作祟。张道陵摄邪归正，分清人妖，行盟威之妙诀。则谓经中“法箓全成。受盟威品而结璘诀。正邪两辨。奇福庭治而化咸泉”。
　　张道陵在鹤鸣山苦节学道、音气养神，专以清虚淡薄为务，简省人事。效“轩辕味道，松乔食霞”潜心修炼。相传，鹤鸣山上有一石鹤，每当有修炼得道者，万鹤就会引吭鸣“”，张道陵隐居此山后，按神人所授《上三皇内文》、《黄帝九鼎太清丹经》，精思玄微。修玄玄之道，成龙虎天丹，三年丹成。石鹤果然再鸣，儿欲起飞。证明修道之大成，通其登真之门，最后丹成于都阳山。故为经中“得就大丹”。
　　祖天师张道陵，道德在躬，动得鬼神之助，一嘘一吸间，天道为之晦冥。雷霆诸神，莫不受命。以此辅国济民，除其妖孽，援其冰旱。其正一盟威之道在蜀中得到发展，以致扶正抑邪。著名道教圣地青城山已成为“宝仙九室第五洞天”。山清水秀，大道兴盛 ; 佑时理化，天下太平，人民安居乐业，是谓经中“道齐七政”。
　　张道陵能使在修炼方面达到更高的境界，曾备修山岳众神之术，功德并著于天下，道法闻名于四海。这种度人济物的慈悲之心，故谓经中“大悲大愿。大圣大慈”。同时也展现了道教“仙道贵生，度人无量“之宗旨。主张以清静为本，受精养神之养生方法。其功盖世，力济无穷。因此，张道陵伴随着历代沿革，曾受到历代皇帝救封。
　　唐玄宗天宝七年 (?48 年 ) 册封天师为太师。
　　唐嘻宗中和四年 (884) ，封三天扶教辅玄大法师。
　　宋神宗熙宁年间 (1068 一 1077 年 ) ，加封三天扶教辅玄大法师。
　　宋徽宗大观二年 (1108 年 ) ，册封正一靖应真君。
　　宋理宗嘉熙三年 (1239 年 ) ，册封为三天扶教辅玄大法师正一静应显佑真君。
　　元成宗元贞元年 (1295 年 ) ，加封正一冲玄神化静应显佑真君。综述之上谓经中“三天扶教。辅玄体道大法天师。雷霆都省。太玄上相。都天大法王。正一冲玄神化静应显佑真君”。
　　张道陵承受经典《北斗经》，“太上老君”册效最高封号。
　　永寿元年 (155 年 ) 七月七日，“太上老君”授给天师张道陵《北 斗经》，从而充实道教经典。为探索人生道路，垂训“人身难得，中土难生，假使得生”，最难最难的就是“正法难遇”。明晓经典辅赞大道，修其至道就归根良命，从而更加完善了道教教义及道教理论。
　　永寿二年 (156 年 ) ，张道陵游渠亭山时，得到“太上老君”的敕封。据《汉天师世家》、《云笈七签》等道书记载，“张道陵奉太上册敕命为六合无穷，高明大帝，封为天师。以盟威都功诸品秘箓、玉册、剑印授子衡，与夫人雍氏仙去，时年一百二十三岁”。上述可知，张道陵降魔护道，册封最高封号。谓经中“六合无穷。高明大帝”，也称“降魔护道天尊”，
　　祖天师宝浩仅仅二十多句， 131 字，是对张道陵的歌颂和赞美二包含了修道证道过程，创立教派，教人悔过奉道，学习《老子》五千字，设立“祭酒”统领教民，用符水咒水法为人治病，奉“五道米道”‘张道陵自从“太上老君”降命天师，也称为天师道、张道陵刳心炼形与道合真。因此上，其地位极其崇高。综述玉浩内容，阐发了教理教义、经典称养生方法及法术符咒等，对后世道教有很大影响。启开性命之学，去追求这朴归养生之道，也符合了人们寻真、至善，贵生乐生的天赋灵性。是宝浩最重要内涵之所在。
　　道教徒日诵祖师宝浩，能知祖师当年艰辛的修道历程令可作为今天道教徒的修持，口诵公持。攒年至外，攒月至日，攒日至时，时时归于自身的“身、心、意”三宝。心不调自静，神不调自清，气不调自和，必息相依。与天地之冥合，得其难得之货。神气绵绵皆相顾，骨肉融和玉液流。诵持宝浩，若周行而不殆，其美妙尽在不言之中……。
