叶法善生卒年辨析
　　唐世特重道教，道士叶法善即以阴阳、占繇、符架之术屡受诏于高宗、武后、中宗三朝 ; 玄宗先天二年 (713), 甚至被拜为鸿胪卿，是唐史上颇为显赫的宗教人物。但史书有关其生卒年的记载却相当混乱，乃至错讹 ; 今人著述，多莫衷一是。试作辨析并考证如下 :
　　《旧唐书》卷一九一“方伎传” : “法善生于隋大业之丙子，死于开元之庚子，凡一百七岁。八年卒。”按，“隋大业之丙子”即大业十二年 (616); 开元凡二十九年，无庚子岁，此处作“开元之庚子”显然有误 ; 其云“八年卒”，如果理解为开元八年庚申岁 (720) ，则法善寿考当为一百五岁，与“一百七岁”又相矛盾。
　　《新唐书》卷二 O 四“方技传” : “ ( 法善 ) 开元八年卒。或言生隋大业之丙子，死庚子，盖百七岁云。”按，此处记载已意识到“百七岁”与“开元八年卒”的矛盾，而列出两种说法 : 一是但知其卒年为开元八年 ; 二是知其生年为大业丙子岁 (616),亦知其卒于庚子岁，凡一百七岁。第二种说法不符合历史纪年，考开元前后之庚子岁，一在久视元年 (700) ，一在上元元年 (760) ，法善若卒于此二庚子之一，其寿考当为八十五岁或一百四十五岁，亦不与一百七岁合。故此处之庚子仍有误，“百七岁”云云也难得确解。
　　《册府元龟》卷五三“帝王部·崇黄老” : “玄宗开元八年六月，道士叶法善卒。……比年九十卒。”卷八七六“总录部·方术”所载同。此二条记载，可推出法善生于贞观五年 (631) ，卒于开元八年 (720) ，享年九十，与两《唐书》所记不一，当别有所据。但《册府元龟》卷七八四“总录部·寿考”又载 : “叶法善世为道士，……生隋大业之丙子，死于开元之庚子，凡一百七岁。”又与两《唐书》所记相同。可见《册府元龟》本身的记载也自相矛盾，难以定夺。
　　元赵道一《历世真仙体道通鉴》向称道家之《资治通鉴》，卷三九云 : “ ( 法善 ) 至开元八年尸解于景龙观。或言生隋大业丙子，卒壬戌，盖百有七岁。”这里的记载亦已意识到两《唐书》等旧闻的矛盾，因此分为二说 : 一是仅知其卒年为开元八年 ; 二是知其生年为隋大业丙子岁 (616) ，且知寿考百有七岁，并推其卒年为壬戌岁，即开元十年(722) 。
　　清代的御制《全唐诗》和《全唐文》都是当时博学的硕儒编定，均撰有叶法善小传。《全唐诗》卷九八 O: “叶法善字道元，……开元八年，年一百七岁，化去。”《全唐文》卷九二三 : “法善，括州括苍人，……开元八年卒，年一百七岁。”这两则记载也都对旧说的矛盾作了处理。按照通常的理解，《全唐诗》、《全唐文》所记叶法善的生年当是隋大业十年 (614) ，卒于开元八年 (720) ，凡一百七岁。
　　综汇以上的记载，关于叶法善的生卒年共有四说 : 一、仅知其卒于开元八年 (? 一 720); 二、生于贞观五年，卒于开元八年 (631－720); 三、生于大业十二年，卒于开元十年 (616－722); 四、生于大业十年，卒于开元八年 (614－720) 。
　　判定历史人物的生平事迹，当时人所作的碑传墓志应是可信的第一手资料。今考《全唐文》卷四一，有玄宗李隆基所撰《故金紫光禄大夫鸿胪卿越国公景龙观主赠越州都督叶尊师碑铭并序》一文，详载叶法善事迹，可确定其生卒年。其云 :“至隋大业，岁在景子，法善是生。凡六百四十二甲子，洎我开元，岁在庚申，形解升云，则春秋百有七矣。”景子即丙子，避唐高祖之父李昺讳改 ;大业景子即大业十二年 (616) ，开元庚申则是开元八年 (720) 。若按常规计算，法善寿考为一百五岁 ; 但玄宗在文中强调共历“六百四十二甲子”，以每年历六个甲子日核计，正好是一百七岁。这里显然是将这一百五年中的闰月也相加为二年算进去了。关于这一点，南宋张道统编纂的《唐鸿胪卿越国公灵虚见素真人传》( 见正统《道藏》孝字函 ) 所录的玄宗该碑铭中有明确解释。在碑文“则春秋百有七矣”下，《道藏》本有小字夹注云 :“自丙子至庚申凡一百有五岁，余闰计诚有六百四十二甲子之数，遂以百有七岁。”
　　玄宗的碑记，所以作百有七岁，实际上是在道教宣扬长生观念下的故弄玄虚。宋代以来的史书不明就里，照录其文而未作详察，遂至难以理喻 ; 分作诸说，均已离谱 ; 两《唐书》等又将庚申讹作庚子，更增疑窦。这些讹误在校勘中均应予以是正。
　　关于叶法善生卒年的正确结论应是 : 生于隋大业十二年丙子岁 (616) ，卒于唐开元八年庚申岁 (720) ，凡一百五岁 ; 按每年历六甲子日计，又为一百七岁。
史崇即史崇玄补证
　　《全唐文》卷九二三著录史崇，小传云 : “崇，武后朝太清观主，授金紫光禄大夫、鸿胪卿员外置同正员、河内郡开国公。”存文一篇曰《妙门由起序》。
　　按，此史崇即史崇玄，傅璇琮、张忱石、许逸民《谈 ( 全唐文 ) 的修订》一文 ( 载《文学遗产》 1980年第 1 期 ) 已指出 : “卷九二三有道士史崇，而据《新唐书》卷五九《艺文志》三，载道士史崇玄与崔湜、沈佺期等撰《道藏音义目录》一百一十三卷，则史崇名字又缺第三字‘玄’。”但史崇名字缺“玄”的原因，《唐五代人物传记资料综合索引》( 傅璇琮、张忱石、许逸民编撰，中华书局 1982 年版 ) 第 587 页认为 : “《全唐文》作史崇，盖修纂者避清康熙讳省‘玄'字，今据《新唐书·艺文志》补。”这一注释，似略欠允当。
　　今考《妙门由起序》一文又载明正统《道藏》仪字函卷一 ( 文物出版社等影印本第 24 册第721 页 ) ，当系《全唐文》所本，其作者已署名作史崇。可见至少在明代，史崇玄之名已省玄字，又考唐至清以前的历代史讳中，讳玄者唯宋始祖赵玄朗。陈国符《道藏源流考·历代道书目及道藏之纂修与镂板》 ( 中华书局 1963 年版第 116 页 )亦云 : “《新唐书·艺文志》所云道士史崇玄，即此领修《一切道经音义》之太清观主史崇。避宋讳故脱玄字。”由此可见，史崇玄当即宋人避玄朗讳而省玄字 ; 《全唐文》作史崇，当由《道藏》出，非特为避清康熙 ( 爱新觉罗·玄烨 ) 讳而省。
　　史崇玄避宋讳而省玄字，更有力的证据是在《道藏》藏字函中收录的北宋陈景元 (1035 -1094) 所撰《上清大洞真经玉诀音义》一卷 ( 影印本第 2 册第 705 页 ) 。该书广采古本《大洞真经》校勘注解，其中征引《一切道经音义》处颇多，而作者史崇玄均省作史崇。如 : “泱泱 : 叶韵音央，见唐光禄大夫太清观主史崇《一切道经音义》。”
　　在《道藏》所载道书中，又有署作史崇秘的，见元赵道一《历世真仙体道通鉴》 ( 《道藏》咸字函，影印本第 5 册第 99 页 ) 卷 39 “申元之”条云 :“申元之者，……唐明皇开元中召入上都开元观。时张果、邢和璞、罗公远、叶法善、尹愔、何思远、史崇秘、口希言同左佑真风，翼戴人主。”史崇秘当即史崇玄，避讳而改玄作秘。赵道一此书所集真仙事迹，均实据旧闻，非出杜撰，“申元之”条也应是据宋代刊刻之道书资料编写，因此也承袭了当时的避讳。
　　又在明道士白云霁编撰的《道藏目录详注》卷四也提及史崇玄，但将《一切道经音义妙门由起》的作者写成了“太清观主史荣等奉敕撰。”既未辨前朝之讳省，又以形近而讹“崇”作“荣”，与原名就相差得更远了。
　　综上可知，唐代道士史崇玄，因避宋讳、而省作史崇、改作史崇秘，又因形近而讹作史荣，实皆一人之名而异写也。
吴筠卒葬之“天柱山”小考
　　吴筠是唐代道教上清派代表人物，玄宗时曾以翰林供奉见召 ; 其所著述，在道教思想史上影响很大。关于他的生平，正史的记载颇有谬误之处 ; 其所终，亦有越中、宣城二说。今人的考辨虽都以宣城为定论，但均未称公允。这里的关键在于对其所葬地“天柱山”归属问题的考证上。试以代表性的两种说法进行考辨如下 :
　　郁贤皓先生《吴筠荐李白说辨疑》 ( 《李白丛考》第 65 页 ) 有两处关于吴筠卒年事迹的论述 :
　　一、“《序》( 指权德舆《唐故中岳宗元先生吴尊师集序》) 说吴筠在大历十三年卒于宣城，葬于天柱山。考吴筠《天柱山天柱观记》 ( 《全唐文》卷九二五 ) 云 : ‘筠与逸人李元卿，乐土是安，舍此奚适 ? 恐将来君子，靡昭厥由，故核而志之，表此贞石。'证明吴筠最后一年确实在宣城天柱山，权《序》是正确的。而《旧唐书·吴筠传》却说他‘终于越中'，则显然又是错误的。” (《 丛考》第 72 页 )
　　二、“上文已提到，大历十三年吴筠到了宣城，写有《天柱山天柱观记》。” (《 丛考》第 76 页 )
　　卢仁龙先生《吴筠生平事迹著作考》 ( 《中国道教》 1990年第 4 期第 34 页 ) 记其卒年事迹云 :“大历十三年，卒于宣城天柱山，弟子葬于山之西麓（《权序》、《仙鉴》。或曰卒于剡中，见《赤城志》、《直斋书录解题》。案 : 当从《权序》。吴筠前此有《天柱观隐所答韦应物》、《天柱观记》、《舟中遇柳伯存归潜山因有此赠》。天柱山，正在潜山内，是可知吴筠活动于安徽境内。且《剡 录》亦未言及其所卒于剡中之事。故从《权序》。 ) ”
　　郁氏之记述将天柱山归于宣城境内，并以《天柱山天柱观记》作于宣城，说法欠妥。卢氏的记述也认定吴筠卒、葬于宣城天柱山，但案语又以此天柱山是在潜山境内，而宣城、潜山虽均在皖地，却是江南、江北，有道里之遥，不可混为一处 ; 又将《天柱观隐所答韦应物》、《天柱观记》认定是在皖之作，也非允当。
　　今考吴筠《天柱山天柱观记》原文有关记载云 : “粤天柱之号，潜、霍及此，三峰一称矣。……自余杭郭溯溪十里，登陆而南，弄潺　，入峥嵘，幽径窈窕。才越千步，忽岩势却倚，襟领环掩，而清宫辟焉。……暨我唐弘道元祀，因广仙迹，为天柱之观。……筠与逸人李元卿，乐土是安，舍此奚适 ? 恐将来君子，靡昭厥由，故核而志之，表此贞石。大历十三年正月十五日，中岳道士吴筠记。”在中国境内以天柱山称名者颇多，日人青山定雄编著的《中国历代地名要览》所载就有六处。这里的天柱山，据文意可知，是在杭州余杭县境内，也就是在道书中通常所称的大涤洞天之天柱峰，或洞霄宫。吴筠大历十三年正月在余杭，愿以天柱山为终焉之地，所以写下了《天柱山天柱观记》以明志。在文章的开篇即说明余杭之天柱山与安徽潜山、霍山二县的天柱山是“三峰一称”，因此这里的天柱山就不可能是在安徽境内的宣城，更不可能是在潜山等地。
　　吴筠的这篇《天柱山天柱观记》，又见于元人邓牧所编的《大涤洞天记》 (《 道藏》当字函 ) 中，题作《天柱观碣》 ; 另外，《辨疑》提及“孙望先生《全唐诗补逸》从孟宗宝的《洞霄诗集》中辑出 ( 吴筠 )诗五首” (《丛考》第 76 页 ) ，洞霄是余杭天柱山的道教称谓，由此二点也可旁证吴筠栖隐之处是浙江余杭的天柱山。
　　又据权德舆《中岳宗元先生吴尊师集序》 :“大历十三年，岁在鹑首，止于宣城道观，焚香返真于虚室之中。门弟子有邵冀元者，率吁其徒宁神于天柱西麓，从其命也。”可知吴筠大历十三年又到宣城，并卒于此地 ; 按照他的遗愿，弟子邵冀元与众徒又将其安葬于天柱西麓。既是遗命，这个天柱山只能是《天柱山天柱观记》所称“乐土是安，舍此奚适”的余杭天柱山。宋陈思《宝刻丛编》、郑樵《通志》均著录《唐内供奉道士吴筠碑》立于贞元十年，在“余杭西十九里天柱观内”，可见吴筠最终确是葬在了遗愿所在的余杭天柱山。这一点，还可以找到的佐证是载于《大涤洞天记》中元代隐士叶林在大德九年 (1305) 所作的《白鹿山房记》 : “唐吴 ( 筠 ) 贞节天师美文词，与太白齐名，尝构石室为藏书地。逮尸解宣城，语弟子当还天柱石室，此石室所以著也。”这里的记述，所据当是有关天柱山的旧志，与权德舆的序所载是一致的。
　　“越中”一词的泛称，在古代也可包括杭州地区在内，以《旧唐书·吴筠传》为代表的说法，认为吴筠”竟终于越中”，大概就是因为吴筠最后是葬于余杭天柱山，却没有详审卒地、葬地的分别而致误吧 ? 今人之以吴筠终于宣城虽然正确，但又将其葬地天柱山误归于皖地，并带来诸如将吴筠有关天柱山诗文归于宣城的一系列错误，可惜未达一间。
　　吴筠卒葬的准确论述应该是 : 大历十三年，卒于宣城道观，归葬于余杭天柱山。
